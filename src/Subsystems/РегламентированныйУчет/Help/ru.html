Регламентированный учет

Назначение подсистемы

Подсистема предназначена для целей единовременного отражения фактов хозяйственной деятельности предприятия в бухгалтерском и налоговом учете. Одной из ключевых задач подсистемы является формирование типовой отчетности для дальнейшего предоставления в различные государственные органы.

Настройка подсистемы

Для настройки подсистемы требуется предварительное включение следующих функциональных опций в разделе НСИ и администрирование:

  • Установить в разделе Финансовый результат и контроллинг - Финансовый результат флаги: Группы настроек финансового учета для возможности классификации объектов по правилам отражения в бухгалтерском учете (группы настроек финансового учета номенклатуры могут использоваться бухгалтером для объединения номенклатуры, имеющей одинаковые счета учета и/или для объединения однотипных расчетов с партнерами); флаг
  • Установить в разделе Предприятие - Организации флаги: Несколько организаций, если планируется осуществлять хозяйственную деятельность от лица нескольких организаций, входящих в состав предприятия; флаг Передачи товаров между организациями, если планируется осуществлять передачи и возвраты товаров между организациями, входящими в состав предприятия; флаг Обособленные подразделения на отдельном балансе, если планируется вести учет обособленных подразделений, выделенных на отдельный баланс; флаг Раздельные операции закупок и продажи для упр. и регл. учета, если планируется использовать управленческую организацию.
  • Установить флаги Комиссионные продажи и Комиссионные закупки в разделах Продажи и Закупки соответственно, если планируется осуществлять продажи и закупки товаров по комиссионной схеме.
  • Установить в разделе Продажи флаг Продажи на экспорт для возможности учета НДС при оформлении экспортных продаж (ставка НДС 0 %).

Работа подсистемы


Раздел Бухгалтерский и налоговый учет отражает настройки и формирование проводок при помощи рабочего места Отражение документов в регл. учете. Механизмы формирования проводок не используют бухгалтерские итоги. Проводки формируются в привязке к первичным документам, что обеспечивает возможность детальной сверки данных бухгалтерских проводок и первичных документов. Также в разделе осуществляется ввод корреспонденции счетов по бухгалтерскому и налоговому учету при помощи документа Операции (регл. учет) и ведения регламентированного бухгалтерского учета и налогового учета по налогу на прибыль в Журнале проводок (регламентированный учет).

Раздел Основные средства отражает текущее состояние в разрезе основных средств, а также данные о государственной регистрации (снятия с регистрации) транспортных средств и земельных участков.

Раздел Лизинг отражает текущее состояние полученных по лизингу основных средств и взаиморасчетов с лизингодателем.

Раздел Нематериальные активы отражает состояние приобретенных и/или созданных результатов интеллектуальной деятельности, а также иных объектов интеллектуальной собственности предприятия.

Раздел Контролируемые сделки - предназначен для формирования уведомлений о контролируемых сделках для их предоставления в органы ФНС России.

Раздел Учет по УСН - предназначена для отражения в учете упрощенной системы налогообложения.

Анализ данных

Корректность отражения хозяйственных операций можно проверить стандартными отчетами регламентированного учета:

  • Оборотно-сальдовая ведомость,
  • Оборотно-сальдовая ведомость по счету,
  • Обороты счета,
  • Обороты между субконто,
  • Анализ счета,
  • Анализ субконто,
  • Карточка счета,
  • Карточка субконто,
  • Отчет по проводкам,
  • Сводные проводки.
