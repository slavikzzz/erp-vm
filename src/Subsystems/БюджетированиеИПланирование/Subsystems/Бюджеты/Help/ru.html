Бюджеты

Назначение подсистемы


Бюджеты позволяют принимать взвешенные управленческие решения, основанные на сводном финансовом планировании и оперативном анализе фактических и плановых результатов финансово-экономической деятельности предприятия.

В общем случае бюджеты можно сгруппировать следующим образом:

  • основные бюджеты (мастер-бюджеты) - бюджет доходов и расходов, бюджет движения денежных средств, прогнозный баланс;
  • операционные бюджеты - бюджет продаж, бюджет производства, бюджет прямых материальных затрат, прямых трудовых затрат и т.д.;
  • вспомогательные бюджеты - бюджет капитальных вложений, кредитный бюджет, бюджеты налогов;
  • дополнительные (специальные) бюджеты - бюджет распределения прибыли, планы отдельных проектов и программ.


Особо следует выделить основные виды бюджетов, так как они являются ключевыми итоговыми формами бюджетного процесса, тесно взаимосвязаны между собой и формируются на основании данных других видов бюджетов. При создании основных бюджетов необходимо соблюдать определенную последовательность работы по их формированию, которая регламентируется внутри системы через настройку Этапов бюджетного процесса по бюджетной модели.

Работа подсистемы


Планирование в подсистеме бюджетирования выполняется документом Экземпляры бюджета. Данный документ формируется на основе определенного Вида бюджета, который определяет бюджетную форму для планирования.

Для анализа данных бюджетирвоания и фактических результатов хозяйственной деятельности используется Бюджетный отчет

Приведем основные направления использования Видов бюджетов:

  • Формирование структуры бюджета для планирования;
  • Формирование отчетного бюджета для сравнения данных разных планов;
  • Формирование отчетного бюджета для проведения план-фактного анализа.


Для установки значений нефинансовых показателей, использующихся при планировании, применяется документ Значения нефинансовых показателей. Значения, устанавливаемые данным дкументом, могут иметь период действия или быть бессрочными.

Анализ данных


Анализ данных бюджетирования и фактических результатов хозяйственных операций производится с использованием бюджетных отчетов:

  • Оборотная ведомость по статьям бюджетов;
  • Оборотно-сальдовая ведомость по показателям бюджетов.
