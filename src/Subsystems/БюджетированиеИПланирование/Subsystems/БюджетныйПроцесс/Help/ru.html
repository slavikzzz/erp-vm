Бюджетный процесс

Назначение подсистемы


Бюджетный процесс – предназначен для организации и координации взаимодействия между участниками с целью обеспечения функций и задач финансового менеджмента, а именно: финансового планирования, анализа и контроля результатов финансово-хозяйственной деятельности предприятия.

Настройка подсистемы


Для включения оповещения пользователям по бюджетным задачам необохдимо включить функцинальную опцию НСИ и администрирование - Казначейство - Использовать оповещения для задач бюджетирования.

Работа подсистемы


Бюджетный процесс выражается в построении план-графика этапов планирования, утверждения, исполнения, анализа бюджетов.

Этапы бюджетного процесса группируются в группы этапов, которые определяют способ выполнения задач: последовательно или параллельно.

На основе расписания, заданного в плане подготовки бюджета, системой автоматически создаются Задачи бюджетного процесса на соответствующих исполнителей.

По мере выполнения одних задач формируются новые задачи по следующим этапам процесса. Нарушение нормативных сроков исполнения текущих задач может привести к сдвигу всех дальнейших задач.

Анализ подсистемы


Для анализа динамики исполнения задач бюджетного процесса используется Монитор бюджетных процессов. С помощью данного рабочего места можно оценить время, требующееся для исполнения последующих невыполненных и неначатых бюджетных задач, что позволяет принимать управленческие решения по оптимизации процесса и увеличении его эффективности.

Основные отчеты:

  • Задачи бюджетного процесса;
  • Статистика по выполнению задач бюджетного процесса.
