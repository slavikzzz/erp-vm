Варианты налогообложения прибыли

Справочник предназначен для определения параметров заполнения декларации при ведении организацией раздельного учета по налогу на прибыль.

Элементы справочника Варианты налогообложения прибыли используются в качестве субконто «Варианты налогообложения прибыли» при отражении документов в регламентированном учете.

Подробное описание приведено в статье Налог на прибыль организаций https://its.1c.ru/db/erpdoc#bookmark:Taxes:TaxOnProfitOfOrganizations
