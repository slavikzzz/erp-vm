Варианты наладки

Варианты наладки предназначены для учета времени переналадок при формировании расписания ключевых рабочих центров.
Варианты наладки создаются в форме, которая доступна при выборе команды Варианты наладки, расположенной в панели навигации формы вида рабочего центра (для вида рабочего центра должен быть установлено Используются варианты наладки.

Цвет оформления

Для варианта наладки можно указать цвет оформления, который используется при формировании расписания рабочих центров.
Цвет оформления позволяет увидеть у каких маршрутных листов одинаковый вариант наладки.

Время работы вида рабочего центра

Время работы вида рабочего центра заполняется, если для вида рабочего применяется синхронная загрузка и применяются варианта наладки.

Длительность переналадки

Длительность переналадки указывается, если при переходе с одного варианта наладки на другой требуется определенное время, которое нужно учесть в расписании.

Длительность переналадки можно указать для следующих случаев:

  • при переходе с указанного варианта наладки на другой указанный вариант наладки;
  • при переходе с любого варианта наладки на указанный вариант наладки;
  • при переходе с указанного варианта наладки на любой другой;
  • при переходе на любой вариант наладки с любых других.
