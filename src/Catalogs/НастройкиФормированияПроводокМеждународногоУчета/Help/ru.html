Настройки формирования проводок (Международный учет)

Справочник Настройки формирования проводок позволяет версионировать применяемые шаблоны проводок или правила настройки счетов международного учета по объектам учета и их корреспонденциям или правила трансляции из регламентированного учета для используемого плана счетов с определенной даты, а также содержит настройки счетов учета для закрытия месяца и расчета курсовых разниц.

Переключатель Корректировки на уменьшение отражать определяет каким образом в периоде действия текущей настройки будут формироваться уменьшающие исправительные проводки при трансляции данных оперативного или регламентированного учетов в проводки международного учета и при отмене проводок международного учета документом Операция (международный учет).
Способ корректировки реверсивными проводками доступен если в системе не используется формирование проводок международного учета по хозяйственным операциям.

Флажок Деление на краткосрочные и долгосрочные активы и обязательства управляет использованием выделения долгосрочных активов и обязательств на отдельных счетах. Деление доступно только при использовании транслятора проводок МФУ по объектам учета и основывается на длительности операционного цикла организации, указываемой в учетной политике финансового учета. Если деление включено, в рабочем месте по настройке счетов учета для объектов учета, поддерживающих деление по срокам, будет предложено указать счета для учета долгосрочной и краткосрочной частей.

Флажок Проводки по выручке без НДС управляет возможностью формировать проводки по выручке без НДС, при этом начисление НДС с реализации выполняется в корреспонденции с расчетами с клиентами.

Можно ознакомиться с описанием в разделе Настройки учета в документации на сайте ИТС https://its.1c.ru/db/erpdoc#bookmark:IFRSReporting:GeneralRegulations.
