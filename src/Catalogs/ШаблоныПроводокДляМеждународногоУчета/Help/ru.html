Настройка шаблона
  • указание Счетов (по умолчанию) для дебета и кредита проводки;
  • выбор Источника уточнения счетов дебета и кредита;
  • выбор Источника заполнения подразделения для дебета и кредита;
  • выбор Источника направления деятельности;
  • выбор Источника валютной суммы;
  • выбор Источника балансовой суммы;
  • переход к настройке уточнения счетов учета по группе финансового учета с возможностью уточнения счета по списку групп финансового учета в справочнике; создания новой Группы финансового учета в справочнике; перехода к форме настройки уточнения счетов в шаблоне, выбора какой-либо группы финансового учета и просмотра списка всех шаблонов проводок, использующих данную группу в своих настройках.
  • использование поля Описание для записи произвольных примечаний.
  • Источник валютной суммы представляет собой сумму хозяйственной операции для заполнения валютной суммы проводки по дебету и кредиту счетов.
  • Заполнение поля Источник валютной суммы является обязательным в шаблонах проводок, при настройке которых выбраны счета с установленным флагом Валютный.
  • Сумма хозяйственной операции, отражаемая по дебету и кредиту счетов бухгалтерской проводки, является балансовой суммой операции.
  • Источник балансовой суммы представляет собой данные аналитических регистров управленческого учета, которые формируются в момент проведения документов в оперативном учете.
  • В случае установленного флага Изменять знак операции при отражении в учете сумма проводки отражается на счетах учета с минусом.

Примечание. В симметричных регистрах (например, Номенклатура - Номенклатура) логика настройки такая:
- Статья соответствует Расходу,
- Кор. статья - Приходу.
То есть при настройке нужно учитывать, что:
- Дебет соответствует кор. сущности,
- Кредит - сущности.
Например, для направлений деятельности необходимо указать:
- Кор. направление деятельности в качестве источника направления для дебета,
- Направление деятельности в качестве источника направления для кредита.

При формировании проводок в международном учете балансовая сумма операции отражается в Функциональной валюте и пересчитывается в Валюту представления.

Закладка Заполнение субконто


Можно включить расширенную настройку заполнения субконто и указать правила заполнения субконто счетов по умолчанию и уточняющих счетов из групп финансового учета.
Способы заполнения: указанными фиксированными значениями или по данным оперативного учета (значениями измерений регистров или их атрибутами).
Без использования расширенной настройки заполнения субконто они будут заполняться автоматически подходящими по типу данных значениями из источника данных оперативного учета (измерениями регистров).

Закладка Дополнительный отбор


На закладке уточняются условия выборки данных оперативного учета. Состав полей, доступных для использования в отборах, зависит от вида хозяйственной операции.
