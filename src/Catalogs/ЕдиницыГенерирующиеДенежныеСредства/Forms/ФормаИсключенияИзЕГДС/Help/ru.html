Исключение из состава ЕГДС

Для исключения активов из состава генерирующей единицы следует отметить их в верхнем списке и нажать кнопку Выбрать.

Выбранные активы добавляются в нижний список.

Далее, следует нажать кнопку Исключить из ЕГДС.

В результате, откроется документ Изменение параметров ОС / НМА с заполненным списком активов и активированным реквизитом Единица, генерирующая денежные средства.

В документе следует указать дату, начиная с которой, активы перестают числиться в генерирующей единице, и провести документ.

Закладка Основные средства - Предназначена для исключения из состава активов генерирующей единицы основных средств.

Закладка Нематериальные активы - Предназначена для исключения из состава активов генерирующей единицы нематериальных активов.
