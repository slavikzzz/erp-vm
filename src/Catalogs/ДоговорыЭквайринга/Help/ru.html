Договоры подключения к платежным системам

Справочник предназначен для ведения договоров с банками по расчетам с использованием эквайринговых терминалов и по системе быстрых платежей (СБП).

Договор обязателен для указания в документах, описывает применяемую учетную схему.

Подробное описание эквайринговых операций приведено в документации https://its.1c.ru/db/erpdoc#bookmark:Acquiring:Acquiring. Описание расчетов по эквайрингу и СБП при оформлении розничных продаж в Рабочем месте кассира (РМК) приведено в разделе https://its.1c.ru/db/erpdoc#bookmark:Retail:Retail.

Возможность учета операций эквайринга определяется установкой функциональной опции НСИ и администрирование – Казначейство и взаиморасчеты – Денежные средства – Оплаты платежными картами / через интернет-эквайринг.
