Группа настроек фин. учета внеоборотных активов

Справочник предназначен для классификации объектов по правилам отражения в бухгалтерском (финансовом) учете.

Заполняется в документах внеоборотных активов. В дальнейшем эта информация используется при настройке счетов учета внеоборотных активов.

Справочник используется в шаблонах проводок при настройке уточнения счетов учета.

  • Вид актива - ограничивает использование настройки для выбранного вида активов и список доступных счетов.
  • Стоимость - определяет счет учета актива, принятого к учету, например, счета 01.01, 04.01.
  • Амортизация - определяет счет учета амортизации актива, например, счета 02.01, 05.
  • Резерв дооценки - определяет счет учета резерва дооценки, например счет 83.01.1. Используется при переоценке актива.
  • Выбытие - определяет счет выбытия, например, счет 01.09. Используется при списании актива.
  • Обесценение - определяет счет, на котором ведется учет обесценения внеоборотных активов например, счета ОБ.01 для ОС, ОБ.04 для НМА. Используется во всех операциях с обесценением.
  • Целевое финансирование - в этой группе определяются счета учета активов приобретенных за счет целевого финансирования.
  • Залоговая стоимость - определяет счет учета арендованных активов, например, счет 001. Используется при поступлении арендованных ОС.
