Договоры аренды (лизинга)

Договоры используются для учета взаиморасчетов по договорам аренды и лизинга.

Взаиморасчеты в оперативном учете ведутся в контуре расчетов по финансовым инструментам.

Проанализировать состояние взаиморасчетов по договорам можно с помощью отчетов:
- Внеоборотные активы - Отчеты - Ведомость расчетов по аренде
- Внеоборотные активы - Отчеты - План-фактный анализ оплат и начислений по аренде.
