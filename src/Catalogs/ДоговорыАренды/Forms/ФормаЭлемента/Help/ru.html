Договор аренды (лизинга)

Договоры используются для учета взаиморасчетов по договорам аренды и лизинга.

Взаиморасчеты в оперативном учете ведутся в контуре расчетов по финансовым инструментам.

Проанализировать состояние взаиморасчетов по договорам можно с помощью отчетов:
- Внеоборотные активы - Отчеты - Ведомость расчетов по аренде
- Внеоборотные активы - Отчеты - План-фактный анализ оплат и начислений по аренде.

Учетная информация

Основное
  • Цель договора - Поддерживаются договоры аренды и лизинга. От цели договора зависит какие параметры договора будут доступны. Доступно, если в организации ведется учет аренды по ФСБУ 25;
  • Стороны договора - Указываются сведения об арендаторе и арендодателе;
  • Номер, дата - Указываются номер и дата договора;
  • Действует - Указывается срок действия договора;
  • Статус - Указывается текущий статус договора (не согласован, действует, закрыт).

Параметры договора
  • Балансодержатель - Для договоров лизинга может быть арендодатель или арендатор, для договоров аренды только арендодатель;
  • Вести учет авансов по текущим арендным платежам в разрезе платежных документов признак ведения по договору учета авансов по текущим арендным платежам в разрезе расчетных документов и регистрации по ним счетов-фактур на аванс
  • Обеспечительный платеж - Признак устанавливается, если по договору предусмотрен обеспечительный платеж (авансовые платежи до принятия к учету предметов аренды);
  • Выкуп предметов аренды - Признак устанавливается, если по договору предусмотрен выкуп предметов аренды. Доступно только для договоров лизинга;
  • Авансирование выкупной стоимости - Признак устанавливается, если по договору предусмотрены авансовые платежи в счет выкупа предметов аренды;
  • Предметы аренды учитываются - Позволяет определить в каком учете будут учитываться предметы аренды. Если предметы аренды не учитываются в одном из учетов, то действуют следующие правила: Принимаются к учету предметы только в том учете, в котором они учитываются в соответствии с договором; Начисление услуг по аренде будет сразу отражаться в расходах того учета, в котором не учитываются предметы аренды, например, если предметы аренды не учитываются в управленческом учете, то услуги будут отражаться в расходах в управленческом учете; Проценты начисляются только в том учете, в котором учитываются предметы аренды. Выбор доступен только для договоров аренды. Для договоров лизинга предметы аренды учитываются в обоих учетах;
  • Способ определения стоимости предметов аренды - Стоимость может определяться путем дисконтирования планируемых платежей (для расчета стоимости нужно ввести график платежей и процентную ставку) или по фактической стоимости (нужно ввести фактическую стоимость). Выбор доступен только для договоров аренды. Для договоров лизинга всегда указывается фактическая стоимость.

Расчеты по договору

На закладке отображается сводная информация о расчетах по договору, если было оформлено заключение договора аренды.
Если документ еще не оформлен, то можно воспользоваться гиперссылкой для его создания.

Учетная информация
  • Группа фин. учета расчетов - Настройка правил отражения операций в бухгалтерском и налоговом учетах. Рекомендуется использовать различные субсчета для счетов учета по договору;
  • Отражение услуг в расходах - Указываются статья, аналитика и подразделение, по которым начисление услуг отражается в расходах. Используется для заполнения значений по умолчанию при начислении услуг;
  • Отражение начисленных процентов в расходах - Указываются статья, аналитика и подразделение, по которым отражается начисление процентов по аренде (лизингу).
