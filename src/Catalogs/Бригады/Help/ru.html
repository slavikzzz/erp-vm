Бригады

Справочник предназначен для описания постоянного или временного коллектива рабочих, выполняющих общие производственные задания и несущих коллективную ответственность за результат своего труда. Бригады указываются в качестве исполнителей видов работ в документах: Этап производства, Производство без заказа для версии управления производством 2.2 и в маршрутных листах для версии 2.1.

Состав бригады определяется документом Выработка сотрудников. Предполагается, что состав бригады, заданный при оформлении выработки, действует, пока не будет оформлен документ выработки с новым составом.

Сумма оплата труда бригады определяется расценками на выполняемые бригадой виды работ. Сумма оплаты каждого члена бригады определяется коэффициентом трудового участия, отработанным временем и тарифной ставкой. Кроме этого сумма оплаты труда каждого члена бригады может быть указана вручную.

Наряду с бригадами, исполнителями производственных заданий могут быть работники. Вид исполнителей, показатели для распределения суммы бригады по исполнителям настраиваются в карточке производственного подразделения.
