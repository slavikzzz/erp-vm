Редактируемое значение

Редактируемое значение - элемент отчета, который организует ввод и хранение произвольных вспомогательных данных для документа Экземпляр бюджета. Значение, введенное в Экземпляре бюджета доступно для формул Производных показателей только в рамках этого документа.

  • Представление - наименование элемента отчета для отображения в дереве элементов в виде бюджета, а так же для отображения в заголовках таблицы бюджета
  • Тип значения - Число, Строка или Булево. Определяет значение какого вида будет вводить пользователь в документе Экземпляр бюджета
  • Комментарий - дополнительное описание, которое не будет отображаться в бюджетном отчете
