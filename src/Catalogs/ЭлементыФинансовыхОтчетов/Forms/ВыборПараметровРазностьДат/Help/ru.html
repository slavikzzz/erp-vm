Выбор параметров разность дат

Форма Выбор параметров разности дат предназначена для настройки параметров функции РазностьДат

  • Начало периода - дата, с которой будет рассчитываться количество периодов
  • Конец периода - дата, до которой будет рассчитываться количество периодов
  • Периодичность - периодичность, количество периодов которой требуется рассчитать

Пример. Можно рассчитать количество периодов от начала отчета до периода группировки, если в поле Начало периода выбрать [Начало периода данных], а в поле Конец периода выбрать [Период группировки]. Такая настройка позволит вычислить среднее значение за прошедшие периоды.
