Выбор измерений итога

Форма Выбор направления суммирования предназначена для выбора измерений, значения по которым будут просуммированы с помощью функции ИТОГ.

Функция ИТОГ позволяет получить сумму значений показателя без учета измерения.

Пример.
Расчет % продаж по виду номенклатуры от общего объема продаж может быть выполнен с помощью такой формулы:
[ПродажиТовара] * 100 / ИТОГ([ПродажиТовара], "Виды номенклатуры")
где
[ПродажиТовара] - операнд Статья бюджетов "Продажи Товаров"
"Виды номенклатуры" - имя измерения, по которому нужно получить итог для расчетов
