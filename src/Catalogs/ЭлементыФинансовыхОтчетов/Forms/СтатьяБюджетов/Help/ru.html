Статья бюджетов

Статья бюджетов - основная составляющая структуры видов бюджетов. Статьи бюджетов формируются из оборотов и оказывают влияние как на другие статьи бюджетов, так и на их остаточные показатели. При этом обороты одной статьи могут изменять несколько других статей или остаточных показателей бюджетов.

В отчет могут быть выведены не все обороты по статье, а только, значения, соответствующие установленным отборам.

  • Представление - наименование элемента отчета для отображения в дереве элементов в виде бюджета
  • Показатель - в отчет могут быть выведены Суммовой или Количественный оборот по статье. В простой таблице можно указать Сумма и количество - тогда система отобразит оба показателя.
  • Гиперссылка Смещение и границы данных - открывает форму настройки периодов источника

Настройки получения данных:

  • Фильтры по Сценарию, Организации и Подразделению из бюджетного отчета. Используются, когда нужно вывести значение статьи бюджетов вне зависимости от настроек отчета. Например, для расчета % оборотов по статье по организации, выбранной в бюджетном отчете в оборотах по всей группе компаний.
  • Исключить данные вводимого документа при расчете - позволяет для расчета производного показателя не учитывать данные статей бюджета текущего документа планирования;
  • Дополнительный отбор - позволяет наложить дополнительный отбор на обороты по статье по значениям аналитических разрезов или измерений
  • Комментарий - дополнительное описание, которое не будет отображаться в бюджетном отчете

Стратегия ограничений прав доступа для операндов формул

В настройках операндов формул, которые задаются в настройках структуры бюджета, возможно указание одного из двух режимов получения данных для операнда формулы. Настройка указывается в разделе Настройки получения данных операнда:

  • Учитывать ограничения доступа к данным - в данный операнд будут выбраны только разрешенные данные;
  • Не учитывать ограничения доступа к данным - в данный операнд будут выбраны все данные, исключая сиситему ограничений доступа к данным. Такая настройка может использоваться при расчете относительных величин, например, клогда требуется определить долю продаж разрешенных организаций в продажах группы компаний в целом.
