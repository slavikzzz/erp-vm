Вид аналитики

Элемент Вид аналитики позволяет вывести в отчет все значения аналитики бюджетирования, по которым имеются обороты по статьям, остатки и обороты по показателям бюджетов или значения по нефинансовым показателям.

  • Представление - наименование элемента отчета для отображения в дереве элементов в виде бюджета
  • Вид аналитики - определяет вид аналитического разреза бюджетирования
  • Гиперссылка Дополнительные поля - открывает форму настройки дополнительных полей, отображаемых в бюджетном отчете
  • Выводить все значения аналитики - позволяет отразить все аналитики из базы данных
  • Комментарий - дополнительное описание, которое не будет отображаться в бюджетном отчете

Пример. На статье бюджетов Продажи товаров ведется учет по Партнерам: В этом случае при формировании бюджетного отчета, в структуру которого добавлены элементы отчета Вид аналитики Партнеры и статья бюджетов Продажи товаров, в отчет будут выведены все Партнеры, по которым введен план.

Источником значений могут быть все статьи и показатели бюджетов, входящие в вид бюджета, либо только выбранные. Для выбора статей и показателей бюджетов требуется установить параметр выбора источников значений в положение Только из указанных источников и выбрать источники значений.

Пример. В виде бюджета для построения отчетов (флаг Использовать для ввода плана снят) в строках выбран вид аналитики Вид номенклатуры, а в колонках выбраны статьи "Продажи" и "Закупки". При установленном источнике значений для вида аналитики Виды номенклатуры статьи бюджетов "Продажи" в отчете будут отображены только виды номенклатуры, по которым есть обороты по статье бюджетов "Продажи".
