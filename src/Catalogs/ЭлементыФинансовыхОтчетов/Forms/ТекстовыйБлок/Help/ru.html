Текстовый блок

Элемент служит для отображения заголовков в видах финансовых отчетов.
В текстовом поле указывается текст заголовка. Могут быть использованы дополнительные параметры заголовка, изменяющиеся в зависимости от параметров Вида финансового отчета или Экземпляра финансового отчета. Для добавления параметров используется подменю Добавить параметр. Состав параметров может быть расширен, если в виде финансового отчета указаны аналитические разрезы. Дополнительные аналитические разрезы финансового отчета также доступны для вывода в заголовки.

  • Наименование - представление текстового блока в структуре элементов вида финансового отчета
  • Комментарий - вспомогательная информация, не влияющая на формирование финансовых отчетов. Отображается в структуре элементов вида финансового отчета
