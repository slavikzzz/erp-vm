Шаблон типовой операции

Перечень шаблонов представляет собой настроенные и сохраненные для повторного использования Операции (международный учет).
Для отражения в учете проводок по хозяйственным операциям, для которых в системе не создано отдельных документов предусмотрена возможность формирования типовых шаблонов операций для регулярного использования.

В форме справочника осуществляется настройка шаблона проводок в зависимости от содержания хозяйственной опреации.

Доступны следующие типы проводок:

  • Реверс,
  • Корректировка,
  • Начисление.
