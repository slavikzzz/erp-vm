Приоритеты

Справочник предназначен для хранения списка приоритетов, определяющих очередность планирования и обеспечения заказов. Приоритеты заказов учитываются при распределении запасов.

Кроме того, приоритет, установленный для Заказа на производство, влияет на очередность планирования итогового графика производства. Заказы с более высоким приоритетом при планировании графика вытесняют заказы с меньшим приоритетом, занимая доступность ВРЦ.

Для приоритета можно указать цвет оформления, который используется при отображении заказов в списке.
Цвет оформления позволяет увидеть у каких заказов одинаковый приоритет.
