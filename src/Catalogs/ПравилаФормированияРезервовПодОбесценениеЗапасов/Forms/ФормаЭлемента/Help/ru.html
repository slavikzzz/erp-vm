Правило формирования резервов под обесценение запасов

Справочник предназначен для хранения информации о настройках начисления резервов под обесценение запасов согласно требованиям ФСБУ 5/2019. В случае если себестоимость товара превышает его чистую стоимость продажи, производится формирование резерва под обесценение на соответствующую разницу. Если сумма накопленного резерва превышает разницу между себестоимостью и чистой стоимостью продажи, производится восстановление резерва.

На форме заполняется информация о правиле формирования резервов:
Организация, для которой действует данное правило.
Приоритет правила. Правило с приоритетом "1" применяются в первую очередь. Если в рамках данной организации введено правило с более низким приоритетом, то оно будет действовать лишь для той номенклатуры, на которую не распространялось правило с высшим приоритетом.
Использовать для бухгалтерского учета - необходимо формировать резервы в регламентированном учете.
Использовать для управленческого учета - необходимо формировать резервы в управленческом учете.
Вид цены. По данному виду цены будет вычисляться чистая стоимость продажи.
Период действия, в рамках которого применяется данное правило.
По ссылке "Отбор не установлен, добавить" можно настроить, в каком разрезе будет производиться формирование резервов по данному правилу. По какой номенклатуре, складу и т.д.
