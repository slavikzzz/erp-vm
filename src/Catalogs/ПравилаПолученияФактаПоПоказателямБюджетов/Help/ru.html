Правила получения фактических данных по показателям бюджетов

Правила предназначены для настройки получения фактических данных по Показателям бюджетов при построении бюджетных отчетов и заполнении Экземпляра бюджета по фактическим данным.

Раздел источника данных определяет, какие источники будут использоваться для получения фактических данных.

Возможные разделы получения фактических данных:

  • Оперативный учет, то источниками данных являются Хозяйственные операции с возможностью определения Источника суммы для этой операции.
  • Регламентированный учет и Международный учет, источниками данных являются остатки по счетам соответствующего плана счетов, при необходимости можно выбрать Корр.Счет Учета и Тип Итога (Оборот, Оборот по Дт, Оборот по Кт).
  • Произвольные данные позволяет настроить произвольную схему получения фактических данных.


Дополнительный отбор позволяет ограничить получаемые данные источника.

Существует два способа определения источников данных для аналитик:

  • При включенном режиме Расширенной настройки заполнения аналитик выбирается способ и значение заполнения каждой аналитики при получении фактических данных.
  • При отключенном режиме - заполнение аналитики определяется системой автоматически, на основании типов данных имеющихся в источнике.
