Группы настроек фин. учета номенклатуры

Справочник предназначен для классификации объектов по правилам отражения в бухгалтерском учете. Может использоваться для объединения номенклатуры, имеющей одинаковые счета учета. Например: покупные товары, материалы, тара и т.п.

Элементы справочника выбираются в карточке позиции номенклатуры. В дальнейшем эта информация используется для определения счетов учета номенклатуры.

Счета учета можно редактировать как непосредственно в элементе справочника, так и в рабочем месте по настройке счетов учета.

Справочник используется в шаблонах проводок при настройке уточнения счетов учета.
