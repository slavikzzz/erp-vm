Графы бухгалтерской отчетности

Справочник предназначен для хранения предопределенного списка граф строки отчетной формы бухгалтерской отчетности.

Примечание. Добавление новых граф для строки не предусмотрено.



Для графы предусмотрена возможность изменения следующих полей, настроек:

  • Наименование;
  • Код графы - порядковый номер графы в форме отчетности;
  • Графа заполняется/Графа не заполняется - переключатель включающий/выключающий возможность заполнения данной графы в отчетности;
  • Год - параметр, указывающий за какой период времени получаются данные для заполнения показателя в графе;
  • Дополнительный алгоритм - настройка для применения специфических алгоритмов при заполнении показателей графы.

Доступны следующие значения настройки "Дополнительный алгоритм":

  • Заполнение значения со знаком минус (-) - при заполнении отчетности вычисленное значение показателя заполняется в графе со знаком "-" ("+" для отрицательных значений).

Примечание. Так как справочник "Графы бухгалтерской отчетности" является поставляемым и преднастроенным для корректного заполнения бухгалтерской отчетности, любое изменение настроек может привести к ошибке или неправильному заполнению отчетности.
