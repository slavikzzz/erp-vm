Вид бюджета представляет собой иерархическую структуру объединенных в группы статьей и показателей бюджетов, а также их аналитик. Существует возможность использования в структуре бюджета нефинансовых показателей для расчета зависимых статей.

Вид бюджета используется для следующих целей:

  • табличного ввода плана в документе Экземпляр бюджета;
  • построения бюджетных отчетов требуемой структуры.

Модель бюджетирования - автоматически заполняется действующей моделью. В списке отображаются виды бюджетов выбранной модели бюджетирования. Для элемента списка доступны действия:

  • Просмотреть вид отчета - будет сформирован Бюджетный отчет на основании вида бюджета, активного в списке.
  • Создать экземпляр бюджета - будет создан Экземпляр бюджета на основании вида бюджета, активного в списке (команда доступна, если в виде бюджета установлен флаг Использовать для ввода плана).
