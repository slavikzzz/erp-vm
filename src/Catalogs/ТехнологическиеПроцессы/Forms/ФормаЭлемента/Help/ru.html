Маршрутная карта

Маршрутные карты используются для ведения детальных схем технологических операций, выполняемых на одном этапе производства (по ресурсной спецификации).

Маршрутные карты создаются в рабочем месте Производство - Нормативно-справочная информация - Маршрутные карты.

Основные маршрутные карты

Производство изделия может описываться одновременно несколькими маршрутными картами.
При этом в подразделении одна из маршрутных карт может быть основной.

Маршрутная карта назначается основной следующими способами:

  • в форме списка маршрутных карт - с помощью команды Назначить основной;
  • в форме маршрутной карты - с помощью команды Назначить основной.

Использование статусов маршрутной карты

Маршрутная карта может иметь следующие статусы:

  • В разработке - черновик маршрутной карты. Могут вноситься любые изменения. Маршрутная карта в этом статусе не должна использоваться в производстве. При возврате маршрутной карты в статус В разработке система предупреждает и предлагает проверить, не используется ли она где-либо;
  • Действует - актуальная действующая маршрутная карта. которая может использоваться в производстве. В этом статусе не допускается правка основной информации маршрутных карт. Для исправления необходимо вернуть в разработку;
  • Закрыта - ранее использовавшаяся маршрутная карта, которая теперь не должна быть доступна для использования. Редактирование закрытых маршрутных карт не допускается.

Применимость

В поле Для вида изделий указывается вид номенклатуры изделия, для изготовления которого описана маршрутная карта. Без заполнения этого поля система не сможет определить, к какому виду относится производимое изделие, так как маршрутная карта может относиться к этапам, необязательно являющимся выпускающими и может не содержать выходных изделий. Указание вида изделий необходимо для настройки автовыбора материалов и ограничения выбора из списка маршрутных карт.
Коэффициент применения в ресурсной спецификации (*) предназначен для возможности описать маршрутную карту, нормированную на количество изделий, отличное от указанного в ресурсной спецификации.
При выборе Указывается вручную коэффициент применения необходимо будет вручную задать в соответствующем этапе ресурсной спецификации, в котором будет использоваться эта маршрутная карта.
При выборе рассчитывается можно указать пропорцию, при помощи которой в этапе спецификации можно будет рассчитать коэффициент применения.

(*) Доступно при включенной настройке Управление производством (версия 2.2) и выключенном управлении версии 2.1

Маршрутная карта описана на изготовление боковой стенки шкафа. Для каждого шкафа должно быть изготовлено 2 стенки. В маршрутной карте указываем, что коэффициент применения рассчитывается как 2 на 1 единицу основного изделия спецификации.
При использовании такой маршрутной карты в спецификации на изготовление 5 шкафов система предложит коэффициент равный 10 (2х5)

Описание процесса производства

Для материалов, выходных изделий и трудозатрат можно указать операцию, на которой они потребляются/возникают.

Производственный процесс в маршрутной карте описывается в виде списка операций.
Последовательность выполнения операций задается при помощи нумерации. В каждой операции указывается ее номер и номер следующей операции. Для завершающего операции указывается номер следующей операции - 0.
Для исключения зацикливания необходимо, чтобы следующая операция всегда имела номер больше, чем предыдущая.
Если необходимо указать, что после операции одновременно могут начинаться сразу несколько операций, этим последующим операциям присваивается одинаковый номер.

Выходные изделия

Продукция

В табличной части указываются изделия, для производства которых составлена маршрутная карта. Первая строчка является основным изделием и выделяется жирным шрифтом.
Все, что описано в выходных изделиях, является минимальной неделимой партией производства по этой маршрутной карте.

Если основное выходное изделие имеет характеристики, но в маршрутной карте характеристика не указана, то такая маршрутная карта будет подходить для таких изделий с любой характеристикой. Для прочих выходных изделий при наличии характеристик их нужно обязательно указывать.

Возвратные отходы

Табличная часть предназначена для дополнительных выходных изделий, стоимость которых фиксирована и определяется в момент выпуска продукции. Такие изделия уменьшают себестоимость продукции на указанную стоимость. Для возвратных отходов указывается Статья калькуляции с типом затрат Возвратные отходы.

Материалы и работы

В табличной части представлен перечень материалов, необходимых для производства выходных изделий.

Если материал или его характеристика не могут быть однозначно определены при составлении маршрутной карты, то можно настроить их уточнение при использовании маршрутной карты на основании свойств вида изделий и его характеристик, либо задать необходимость ручного уточнения. Это делается в настройке автовыбора материала.

Количество материалов, требуемое на все количество выходных изделий, может быть задаваться в явном виде в поле Количество или рассчитываться по формуле.

Номенклатура с типом Набор добавляется в маршрутную карту в виде отдельной позиции. Далее при формировании документов Производственных операций номенклатура распаковывается на комплектующие.

Трудозатраты

В табличной части указываются виды работ сотрудников, необходимые для производства выходных изделий. Количество требуемых трудозатрат можно либо задать в явном виде, либо по кнопке Расчет по формуле настроить формулу для расчета количества аналогично тому, как это может быть сделано для материалов.

Настройка состава маршрутной карты

При необходимости можно изменить состав маршрутной карты с помощью гиперссылки Настроить состав маршрутной карты.
Например, можно указать, что маршрутная карта содержит только операции. В этом случае при заполнении маршрутных листов данные о материалах, трудозатратах будут браться из спецификации заказа на производство, а операции из маршрутной карты.
