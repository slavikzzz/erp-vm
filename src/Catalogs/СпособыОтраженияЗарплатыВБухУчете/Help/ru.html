Способы отражения зарплаты

Справочник предназначен для задания направлений и аналитик отражения расходов на оплату труда в финансовом учете предприятия. В подсистеме расчета зарплаты способы отражения задаются для начислений, сотрудников, регламентированных подразделений. В карточке способа задается направление распределения:

  • Для производственного персонала - вариант распределения по выработке, статьи калькуляции основной зарплаты и страховых взносов.
  • Для прочих сотрудников - статьи и аналитики расходов (активов \ пассивов) основной зарплаты и страховых взносов.

Вне зависимости от направления распределения в способе отражения доступно указание отдельной статьи и аналитики расходов (активов \ пассивов) для взносов на травматизм.

Задание способа отражения в подсистеме расчета зарплаты

Способы отражения, заданные для сотрудника, начисления, подразделения имеют разный приоритет. При заполнении документов Отражение зарплаты в финансовом учете и Начисление оценочных обязательств по отпускам по данным подсистемы расчета зарплаты выбор способа отражения выполняется в следующем порядке:

  • Начисление
  • Сотрудник
  • Подразделение организации
  • Организация

Если способ отражения на верхнем уровне не задан, то используется следующий. Таким образом, наиболее приоритетным является способ, указанный в карточке начисления.

Распределение на производство

Применяется для производственного персонала, выполняющего работы по этапам производства или производству без заказа с оформлением выработки сотрудников. При использовании этого направления распределения расходы по сотруднику в разрезе выполненных видов работ относятся непосредственно на партии производства и являются прямыми производственными затратами. Такие затраты образуют базу распределения постатейных расходов Сумма расходов на оплату труда.

Распределение суммы по партиям производства может быть выполнено по одному из двух вариантов: пропорционально нормативной стоимости видов работ или пропорционально длительности видов работ. Первый вариант наиболее подходит для распределения сдельной оплаты труда, поскольку суммы, начисленные сдельно, коррелируют с суммами по выработке. Распределение по длительности наиболее подходит распределения начисленных сумм по сотрудникам с повременной оплатой труда, поскольку в этом случае оплачивается отработанное время сотрудника без учета расценок.

При выборе направления распределения на производство доступно указание статей калькуляции основной зарплаты и страховых взносов. Если статьи калькуляции не заданы, то сумма будет отражена по статье калькуляции вида работ, указанного при оформлении выработки.

Отражение зарплаты по статье расходов

Применяется для отражения зарплаты не производственного персонала или в случаях, когда выработка производственного персонала не оформляется. При выборе такого режима в карточке способа отражения задаются статьи расходов и аналитики, на которых будет отражена сумма начисления зарплаты и страховых взносов. Далее постатейные расходы будут распределены в соответствии с настройкой статьи. Счет регламентированного учета для отражения в регл. учете определяется подразделением и статьей расходов и настраивается в рабочем месте Отражение документов в регламентированном учете.
Для расходов, относящихся к деятельности с особым порядком налогообложения, предусмотрен отдельный комплекс настроек.

Отнесение на прочие активы / пассивы

При включенной опции НСИ и администрирование - Финансовый результат и контроллинг - Учитывать прочие активы и пассивы в случаях, когда начисления выполняются за счет ранее сформированных резервов или по другим причинам не относятся на расходы предприятия, в способе отражения возможно указание статьи активов / пассивов. Для таких начислений счет регламентированного учета по дебету задается непосредственно в способе отражения. При отражении зарплаты в финансовом учете такое начисление будет отражено в учете прочих активов / пассивов и не будет отнесено на расходы предприятия.

Подбираемые аналитики

При отнесении зарплаты на статьи расходов (активов \ пассивов) аналитики следующих типов могут быть заполнены автоматически:

  • Организация
  • Подразделение
  • Работник
  • Вид страхового взноса (только для взносов)

Подбор аналитики выполняется при отражении зарплаты в финансовом учете. Перечень подбираемых аналитик определяется данными документа отражения. Например, аналитика по работникам будет заполнена, если зарплата принимается к учету в разрезе работников организации.
