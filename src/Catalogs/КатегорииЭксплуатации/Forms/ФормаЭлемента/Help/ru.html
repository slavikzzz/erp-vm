Категория эксплуатации

Классификатор вариантов эксплуатации товарно-материальных ценностей. Предназначен для объединения спецодежды, спецоснастки, инвентаря и прочих хозяйственных принадлежностей, по общим признакам учета в эксплуатации и нормам погашения стоимости

Основные параметры

  • Имущественный учет - Если имущественный учет отключен, то ТМЦ в количественном выражении списываются сразу при передаче в эксплуатацию и дальнейший их учет недоступен (перемещение, списание, возврат);
  • Срок эксплуатации - Указывается количество месяцев, в течение которых, предполагается эксплуатация ТМЦ. В конце срока использования будет предложено списать ТМЦ и заказать новые. Допускается досрочное списание;
  • Учет по инвентарным номерам - При учете по инвентарным номерам каждой единице ТМЦ присваивается номер, по которому она идентифицируется в учете. Инвентарный номер можно сгенерировать автоматически при передаче в эксплуатацию и распечатать;
  • Учет по физ. лицам - Учет по физ. лицам позволяет указывать физ. лицо, у которого учитывается ТМЦ. Если учет отключен, то ТМЦ учитывается только в подразделении;
  • Учитывать в виде группового ОС - По умолчанию стоимость ТМЦ списывается на расходы сразу при передаче в эксплуатацию. Но если стоимость всей совокупности ТМЦ является существенной и не целесообразно ее списывать при передаче в эксплуатацию, то ТМЦ можно учитывать в виде группового ОС. В этом случае стоимость списывается на расходы в течение срока использования при амортизации группового основного средства;
  • Группа ОС - Классификационная группа основных средств, используется при заполнении принятия к учету ОС;
  • Статья - Статья, по которой формируется стоимость группового ОС, используется при заполнении передачи в эксплуатацию;
  • Статья расходов - Статья, по которой списывается стоимость ТМЦ при передаче в эксплуатацию, используется при заполнении передачи в эксплуатацию;

Учет ТМЦ в виде группового ОС

Для учета ТМЦ в виде группового ОС необходимо:

  • В категории эксплуатации включить учет в виде группового ОС;
  • Оформить передачу в эксплуатацию ТМЦ. При передаче в эксплуатацию можно не указывать групповое ОС – в этом случае оно будет создано автоматически;
  • Принять к учету групповое ОС. Можно использовать рабочее место Принятие к учету ОС, в котором будет видно еще не принятые групповые ОС;
  • Оформить выбытие ТМЦ. Можно использовать рабочее место Списание из эксплуатации, в котором отображаются ТМЦ у которых истек срок эксплуатации;
  • При выбытии ТМЦ скорректировать стоимость группового ОС. Можно использовать рабочее место Списание ОС при выбытии ТМЦ, в котором отображаются групповые ОС, по которым есть выбытие ТМЦ.
