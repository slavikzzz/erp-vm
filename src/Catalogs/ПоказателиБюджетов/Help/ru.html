Показатели бюджетов

Справочник предназначен для регистрации остаточных показателей (показателей бюджетов), формирующих прогнозный баланс.

Показатели бюджетов являются расчетными значениями суммы фактических остатков на начало периода и запланированных оборотов по статьям бюджета.

В момент создания связи между статьей бюджета и показателем бюджета определяется, каким образом статья влияет на показатель: увеличивает или уменьшает его целевое значение, и с каким коэффициентом.

Показатель бюджета характеризуется аналитическими измерениями, максимальное возможное количество измерений - 6. Показатель бюджета может быть не только стоимостным, но и количественным. Для этого предусмотрен флаг По количеству. При включении данного флага можно выбрать либо использование определенной единицы измерения, выбрав опцию Указанная единица измерения, либо определить единицу измерения из аналитики при выборе опции Единица измерения определяется аналитикой, аналогично с валютным учетом.

Закладка Настройка типа показателя определяет характер получения данных по показателям. Опция Расчетный позволяет производить автоматизированный расчет остатков, опция Целевой позволяет вводить данные вручную.
