Виды финансовых отчетов

Вид финансового отчета представляет собой созданную и настроенную форму финансовой отчетности.

Для настройки Вида отчета необходимо создать структуру отчета из элементов оформления, группировок и показателей.

Сформировать печатную форму настроенного вида отчета можно по кнопке Проверить вид отчета.

При записи вида финансового отчета производится проверка наличия зацикливания связанных элементов финансовых отчетов.
