Классификатор продукции по видам деятельности (ОКПД)

Общероссийский классификатор продукции по видам деятельности входит в Единую систему кодирования и классификации технико-экономической и социальной информации. Используется для детализации сведений, подаваемых в составе форм статистики.

Добавление элементов из поставляемого классификатора осуществляется с помощью команды Подобрать из классификатора.
