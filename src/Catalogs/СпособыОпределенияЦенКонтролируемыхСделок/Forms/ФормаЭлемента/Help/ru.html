Способ определения цены контролируемой сделки

Данный справочник используется в подсистеме Контролируемые сделки и предназначен для заполнения уведомления о контролируемых сделках.

Каждый элемент этого справочника отражает набор информации, необходимый для заполнения пп. 3-6 раздела II Раздела 1А Уведомления.

Принцип заполнения элемента справочника

В том случае, если в сделке не применяются регулируемые цены (ст. 105.4 НК РФ), для нее должен быть заполнен только пункт 3 указанного раздела, для этого в указанном в договоре способе определения цены должен быть снят флаг Цена является регулируемой.
Если же применяются регулируемые цены, то способ определения цены, указанный в дополнительных данных договора, должен содержать информацию о коде определения цены сделки в соответствии со статьей 105.3 НК РФ.
Прочая информация, такая как код метода ценообразования и источники информации, используемые налогоплательщиком, является справочной и не обязательна для заполнения.
