Строки бухгалтерской отчетности

Справочник предназначен для хранения списка строк бухгалтерской отчетности.

Форма строки бухгалтерской отчетности - это основное место настройки заполнения показателей отчетности.

Примечание. Справочник "Строки бухгалтерской отчетности" является поставляемым, т.е. существует механизм автоматического заполнения, который заполняет справочник всеми необходимыми данными для корректного заполнения отчетности. Работу со справочником рекомендуем производить через настройку загруженных строк отчетности.


Строки бывают двух видов: Основные (например строка 1110 - "Нематериальные активы"), Дополнительные т.е. уточняющие основные (например 11101 - "Нематериальные активы в организации").


Основные настройки производятся на вкладе "Графы и алгоритм заполнения".
Здесь для каждой графы строки можно задать список слагаемых показателя, настроить тип показателя, счета, кор. счета, задать отбор по субконто и его реквизитам.

Примечание. Задание отбора по субконто и его реквизитам могут существенно увеличить время заполнения отчетности, поэтому такой способ настройки является нерекомендуемым.

Доступные типы показателя:

  • СНД - сальдо на начало периода по дебету счета;
  • СНК - сальдо на начало периода по кредиту счета;
  • СКД - сальдо на конец периода по дебету счета;
  • СКК - сальдо на конец периода по кредиту счета;
  • СКДР - развернутое сальдо на конец периода по дебету счета;
  • СККР - развернутое сальдо на конец периода по кредиту счета;
  • ДО - оборот по дебету счета;
  • КО - оборот по кредиту счета;
  • ОБ - оборот между счетами.
