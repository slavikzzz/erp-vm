Узлы объектов эксплуатации

Справочник Узлы объектов эксплуатации предназначается для ведения технологической структуры объекта эксплуатации.

На странице Основные задаются основные реквизиты и параметры учета узла объекта эксплуатации.

Страница Параметры учета наработок предназначается для заполнения свойств показателей наработок данного узла объекта эксплуатации. В случае если заданы соответствующие настройки для перечня контролируемых показателей на этапе классификации, для каждого показателя наработки узла можно задать Источник получения текущих значений наработки и Назначенный ресурс.
