Типы документов

Справочник предназначен для хранения типов документов, подтверждающих применение льготы по НДС. Типы документов указываются в графе 7 Реестра к разделу 7 декларации по НДС

В справочнике есть предопределенные элементы, которые подбираются при автоматическом заполнении документа Формирование записей раздела 7 декларации НДС:

  • Договор;
  • Акт;
  • Накладная;
  • УПД;
  • Справка-расчет.


Справочник можно наполнять новыми значениями, если при подтверждении льготы по НДС используются другие типы документов.
