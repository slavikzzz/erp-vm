Модели бюджетирования

Справочник предназначен для регистрации используемых моделей бюджетирования.

Модель бюджетирования определяет совокупность используемых бюджетных форм и отчетов для ввода и анализа данных, а также определяет систему показателей для целей планирования, контроля и анализа финансово-хозяйственной деятельности предприятия. Модель бюджетирования также определяет регламент бюджетного процесса на предприятии, выражающегося в последовательности действий из планирования бюджетных показателей и статей, установления бюджетных параметров, использующихся для расчетов, утверждения бюджетов, задания лимитов расхода ДС и прочих операций.

Подробное описание Модели бюджетирования приведено на сайте ИТС:

https://its.1c.ru/db/erpdoc#bookmark:BudgetStructure:ModelOfBudgeting

Настройке модели бюджетирования предшествует заполнение справочников:

  • Статьи бюджетов
  • Показатели бюджетов
  • Сценарии
  • Связи показателей бюджетов
  • Правила получения фактических данных


Модель бюджетирования определяет:

  • Использование основных аналитик бюджетов: Организации и Подразделения;
  • Этапы подготовки бюджетов, задающие регламент планирования и контроля по используемым в модели видам бюджетов. Необходимость явного утверждения бюджетов в рамках модели определяется установкой флага Утверждение бюджетов. В этом случае в документе Экземпляр бюджета включается использование статусов, а в рамках бюджетного процесса появляется действие Утверждение бюджетов;
  • Настройки заполнения лимитов расхода ДС по данным бюджетирования.


Модель бюджетирования может определять систему контроля за расходованием ДС. В случае, если используется подсистема бюджетирования, то возможность настройки лимитов расхода ДС определяется функциональной опцией Контролировать превышение лимитов в разделе НСИ и администрирование - Казначейство. Лимиты расхода ДС задаются в справочнике Правило лимитов по данным бюджетирования.

Для модели бюджетирования может быть задано произвольное количество лимитов с разной периодичностью, а также с различной детализацией контроля. При помощи лимитов контролируются статьи бюджетов.

Модель бюджетирования имеет сроки своего действия. Реквизиты Статус и Период действия модели определяют возможность использование модели в интервале планирования. Причем одновременно могут действовать несколько моделей.
В процессе функционирования предприятия может изменяться финансовая структура предприятия, может изменяться система анализа, прогноза, планирования показателей, может изменяться бюджетный регламент, определяющий бюджетный процесс, может изменяться внешняя конъюнктура по отношению к предприятию. Для поддержки разных систем и принципов организации бюджетирования возможно формирование различных моделей бюджетирования. Для поддержки применения различных моделей управления финансами в оперативном режиме возможно одновременное использование нескольких моделей.

Для формирования задач в рамках бюджетного процесса система позволяет формировать задачи автоматически либо с использованием специального документа:

  • Формировать задачи бюджетного процесса автоматически по расписанию - система будет формировать задачи по настроенному расписанию для каждой модели бюджетирования согласно настройкам группы этапов бюджетного процесса
  • Запускать вручную документом Процесс подготовки бюджетов - для запуска бюджетного процесса требуется ввести документ Процесс подготовки бюджетов. Задачи, сформированные на основании первого запущенного процесса, при план-фактном анализе в Мониторе бюджетных процессов будут отображаться в основной ветви плана подготовки бюджетов. Последующие пересчеты будут отображаться в группировке с документом Процесс подготовки бюджетов


Для связи с данными Планирования (планы продаж, закупок, производства) используются настройки в:

  • Сценариях планирования;
  • Видах планов.

Использование моделей бюджетирования в организациях

Для организаций с небольшим числом участников бюджетного процесса


Удобно использовать одну основную модель бюджетирования с ручным запуском бюджетного процесса, а документ Процесс подготовки бюджетов может использоваться как самостоятельное рабочее место, из которого можно создавать бюджеты в необходимом порядке "в один клик".

Для средних организаций

  • Рекомендуется использовать механизмы автоматического формирования задач бюджетного процесса по расписанию
  • В случае изменений после подготовки планов – запускать процесс перепланирования документом Процесс подготовки бюджетов.


Если задачи выполняются интерактивно пользователем – то документ Процесс подготовки бюджетов повторно сформирует задачи пользователям в заданном порядке. В документе можно указать дополнительные сведения, которые будут отражены в описаниях формируемых задач.

Для крупных организаций
  • Для контроля подготовки функциональных бюджетов в подразделениях может использоваться модель бюджетирования с автоматическим формированием задач по бюджетному процессу
  • Для расчетов мастер-бюджетов в разных сценариях с разными экономическими коэффициентами можно использовать модель бюджетирования с ручным формированием задач
