План счетов международного учета

Международный план счетов (Международный финансовый учет - Настройки и справочники - План счетов «Международный») представляет собой описание счетов учета по разделам, приведенным в определенной последовательности, и является общим для всех организаций, учет которых ведется в информационной базе.

Подробное описание приведено в разделе План счетов в документации на сайте ИТС https://its.1c.ru/db/erpdoc#bookmark:IFRSConcepts:IFRSAccountsChart.

Международный план счетов не является предопределенным, он может быть полностью очищен и создан заново.

План счетов, созданный для ведения учета и составления отчетности по международным стандартам, имеет ряд особенностей, отличающих его от Плана счетов бухгалтерского учета :

  • Отсутствие признака активно-пассивного счета на счетах нижнего уровня иерархии. Поэтому расчеты с разными дебиторами и кредиторами учитываются на отдельных для каждого вида задолженности активных или пассивных счетах. Аналогично, для учета других видов активов, обязательств или капитала открываются отдельные счета, которые корреспондируют по дебету или кредиту в соответствии со своим функциональным назначением.
  • Введение дополнительных счетов, которые в настоящее время не используются в регламентированном учете. Например, в плане счетов бухгалтерского учета не предусмотрены отдельные счета для учета расчетов со связанными сторонами или ряда финансовых инструментов.
  • Международный план счетов не закреплен законодательством, а логической основой для его составления являются Отчет о финансовом положении (Баланс) и Отчет о совокупном доходе. Это принципиально отличает данный подход от установленной системы нумерации и наименований счетов в регламентированном учете, в которой частично применяется блоковая система.

Структура Международного плана счетов построена следующим образом: сначала перечислены счета учета статей отчета о финансовом положении (они расположены в порядке своей ликвидности), затем – счета прибылей и убытков.

Составляя отчетность из года в год, предприятие должно соблюдать принцип последовательности представления и сопоставимости данных, определенный п.45 МСФО (IAS) 1 «Представление финансовой отчетности».

С целью реализации данного требования в разработанный план счетов заложена возможность добавления в каждый блок нумерации неиспользуемых (пустых) счетов.

Это позволяет по мере заполнения (настройки) пустых номеров счетов вносить изменения в международный план счетов в соответствии с текущей деятельностью предприятия без его кардинальной перестройки.

При добавлении нового счета необходимо задать его код и наименование, а так же свойства счета:

  • признаки активных, пассивных и активно-пассивных счетов;
  • учет по подразделениям;
  • учет по направлениям деятельности;
  • учет по количеству;
  • валютный учет, исключен ли счет из расчета курсовых разниц в функциональной валюте;
  • запрещено использовать в проводках - для групповых счетов;
  • настройки аналитического учета по субконто.

Настройки аналитического учета - это Виды субконто, которые задаются в качестве свойств счетов. По каждому счету может параллельно вестись аналитический учет с использованием до трех видов субконто. Предоставляется возможность самостоятельно добавлять новые субконто.

По выбранному счету можно получить дополнительную информацию в виде:

  • наименования счета на английском языке - поле Международное;
  • ссылки на нормативный документ (пункт МСФО), который обосновывает включение счета в план счетов - поле Описание.


Вывести на экран и распечатать Международный план счетов можно по кнопке Еще - Вывести список в виде простого списка счетов или списка с подробным описанием каждого счета.

При настройке своего плана счетов обратите внимание, что он должен обеспечивать простое и удобное построение финансовых отчетов, возможность расширения в будущем, детализацию, достаточную для принятия управленческих решений.

Проанализировать данные Международного финансового учета можно с помощью стандартных отчетов:

  • Оборотно-сальдовая ведомость. Пробный баланс (международный учет);
  • Оборотно-сальдовая ведомость по счету (международный учет);
  • Анализ счета (международный учет);
  • Анализ субконто (международный учет);
  • Карточка счета (международный учет);
  • Сводные проводки (международный учет);
  • Шахматная ведомость (международный учет) и других.
