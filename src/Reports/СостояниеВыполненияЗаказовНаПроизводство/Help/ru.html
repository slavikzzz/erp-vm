Состояние выполнения заказов на производство

Отчет предназначен для анализа выполнения графика производства по заказу (заказам) на текущую дату в разрезе этапов и планируемых выпусков подразделений.
В отчете предусмотрена возможность не отображать уже выполненные ветви структуры изделия.

Для отображения списка связанных с выбранным этапом заказа документов предназначена команда контекстного меню Перейти к документам.

По выполненным этапам и полуфабрикатам отображаются отклонения фактических затрат от планируемых по строке заказа на производство. План строится по спецификации заказа, при этом стоимость материалов и работ определяется плановой калькуляцией. Если плановая калькуляция не задана, то стоимость материалов определяется по плановому виду цен на дату потребности. Вид цены плановой стоимости материалов задается в панели администрирования раздела Производство.

По умолчанию на форме отображаются колонки:
-процент отклонения - процент относительно планового значения по этапу (полуфабрикату);
-значимость - процент относительно плановой стоимости строки заказа в целом;

Например, всего плановые затраты по заказу 10000 руб., а по одному из этапов 5000 руб.. На этапе зафиксировано отклонение на 100 руб.
Тогда процент отклонения по этапу составляет 2 % (100 * 100 / 5000), а значимость 1 % (100 * 100 / 10000).

Отображение абсолютных значений плановых и фактических затрат можно настроить в настройках формы.

При двойном нажатии открывается отчет Отклонения фактических затрат от спецификации заказа. Отчет показывает расхождения с точностью до статей калькуляции и конечных затрат.
