Исполнение плана производства

Отчет предназначен для контроля выполнения плана производства продукции.
Позволяет оценить фактическое состояние производства продукции по плану и созданные заказы на производство, которые еще не поступили на склад.
Важно понимать, что этим отчетом учитывается исполнение плана только через Заказы на производство. В качестве подразделений подразумеваются подразделения-диспетчеры. Документы Производство без заказа в этом отчете не учитываются.

Общие принципы работы с отчетами описаны в документации на ИТС https://its.1c.ru/db/erpdoc#bookmark:Reports:ReportSetup.

Особенности отчета:

  • вывод показателей (план, факт, исполнение плана и не отгруженных заказов) по периодичности указанной в сценарии за указанный период;
  • быстрый отбор нерелевантных строк с учетом допустимого отклонения процента исполнения плана по количеству;
  • настройка Только номенклатура из планов определяет, будет ли учитываться отбор по номенклатуре, по которой введены планы за указанный период.
