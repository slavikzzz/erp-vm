Отчет "Анализ РНПТ документов

Отчет предназначен для контроля  состояния проверки РНПТ в документах.

При формировании отчета можно задать параметр "Показывать документы" - возможные значения:

  • Все - учитываются все документы, по которым выполнялась проверка РНПТ,
  • Без ошибок - при построении отчета учитываются документы, по которым не обнаружено ошибок при проверке РНПТ,
  • С ошибками - учитываются документы, по которым были обнаружены ошибки при проверке РНПТ.

Результаты формирования отчета выводятся в разделе Результат
