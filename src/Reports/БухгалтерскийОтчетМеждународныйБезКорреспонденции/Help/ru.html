Бухгалтерский отчет (международный учет)

Отчет предназначен для оценки развернутого сальдо по счетам предприятия.
В случае установленного флага Сумма в тысячах данные отчета выводятся в тысячах.

Доступны варианты отчета:

Анализ взаиморасчетов контрагентов по договорам (международный учет)

Анализ субконто (международный учет)

Анализ счета (международный учет)

Оборотно-сальдовая ведомость. Пробный баланс (международный учет)

Оборотно-сальдовая ведомость по счету (международный учет)

ОСВ по счету с корреспонденцией (международный учет)

Анализ взаиморасчетов контрагентов по договорам (международный учет)
  • отображается общая информация по взаиморасчетам с контрагентами (клиентами и поставщиками, дебиторами и кредиторами).
  • начальное сальдо, обороты и конечное сальдо по всем имеющимся задолженностям контрагентов в разрезе договоров.
  • разбивка по периодам времени: месяц, год и другие, а так же в Функциональной валюте или в Валюте представления.
  • развернутое сальдо (рассчитывается для каждого уровня группировки и по контрагенту в целом).

Анализ субконто (международный учет)


Отчет предназначен для представления данных по выбранному виду субконто: начальное и конечное сальдо, обороты за период по счетам. Отчет можно сформировать не только по выбранному субконто или нескольким субконто, но и по значению субконто или реквизиту значения субконто.

  • детализация по субсчетам (счетам в группе),
  • детализация по периодам времени: месяц, год и другие, а так же в функциональной валюте или в валюте представления,
  • развернутое сальдо (рассчитывается для каждого уровня группировки и по счету в целом).

Анализ счета (международный учет)


Отчет предназначен для представления данных по оборотам между выбранным счетом и всеми остальными счетами за определенный период.

  • детализация по субсчетам (счетам в группе) или по объектам аналитического учета (субконто).
  • детализация по периодам времени: месяц, год и другие, а так же в функциональной валюте или в валюте представления.
  • развернутое сальдо (рассчитывается для каждого уровня группировки и по счету в целом).

Оборотно-сальдовая ведомость. Пробный баланс (международный учет)


Отчет предназначен для формирования оборотно-сальдовой ведомости по всем счетам (субсчетам) за определенный период времени.
Отчет является основой для составления Отчета о финансовом положении (Баланса) по МСФО.

  • строки отчета соответствуют определенному счету или субсчету. Счета упорядочены в порядке возрастания.
  • по каждому счету показаны суммы остатков на начало и на конец периода по дебету и кредиту и суммы оборотов за период в функциональной валюте или валюте представления.
  • развернутое сальдо (рассчитывается для каждого уровня группировки и по счету в целом).

Оборотно-сальдовая ведомость по счету (международный учет)


Отчет предназначен для формирования оборотно-сальдовой ведомости по выбранному счету за определенный период времени.
По составу выводимой информации отчет напоминает фрагмент отчета Оборотно-сальдовая ведомость. Пробный баланс (международный учет).

  • детализация по субсчетам (счетам в группе) или по объектам аналитического учета (субконто).
  • детализация по периодам времени: месяц, год и другие, а так же в функциональной валюте или в валюте представления.
  • развернутое сальдо (рассчитывается для каждого уровня группировки и по счету в целом).

ОСВ по счету с корреспонденцией (международный учет)


Отчет предназначен для представления данных по сальдо и оборотам между выбранным счетом и всеми остальными счетами за определенный период.
По составу выводимой информации отчет похож на Анализ счета (международный учет), но дополнительно раскрывает информацию по субконто выбранного счета.

  • детализация по субсчетам (счетам в группе) и по объектам аналитического учета (субконто).
  • детализация по периодам времени: месяц, год и другие, а так же в функциональной валюте или в валюте представления.
  • развернутое сальдо (рассчитывается для каждого уровня группировки и по счету в целом).
