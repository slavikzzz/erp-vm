Выполнение задач бюджетного процесса

Отчет предназначен для контроля за выполнением задач бюджетных процессов.
Варианты отчета:

Задачи бюджетного процесса

Справка по исполнительской дисциплине

Задачи бюджетного процесса

Отчет позволяет анализировать список задач по бюджетному процессу указанной модели бюджетирования за период по дате создания задач со следующими возможностями отбора:

  • только открытых задач;
  • только просроченных и задач;
  • конкретного исполнителя.

Справка по исполнительской дисциплине

Отчет позволяет анализировать статистику выполнения задач исполнителями по шагам бюджетного процесса указанной модели бюджетирования за период по дате создания задач.
