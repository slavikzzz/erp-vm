Остатки и доступность товаров

Отчет предназначен для контроля состояний запасов товаров и анализа общего и доступного остатка товаров.

Особенности отчета:

Группа колонок Сейчас:

В наличии - общее количество товаров в наличии на складе.
Отгружается - количество товаров, запланированных к отгрузке по заказам (заказы в статусе к выполнению с действием Отгрузить).
В резерве - количество товаров, зарезервированных на складе по заказам на отгрузку с действием Резервировать по мере поступления, Резервировать на складе или по документам реализации товаров к предоплате.
Доступно - количество товаров, свободное для продажи, равно общему количество товаров в наличии за вычетом количества в резерве на складе.

Группа колонок Ожидается:

Поступит - общее количество товаров ожидаемых к поступлению на склад по заказам на поступление и накладным на поступление на ордерный склад.
В резерве - количество товаров из ожидаемых поступлений, зарезервированных по заказам на отгрузку с действием Резервировать по мере поступления.
Доступно - количество товаров из ожидаемых поступлений не зарезервированное по заказам на отгрузку.

К обеспечению - количество товаров, потребность в которых зафиксирована заказами и не обеспечена складским запасом и ожидаемыми поступлениями. Если нет доступного остатка, в эту колонку попадают все заказы с действием К обеспечению, Резервировать по мере поступления.

В отчете предусмотрена расшифровка по заказам, формирующим резерв (поле Заказ на отгрузку).
В отчете предусмотрена возможность детализации группы колонок Ожидается по датам поступлений, через изменение структуры отчета.

Если в программе включено обособленное обеспечение и допускаются назначения без указания заказа (например, включена давальческая схема, производство силами переработчика или есть направления деятельности с учетом затрат), то в отчет, наряду с назначением, можно вывести направление деятельности, партнера/договор с давальцем или переработчиком для которого предназначен обособленный запас.

Заказы на отгрузку с вариантом обеспечения Не обеспечивать, не ограничивают доступность товаров и не отображаются в отчете.

Общие принципы работы с отчетами описаны в документации на ИТС https://its.1c.ru/db/erpdoc#bookmark:Reports:ReportSetup.
