Потребность в материалах по модели графика производства

Отчет вызывается из формы планирования графика производства по заказу и показывает информацию о потребности в материалах, которая возникла бы по модели графика. Эта информация может быть сопоставлена с реальным обеспечением для анализа возможности предпринять какие-либо действия для ускорения выполнения заказа на производство.
