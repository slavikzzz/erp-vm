Сверка расчетов

Отчет предназначен для сверки взаиморасчетов с партнерами, а также для поиска и анализа расхождений образовавшихся по результатам двустороннего обмена данными с партнерами (Наши данные; Данные контрагента),.

Особенности отчета

  • В отчет выводится список всех проведенных документов сверки за указанный период и строки с расхождениями по каждому документу сверки если есть.
  • В группировке первого уровня по организации в колонках Наш долг, Долг контрагента по данным организации и контрагента выводится конечное сальдо по текущей организации рассчитанное по введенным документам сверки.
  • В группировке второго уровня по документу сверки в колонках Наш долг, Долг контрагента по данным организации и контрагента также выводится конечное сальдо по текущему документу. Это конечное сальдо должно совпадать с тем, что выводится в печатной форме акта сверки.
  • В группировке третьего уровня по расчетному документу в колонках Наш долг, Долг контрагента выводятся суммы для конкретного расчетного документа по которому в детальных записях документа сверки указано расхождение.
  • В группировке по расчетному документу выводятся только строки с расхождениями. Если нет расхождений, то колонка Наш долг по данным организации равняется колонке Наш долг по данным контрагента. Для колонок Долг контрагента аналогично.
