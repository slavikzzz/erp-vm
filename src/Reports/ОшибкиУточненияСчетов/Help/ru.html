Ошибки уточнения счетов (международный учет)

Отчет предназначен для контроля ошибок, допущенных в Шаблонах проводок при работе с Группами финансового учета.

Особенности отчета:

  • Отчет информирует об ошибке настройки счетов, если в разных шаблонах проводок, созданных при одном Счете по умолчанию, выбраны разные Группы финансового учета для уточнения счетов.
  • Отчет имеет дополнительные критерии отбора по счету и группе.
