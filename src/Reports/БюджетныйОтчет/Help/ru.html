Бюджетный отчет

Отчет предназначен для анализа данных бюджетирования в соответствии со структурой заданной Видом бюджета.

Особенности отчета:

  • если по Модели бюджетирования, к которой относится Вид бюджета, определено формирование бюджетов по подразделениям и/или организациям, то в отчете может быть установлен соответствующий дополнительный отбор.
  • если в Виде бюджета выбраны Аналитические разрезы бюджета - то можно будет установить фильтр в отчете по значениям таких аналитик.
  • Граница фактических данных - настройка доступна в отчете, если в Виде бюджета настроено смещение при заполнении Статей бюджетов с использованием границы фактических данных. Дата заполняется по умолчанию из настроек Вида бюджета, но может быть указана вручную для иного учета периода выборки фактических данных при расчетах.
  • при использовании в рамках Модели бюджетирования утверждения бюджетов в отчет попадают утвержденные данные. Если необходимо дополнительно выводить данные в статусах К утверждению или В подготовке, то следует установить соответствующие флаги.
  • Отчет может быть сформирован либо в стандартном режиме, либо в режиме сравнения данных по разным датам актуальности плановых данных. Для выбора режима отчета предусмотрена опция Режим формирования, принимающая значения Отчет или Сравнение по датам актуальности.
