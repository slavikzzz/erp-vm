Ведомость расчетов по аренде

Отчет предназначен для контроля обязательств по договорам аренды и договорам лизинга.

Предметы аренды

  • Стоимость - восстановительная стоимость основных средств;
  • Амортизация - начисленная амортизация по основным средствам;
  • Остаточная стоимость - остаточная стоимость основных средств;

Обязательства

  • Начислено - Сумма начисленных обязательств по аренде;
  • Погашено - Сумма погашенных обязательств по аренде;
  • Осталось погасить - Сумма обязательств к погашению;
  • НДС к начислению - Сумма НДС, которую осталось начислить за услуги по аренде;

Обеспечительный платеж

  • Оплачено - Сумма оплаты обеспечительного платежа;
  • Зачтено - Сумма зачета обеспечительного платежа;
  • Осталось зачесть - Сумма обеспечительного платежа, которую осталось зачесть;

Арендные платежи

  • Оплачено - Сумма платежей за услуги по аренде;
  • Начислено - Сумма начисленных услуг по аренде;
  • Осталось оплатить - Сумма, которую осталось оплатить за начисленные услуги по аренде;

Выкупная стоимость

  • Оплачено - Сумма оплат выкупной стоимости;
  • Начислено - Сумма начислений выкупной стоимости;
  • Осталось оплатить - Сумма, которую осталось оплатить в счет выкупа предметов аренды;

Проценты

  • Всего - Общая сумма процентов по договору;
  • Начислено - Сумма процентов, начисленная по графику;
  • Осталось начислить - Сумма процентов, которую осталось начислить;
  • Ставка - Процентная ставка по договору;

Остаток по расчетам - Остаток по всем типам расчетов по договору.
