Настройка уточнения счета для шаблона проводки

Форма предназначена для настройки Уточняющих счетов в Шаблоне проводки.

Форма открывается из Шаблона проводки по ссылке Настроить уточнение счета.

Счета дебета и кредита, настроенные в шаблоне проводки по умолчанию, могут быть уточнены с помощью Групп финансового учета.

Для этого в форме Шаблона проводки необходимо:

  • в поле Источник уточнения счета выбрать справочник группы финансового учета, из которого будет выбираться группа для уточнения счета;
  • перейти по ссылке к форме Настроить уточнение счета шаблона проводки и создать или выбрать нужную группу, а затем задать для нее счет из Плана счетов "Международный".


Справочники Групп финансового учета пополняются при необходимости добавления в настройку шаблона проводки уточняющих счетов.
