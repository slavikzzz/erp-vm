Настройка уточнения счетов

Представляет собой форму вывода списка для просмотра и редактирования всех Шаблонов проводок, использующих выбранную Группу финансового учета в своих настройках.

Для перехода к данной форме из формы Шаблон проводки необходимо:

  • перейти по одной из ссылок Настройка уточнения счета, а затем двойным щелчком мыши перейти к выбранной Группе финансового учета;
  • в открывшемся окне перейти по ссылке Изменить.


Переход к форме так же возможен непосредственно из выбранной группы следующих справочников:

  • Группы финансового учета номенклатуры;
  • Группы финансового учета расчетов;
  • Группы финансового учета денежных средств;
  • Группы финансового учета доходов/расходов.
