Новый пакет 1С:ФинОтчетности

Данная форма открывается при создании нового пакета из формы 1С:ФинОтчетности.

Стартовая форма предназначена для выбора:

  • организации;
  • получателя;
  • отчетного периода;
  • наименование пакета отчетности.

«Создать» - создает новый пакет отчетности.

«Организация» - организация, от имени которой создается новый пакет отчетности.

«Получатель» - получатель, которому отправляется новый пакет отчетности.

«Стрелка вправо» / «Стрелка влево» - кнопки предназначены для выбора отчетного периода. Нажатие кнопок изменяет период, за который может составляться данный отчет.

«Наименование» - вид пакета отчетности в зависимости от цели его предоставления.
