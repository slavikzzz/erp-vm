Журнал документов

Журнал проверки документов - рабочее место для работы со статусами проверки документов (если включена функциональная опция Использовать проверку финансовых документов). По кнопке Проверить для выбранных документов можно изменить статус проверки. Состав выводимых в рабочем месте документов можно определить по гиперссылке Настроить.

Подробное описание приведено в статье Отметка о проверке документов бухгалтером
