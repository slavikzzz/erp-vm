Форма настроек

В форме Настройки налогообложения прибыли для организации, ведущей раздельный учет доходов и расходов для целей налога на прибыль, определяются следующие параметры:

  • Дата начала применения настройки;
  • Организация, для которой выполняется настройка налогообложения прибыли;
  • признак Настройка для обособленного подразделения, если настройка выполняется для обособленного подразделения организации, выделенного на отдельный баланс. При установке данного признака необходимо выбрать подразделение, для которого выполняется настройка, из справочника Организации;
  • Варианты налогообложения прибыли распределяются по подразделениям, объектам учета (номенклатура или статьи доходов) и направлениям деятельности на соответствующих закладках.

Подробное описание приведено в статье Налог на прибыль организаций https://its.1c.ru/db/erpdoc#bookmark:Taxes:TaxOnProfitOfOrganizations
