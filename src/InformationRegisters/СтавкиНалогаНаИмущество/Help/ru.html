Ставки налога на имущество

Регистр предназначен для хранения ставок налога на имущество и налоговых льгот по каждой организации.
Каждая запись настроек имеет определенный период действия. Начало периода задается в форме записи настройки. Данная запись будет действовать до даты, предшествующей дате начала действия следующей записи настроек. Ознакомиться с историей изменения настроек по организации можно при нажатии по гиперссылке История изменений

  • Организация - организация, для которой устанавливается налоговая ставка;
  • Применяется с - дата, с которой налог на имущество рассчитывается исходя из указанной налоговой ставки;
  • Общая ставка налога - налоговая ставка в процентах, применяемая в организации;
  • Ставка налога для движимого имущества, принятого на учет с 1 января 2013 года - налоговая ставка в процентах, применяемая для движимого имущества, принятого на учет с 1 января 2013 года. Поле доступно для ставок, применяемых с 2018 г (не используется с 2019 года);
  • Снижение налоговой ставки - имущество организации облагается по налоговой ставке, сниженной относительно обычно применяемой ставки;
  • Все имущество освобождено от налога - имущество организации освобождается от налогообложения. При выборе этого варианта указывается код льготы - причина освобождения от налогообложения;
  • Движимое имущество, принятое на учет с 1 января 2013 года, освобождено от налога - поле доступно для ставок, применяемых с 2018 г (не используется с 2019 года);
  • Уменьшение суммы налога - уменьшение рассчитанной суммы налога в процентах. При выборе этого варианта указывается процент уменьшения суммы.

Подробное описание приведено в статье Параметры регламентированного учета https://its.1c.ru/db/erpdoc#bookmark:RegulatoryAccountingSettings:ParametersOfRegulatedAccounting
