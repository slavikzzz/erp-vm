Настройка заполнения форм статистики

С помощью формы выполняется настройка заполнения показателей форм статистики.

Для настройки необходимо выбрать строку и настроить отбор данных для заполнения показателя формы статистики.

По некоторым строкам выполняется настройка детализации показателя по кодам ОКВЭД, ОКПД, ОКУН.
При настройке определяется отбор данных для заполнения строки, соответствующей коду из классификатора.

При заполнении строк по кодам классификатора ОКПД может потребовать пересчет из единицы измерения номенклатуры в единицу статистической отчетности.
При необходимости пересчет может осуществляться через единицу отчетов или единицы учета длины, веса, объема площади, заданные для Номенклатуры.
