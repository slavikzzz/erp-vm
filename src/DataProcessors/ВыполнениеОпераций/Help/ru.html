Выполнение операций

Рабочее место предназначено для отражения состояния выполнения операций маршрутных листов, принадлежащих подразделениям с методиками управления Пооперационное планирование и Регистрация операций (без планирования). Для редактирования доступны операции маршрутных листов, находящихся в статусе К выполнению.

В зависимости от методики управления маршрутными листами и способа пооперационного управления, настроенных для подразделения, в рабочем месте требуется:

  • При использовании способа управления Регистрация отклонений (методика Пооперационное планирование) - отражать только отклонения от построенного расписания;
  • При использовании способа управления Регистрация факта (методика Пооперационное планирование) либо методики Регистрация операций (без планирования) - отражать состояние выполнения всех операций маршрутных листов.


При использовании методики управления маршрутными листами Пооперационное планирование, для отражения состояния выполнения операций предварительно требуется рассчитать пооперационное расписание производства.
Операции могут иметь следующие состояния выполнения:

  • Ожидание предшествующих – маршрут находится на одном из шагов, предшествующих данной операции;
  • Начата предшествующая – начато выполнение предшествующей операции маршрута;
  • Можно выполнять – все предшествующие операции маршрута завершены;
  • Выполняется – выполнение операции начато;
  • Завершена – выполнение операции завершено.


При изменении состояния выполнения операций, принадлежащих синхронно загружаемым рабочим центрам, может быть открыта вспомогательная форма группового выбора операций. Форма будет открыта при соблюдении следующих условий:

  • В подразделении используется методика управления маршрутными листами Пооперационное планирование;
  • В списке выделена одна строка;
  • В выбранном интервале времени размещены две и более операции.
