Формирование заказов на передачу в производство по планам

Шаг 1

На первом шаге необходимо выбрать сценарий, по которому созданы и утверждены планы производства. В выбранном сценарии должен быть установлен флаг Создавать по сценарию: Заказы материалов. Дополнительно требуется указать период заказа/поступления и подразделение, по которому созданы планы. Если отбор по подразделению не используется, будут выбраны планы по всем подразделениям.
Режим расчета потребностей по плану - в режиме расчета раздельно по периодам потребности разделены по периодам как и в исходных планах. Если выбран режим расчета нарастающим итогом, тогда количество к заказу рассчитывается за весь период, начиная от заданной даты по текущий период планирования. В таком режиме можно учесть перевыполнение/недовыполнение планов по заказам.
Период заказа - рассчитывается по рабочим дням компании, с учетом сроков исполнения заказов на закупку в способах обеспечения. Позволяет указать стандартный (рассчитываемый) период, например Сегодня, Завтра, Эта неделя, Следующие 7 дней для периодического формирования заказов поставщику, которые требуется создать для выполнения плана.
Период поступления - отбор по периоду планируемого поступления товара на склад. Период указывается в документе плана закупок.
Если заполнена Организация, то она будет подставлена в сформированные заказы.

Шаг 2

На втором шаге в верхней таблице показываются данные планов, соответствующие указанным отборам. Данные планов разбиваются на строки согласно указанным:

  • Периодичности плана (например, если выводятся планы производства на месяц с периодичностью Неделя, то месячный план будет разбит на понедельные интервалы).
  • Подразделению.


В рабочем месте отображается информация:

  • Склад, с которого будет передан материал и цеховая накладная, если используется управление производством 2.2. Все эти данные могут уточняться непосредственно в таблице. Сервисы для заполнения указанных параметров во всех выбранных строках доступны в меню Заполнить.
  • В графе К заказу указывается количество товаров, которое необходимо заказать согласно потребности из планов производства. Количество можно разбить, указав вычитаемое количество, или распределить выделенные строки по правилам.
  • В графе В планах выводится общая сумма планов, запланированных по каждой строке. Возможна детализация до документов, являющихся источниками плановых данных.
  • В графе В заказах выводится общая сумма ранее сформированных заказов, совпадающих по номенклатуре, подразделению, складу и цеховой накладной. Данные выводятся только по заказам с датой отгрузки в указанном периоде.
  • Дата заказа - планируемая дата создания заказа, по умолчанию скрыта. Отображение включается в меню Еще - Изменить форму....
  • Дата отгрузки - период плана в котором запланирована отгрузка. Дата отгрузки по умолчанию является началом периода.
  • Для формирования заказа на передачу в производство - нужно поставить флаг в колонке К заказу. В таблице Заказы к оформлению будет отображена информация о подготовленных заказах.
  • Таблица Заказы к оформлению может быть заполнена при помощи перетаскивания из верхней таблицы.

Шаг 3

При переходе к шагу 3 происходит формирование документов Заказ материалов в производство. На этом шаге можно дополнительно уточнить в документах необходимые данные и провести их.
