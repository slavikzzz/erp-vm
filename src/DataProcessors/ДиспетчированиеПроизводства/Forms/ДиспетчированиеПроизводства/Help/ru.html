Диспетчирование маршрутных листов

Закладка используется для создания и диспетчирования маршрутных листов.

Создание маршрутных листов

Для создания маршрутных листов используется команда Сформировать маршрутные листы.
Команда становится доступной, если есть необходимость в создании.

Маршрутные листы создаются на каждый этап производства. Количество маршрутных листов соответствующих одному этапу определяется в настройке этапа Разбивать маршрутные листы. При необходимости маршрутный лист можно разбить на несколько с помощью команды Разбить маршрутный лист.

Формирование расписания

Для формирования расписания используется команда Сформировать расписание, которая открывает специальное рабочее место.

Оформление связанных документов

При выполнении маршрутных листов необходимо оформить следующие документы:

  • Выработка сотрудников - оформляется при выполнении маршрутного листа, если указано какие бригады выполняют работы;
  • Выпуск продукции и выполнение работ - оформляется после выполнения маршрутного листа.

Если на основании маршрутного листа нужно оформить указанные документы, то в соответствующей колонке будет выведен индикатор.
Создать необходимый документ можно щелчком на индикаторе или при помощи команды расположенной в подменю Задачи.

Индикаторы

Для маршрутных листов выводятся различные индикаторы, которые позволяют определить наличие проблем или показывают справочную информацию.

  • Не сформировано расписание;
  • Дефицит материалов в подразделении - Красный индикатор показывает, что хотя бы одного материала не хватает. Желтый индикатор показывает, что в подразделении количество материалов не меньше, чем требуется для выполнения текущего маршрутного листа, но меньше, чем требуется всем маршрутным листам. Отсутствие индикатора показывает, что в подразделении нет дефицита материалов;
  • Предыдущие этапы не завершены;
  • Прохождение точек маршрута - по мере выполнения маршрутного листа требуется отмечать прохождение следующих точек маршрута: начато выполнение маршрутного листа, готовность к работе ключевого рабочего центра, завершена работа ключевого рабочего центра, завершено выполнение маршрутного листа. Здесь же выводится статус буфера;
  • Требуется оформить выработку сотрудников;
  • Требуется оформить выпуск продукции или выполнение работ.

Диспетчирование рабочих центров

На закладке отображаются сведения о рабочих центрах, которые являются ключевыми в выбранном периоде.
Ключевые рабочие центры определяются автоматически на основании графика производства по наибольшей относительной загрузке - отношении времени занятости к времени доступности загружаемых видов рабочих центров.

Список рабочих центров

Для рабочих центров выводится следующая информация:

  • Запланировано, ч - объем работ, который должен быть выполнен на текущий момент времени согласно расписанию;
  • Выполнено, ч - объем работ, который фактически выполнен на текущий момент времени и отклонение от расписания (отставание, опережение, без отклонений);
  • В работе маршрутные листы - показывает какие маршрутные листы в текущий момент в работе на этом рабочем центре;
  • Дата сведений - фактическая дата завершения работ последнего маршрутного листа;
  • Подразделение - подразделение, в котором выполняется производство.

График занятости рабочих центров

Показывает на диаграмме Ганта график занятости рабочих центров на основании сформированного расписания работы рабочих центров.
