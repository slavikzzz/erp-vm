Помощник подготовки уведомления о контролируемых сделках

Рабочее место предназначено для подготовки уведомлений о контролируемых сделках в проверяющие органы. Во время работы с помощником можно пройти все этапы формирования уведомления: от его создания, до выгрузки данных в электронном формате.

Признание сделки контролируемой влечет обязанность налогоплательщика уведомить налоговые органы. Уведомление готовится и подается в налоговый орган по месту нахождения (жительства) налогоплательщика не позднее 20 мая года, следующего за годом совершения контролируемых сделок (ст. 105.16 НК РФ). Формат и порядок заполнения уведомления налоговых органов о таких сделках утверждены приказом ФНС России от 27.07.2012 № ММВ-7-13/524@.
Возможность подготовки уведомления о контролируемых сделках определяется функциональной опцией НСИ и администрирование – Регламентированный учет - Управление параметрами ведения регламентированного учета - Контролируемые сделки.

Работа с помощником реализована удобным механизмом, позволяющим зарегистрировать необходимые дополнительные данные для каждой сделки, попадающей под определение контролируемой, сформировать и проанализировать сводную информацию и на выходе получить уведомление о контролируемых сделках, готовое для передачи в ИФНС.

Для того чтобы отразить в программе наличие контролируемых сделок в отчетном периоде не нужно «поднимать» первичные документы и что-то в них изменять. Все дополнительные данные, предназначенные только для определения и анализа контролируемых сделок, вводятся в специализированные справочники и регистры программы, не затрагивая основной регламентированный учет.

Эту информацию можно вводить в программу как в течение года, так и непосредственно перед формированием уведомления.

Подробное описание приведено в статье Контролируемые сделки https://its.1c.ru/db/erpdoc#bookmark:ControlledTransactions:ControlledTransactions

Дополнительная информация на ИТС

Уведомление формируется от имени каждого юридического лица отдельно, поэтому в форме Помощника, помимо отчетного года, в первую очередь указывается организация, от имени которой формируется уведомление.

Работа с помощником требует наличия документа Уведомления о контролируемых сделках за текущий год и по текущей организации, поэтому если такого уведомления ранее не было создано - по соответствующей гиперссылке его можно будет создать, а выполнение каких либо действий будет заблокировано (программа будет требовать создания уведомления).

Помимо создания уведомления, функционал предоставляет возможность создания корректировки текущего уведомления (по текущей организации и отчетному периоду), или открыть список уведомлений для просмотра.

Работа с помощником сгруппирована в 4 этапа, которые необходимо выполнять последовательно, однако если сведения для какого-либо этапа не требуют корректировки, то его всегда можно пропустить.

Установка условий включения сделок в разряд контролируемых


На этом этапе задаются условия для включения сделок в состав контролируемых.

К числу контролируемых сделок, согласно статье 105.1 НК РФ, могут относиться сделки с взаимозависимыми лицами, с контрагентами, зарегистрированными в офшорах и внешнеэкономические сделки с товарами мировой биржевой торговли.

Для того чтобы выделить из всего объема хозяйственных операций предприятия сделки, которые могут подлежать контролю, в программу необходимо внести информацию о том, кто из контрагентов является для указанной организации взаимозависимым лицом, кто зарегистрирован за рубежом, а также перечислить товары мировой биржевой торговли.

Заполнение взаимозависимых лиц


Для хранения списка взаимозависимых лиц предназначен одноименный регистр сведений Взаимозависимые лица. Помимо этого, для того чтобы классифицировать тип контролируемой сделки, нужно уточнить, является ли взаимозависимое лицо плательщиком одного из указанных в статье 105.14 НК РФ налогов и зарегистрирован ли он в особой экономической зоне, эти данные хранятся в регистре сведений Данные по контрагентам контролируемых сделок. И те и другие данные можно просмотреть и по необходимости отредактировать в специальной форме, которая открывается из помощника.

Тип взаимозависимости может быть выбран из следующих вариантов:

  • Не взаимозависимы;
  • Взаимозависимы по Кодексу (п.2 ст.105.1 НК РФ);
  • Самостоятельное признание взаимозависимости (п. 6 ст. 105.1 НК РФ);
  • Взаимозависимы по решению суда (п. 7 ст. 105.1 НК РФ);
  • Не взаимозависимый посредник (п. 1.1 ст. 105.14 НК РФ).

Взаимозависимость с контрагентом устанавливается на определенный период времени.

Например, ООО «Лютик» было взаимозависимо с организацией ООО «Веселый ветер» в 2012 году, а с середины 2013 года таковым уже не будет. Для этого нужно создать две записи регистра Взаимозависимые лица для контрагента ООО «Лютик», в одной из которых указать дату применения - 01.01.2012 и нужный тип взаимозависимости, а в другой - 01.07.2013 и тип взаимозависимости - Не взаимозависимы.

Классификация товаров для контролируемых сделок


В соответствии с п.1 ст.105.14 НК РФ, контролю могут подлежать сделки в области внешней торговли товарами мировой биржевой торговли. Список товаров мировой биржевой торговли можно задать по гиперссылке Классификация товаров для контролируемых сделок.

По нажатию на эту гиперссылку откроется форма изменения реквизитов контролируемых сделок, с параметрами изменения справочника номенклатуры по реквизитам из следующего списка:

  • Код ТНВЭД - ссылка на справочник Классификатор ТНВЭД;
  • Код ОКВЭД - ссылка на справочник Классификатор видов экономической деятельности;
  • Код ОКП - ссылка на справочник Общероссийский классификатор продукции;
  • Облагается НДПИ по процентной ставке - булевское значение, определяющее является ли предмет сделки полезным ископаемым для которого применяется налог на добычу полезных ископаемых.

Флажок перед наименованием кода ТНВЭД, при открытии формы из данной гиперссылки, показывает, что имеющие данный код номенклатурные позиции относятся к товарам мировой биржевой торговли.

Страны регистрации контрагентов для контролируемых сделок


Поскольку контролируемыми являются не только сделки между не взаимозависимыми лицами, но и сделки, один из участников которых зарегистрирован в оффшорной зоне (оффшорная зона - государство или его часть, в пределах которой для компаний-нерезидентов определяется особый режим регистрации и деятельности), такую информацию тоже нужно отразить в программе. Кроме того, в программе обязательно нужно отметить контрагентов-иностранцев, предметами сделок с которыми являются товары, перечисленные в пункте 5 статьи 105.14 НК РФ. Данная информация указывается в карточке контрагента (поле Страна регистрации).

На данном этапе работы с помощником, можно проверить список контрагентов по странам регистрации, нажав на гиперссылку Страны регистрации контрагентов контролируемых сделок.

По нажатию на эту гиперссылку откроется форма изменения реквизитов контролируемых сделок, с параметрами изменения справочника контрагентов по реквизиту Страна регистрации.

Флажок перед наименованием страны регистрации, в данном случае, будет показывать, что данная страна является оффшорной зоной.

Заполнение данных о контролируемых сделках в договорах


Для сделок, относимых к контролируемым, предусмотрена возможность указать общие данные по сделке через настройку параметров договоров, которые были использованы для отражения сделок в информационной базе. Заполнение доступно в отдельной форме по гиперссылке Заполнение данных о контролируемых сделках в договорах.

По нажатию на эту гиперссылку откроется форма списка дополнительных сведений по договорам участников контролируемых сделок, с возможностью изменения доп. реквизитов договоров с контрагентами, между организациями, кредитов и депозитов и лизинга по реквизитам из следующего списка:

  • Код наименования сделки - справочная информация необходимая для предоставления уведомления в налоговые органы, список кодов предопределен, если не задан будет принимать значения 019 (для работы или услуги) и 015 (во всех остальных случаях);
  • Вид налогообложения - информация влияет на отнесение сделки к разряду спецрежимов, в случае когда сделка и одно из лиц совершающих сделку облагается ЕНВД, данная сделка будет относиться к спецрежиму и проверка будет происходить по соответсвующей контрольной сумме из регистра сведений "Границы контролируемости сделок";
  • Код условий поставки - справочная информация для предоставления уведомления в налоговые органы, список кодов предопределен, возможно пустое значение;
  • Способ определения цены сделки - ссылка на справочник Способы определения цен контролируемых сделок;
  • Стороны применяют разные ставки налога на прибыль - если ставки по налогу на прибыль у участников одной сделки различаются, то такую сделку следует включать в уведомление;
  • Сделка относится к деятельности нового морского месторождения - если у одной из сторон сделки доходы и расходы учитываются по признаку добычи углеводородного сырья на новом морском месторождении, то такую сделку нужно включить в уведомление;
  • Сделка облагается налогом на доп. доход от добычи углеводородного сырья - если сделка заключена в рамках этого признака, она будет учитываться в уведомлении.

Формирование списка контролируемых сделок

Предварительно заполняется список потенциально контролируемых сделок - все те сделки, которые могут попасть в уведомление, удовлетворяющие условиям взаимозависимости, являющимися посредниками, или зарегистрированные за рубежом. При этом условие на превышение общей суммы по итогам годам еще не проверяется.

Формирование списка потенциально контролируемых сделок


Для получения списка контролируемых сделок (по кнопке Сформировать список сделок) документы информационной базы будут проанализированы на соответствие требованиям ст.105.14 НК РФ. Если по ранее указанным настройкам возможно определить контролируемую сделку, то она будет добавлена в список.

Является аналогом команды Сформировать список, вызванной из документа Уведомления о контролируемых сделках.

На основании учетных данных системы формируется список потенциально контролируемых сделок (без проверки на контрольные суммы в течении года).

Включение прочих сделок в состав контролируемых


При необходимости автоматически сформированный список можно дополнить ручной регистрацией сделок. Пополнение списка контролируемых сделок выполняется в отдельной форме Прочие контролируемые сделки, которая открывается по гиперссылке Включение прочих сделок в состав контролируемых. В форме показываются только сделки, относящиеся к тому уведомлению, с которым сейчас идет работа.

Является аналогом гиперссылки "Прочие контролируемые сделки", вызванной из формы документа "Уведомления о контролируемых сделках".

Позволяет включать другие сделки в состав контролируемых с помощью документа Прочие контролируемые сделки.

Просмотр предварительного списка контролируемых сделок


Предварительный список сделок, относимых к разряду контролируемых, можно проанализировать по гиперссылке Список контролируемых сделок.

По нажатию на данную гиперссылку откроется отчет Контролируемые сделки.

Проверка данных для подготовки уведомления о контролируемых сделках


Для проверки ранее введенных сведений о взаимозависимых лицах и предметах контролируемых сделок доступны одноименные отчеты, которые вызываются по гиперссылкам Сведения о взаимозависимых лицах и Сведения о предметах контролируемых сделок.

Просмотр и редактирование сведений о взаимозависимых лицах контролируемых сделок


На этом этапе можно посмотреть отчет Сведения о взаимозависимых лицах. В нем показываются все юридические лица, участвующие в сделках. При необходимости можно открыть карточку контрагента или форму с данными по взаимозависимому лицу и уточнить параметры настройки.

Просмотр и редактирование сведений о предметах контролируемых сделок


На этом этапе можно посмотреть отчет Сведения о предметах контролируемых сделок. В нем отражаются все предметы (номенклатурные позиции), участвующие в сделках. При необходимости можно открыть карточку номенклатуры или форму с данными по предмету контролируемой сделки. В форме Форма изменения данных по предмету сделки для выбранной номенклатуры можно назначить признак уплаты НДПИ (Налог на добычу полезных ископаемых) и указать коды согласно классификаторов ОКВЭД, ТНВЭД, ОКП. Внесенные данные будут сохранены в элементе списка номенклатуры.

Подготовка уведомления о контролируемых сделках


После того, как все дополнительные сведения о сделках заполнены, можно приступать к формированию уведомления.
На этом этапе происходит окончательное формирование уведомления о контролируемых сделках: проверяются контрольные суммы, анализируется результат, формируются печатные формы и выгрузке в электронном формате.

Просмотр и редактирование сведений о границах включениях сделок в уведомление


Включению в уведомление подлежит только та часть из всех выбранных на предыдущих этапах контролируемых сделок, по которым сумма доходов превышает установленные статьей 105.14 НК РФ границы.

В программе информация о границах включения сделок в уведомление на каждый отчетный год хранится в регистре сведений Границы контролируемости сделок. Он заполняется автоматически в соответствии с налоговым кодексом при начале работы с информационной базой (или при обновлении), их ручное изменение не требуется (хотя возможно).

Форму для просмотра границ, действующих на отчетный год, за который формируется уведомление, можно открыть из Помощника, нажав на гиперссылку Границы включения сделок в уведомление. Данные, указанные в форме, действуют на отчетный год на все уведомления.

Просмотр контролируемых сделок


Для проверки перечня сделок, которые попадут в уведомление предназначен отчет Контролируемые сделки для включения в уведомление. Информация в данный отчет выводится в разрезе контрагентов, аналитики (может быть представлена как в виде соглашений, так и договоров в зависимости от хозяйственной операции) и предметов сделок, сгруппированная по факту включения в уведомление и виду контролируемых сделок:

  • Торговля товарами мировой биржевой торговли;
  • По признаку применения спец. режимов (ЕНВД) (пп. 3 п. 2 ст. 105.14 НК РФ);
  • По признаку применения спец. режимов (ЕСХН) (пп. 3 п. 2 ст. 105.14 НК РФ);
  • По признаку уплаты налога на прибыль (пп. 4 п. 2 ст. 105.14 НК РФ);
  • По признаку уплаты НДПИ (пп. 2 п. 2 ст. 105.14 НК РФ);
  • По признаку особой экономической зоны (пп. 5 п. 2 ст. 105.14 НК РФ);
  • На общих основаниях (пп. 1 п. 2 ст. 105.14 НК РФ);
  • Сделка с иностранным взаимозависимым лицом.

Настройка заполнения уведомления


Уведомление представляет собой список листов раздела 1А. Каждый такой лист – это отдельный документ. Список листов, входящих в соответствующее уведомление можно открыть из формы помощника (гиперссылка Состав уведомления (листы 1А и 1Б)). Из списка можно добавлять новые листы, помечать листы на удаление, а также редактировать любые их параметры. Листы раздела 1Б, указываются в соответствующей табличной части документа Лист раздела 1А.

Перед заполнением уведомления можно указать вариант формирования листов 1Б и код места представления. Эти данные доступны по гиперссылке, расположенной перед кнопкой Заполнить уведомление. Название гиперссылки отражает действующие значения рассматриваемых параметров.

Заполнение уведомления


Уведомление формируется по кнопке Заполнить уведомление. Если уведомление уже было сформировано – то все старые листы будут помечены на удаление, и будут созданы новые. При этом, если уже существовал лист 1А с такими же свойствами, он будет использован повторно.

Является аналогом команды Заполнить уведомление, вызванной из формы документа Уведомления о контролируемых сделках.

Проверка заполнения уведомления


Является аналогом команды Проверить, вызванной из документа Уведомления о контролируемых сделках.

Ввод дополнительных сведений о контролируемых сделках


Уточнить сведения о сделках, включенных в состав уведомления можно по гиперссылке Дополнительные сведения о контролируемых сделках. В форме Настройка дополнительных сведений о контролируемых сделках для выделенных элементов новые значения реквизитов задаются по кнопке Установить новые значения.

Просмотр состава уведомления


Посмотреть список получившихся документов Контролируемая сделка (листов А1), включенных в состав уведомления можно по гиперссылке Состав уведомления. В открывшемся списке будут представлены листы уведомления в разрезе взаимозависимых лиц и предметов сделки. При необходимости можно вывести на печать каждый отдельно взятый лист.

Анализ уведомления


Отчет Анализ уведомления (вызывается по одноименной гиперссылке) позволяет представить уведомление в удобном для анализа и редактирования виде. Доступны различные группировки и отборы, а также вывод дополнительных сведений. Строки отчета расшифровываются, из него можно открыть соответствующий документ Лист раздела 1А и отредактировать.

Выгрузка уведомления в электронном виде


По кнопке Выгрузить уведомление создается файлы в формате XML, который соответствует утвержденному ФНС формату

При выгрузке нужно указать каталог размещения сформированного файлов.

Передача уведомления в налоговые органы по каналам связи будет реализована после утверждения форматов документооборота.

Печать уведомления

По кнопке Печать предусмотрена возможность вывести уведомление на печать в регламентированном виде.
