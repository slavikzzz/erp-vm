Форма взаимозависимые лица

Данная форма предназначена для просмотра и редактирования регистров сведений "Взаимозависимые лица" и "Данные по контрагентам контролируемых сделок". Сведения этих регистров используется для определения какие сделки являются контролируемыми, для последующего включения их в уведомление.

При открытии организация и отчетный год, по которым показываются данные взаимозависимых лиц, берутся из данных основной формы обработки помощника. При желании их можно менять.

В список будут попадать все взаимозависимые лица, указанные для рассматриваемой организации в пределах рассматриваемого года. Если в течении года тип взаимозависимимости менялся или менялись вспомогательные данные контролируемого лица, эта информация так же будет отображаться на форме.

Если ранее взаимозависимые лица сейчас стали не взаимозависимыми, необходимо добавить еще одну запись, в которой выбрать тип Не взаимозависимы и дату, начиная с которой контрагент перестает быть взаимозависимым с организацией.

Если список взаимозависимых лиц одной организации во многом повторяет список взаимозависимых лиц другой организации, можно воспользоваться командой Заполнить из организаций, которая поможет выбрать необходимые данные для заполнения и скопировать взаимозависимых лиц из какой-либо организации в текущую.
