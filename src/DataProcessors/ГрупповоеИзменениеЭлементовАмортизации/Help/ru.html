Групповое изменение элементов амортизации

Обработка предназначена для групповой обработки множества ОС.

Позволяет загрузить из табличного документа или подобрать из справочника объекты, по которым нужно изменить элементы амортизации, и массово создать документы Изменение параметров ОС и Корректировка стоимости и амортизации ОС.

Можно выгрузить остатки ОС в табличный документ, указать там новые остаточные сроки, ликвидационную стоимость и загрузить назад через обработку.

Для небольшого количества ОС можно заполнить список через Подбор.
При подборе будут автоматически заполнены текущие значения элементов амортизации ОС. Их необходимо будет изменить вручную на пересмотренные значения.
