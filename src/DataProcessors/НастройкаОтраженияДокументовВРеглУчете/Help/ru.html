Настройка отражения документов в регл учете

Рабочее место предназначено для настройки отражения операций документов в проводках регламентированного учета и для непосредственного отражения документов в соответствии с указанными настройками.

Рабочее место позволяет настроить порядок отражения документов в регламентированном учете заранее вручную, либо указывать необходимые счета по мере появления новых операций. В последнем случае новые строки для отражения будут добавлены автоматически и счета, необходимые для указания, будут выделены.

Настройка осуществляется в разрезе:

  • аналитик учета
  • мест учета

Не рекомендуется изменять счета учета для уже настроенных операций, т.к. подобное изменение затронет не только новые документы, но и уже отраженные в регламентированном учете, если для них потребуется переформирование проводок. В этом случае счета будут установлены в соответствии с новой настройкой.

Механизм формирования проводок построен таким образом, что для каждой отдельной операции не требуется указание счетов учета и значений субконто. Необходимо настроить порядок отражения операций.
Проведение документа и отражение этого документа на счетах регламентированного учета разнесено по времени, т.е., если для какой-то операции еще не настроено отражение в регламентированном учете, это не помешает проведению документа и формированию движений по другим системам учета.
