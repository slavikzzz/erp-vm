Назначение рабочего места

В рабочем месте оформляются и пересчитываются планы.
Рабочее место доступно для сценариев планирования, в которых указана настройка Управление процессом планирования. В таком сценарии можно описать процесс планирования с помощью этапов. Структура планирования сценария по этапам и видам планирования будет отображаться в рабочем месте.

Структура рабочего места

Рабочее место разделено на три части:

  • Слева видна структура планирования сценария по этапам и видам планирования. Для периода отображается минимальный статус, указанного в нём документа. Специальной пиктограммой помечаются планы требующие пересчета, проверки корректировок. При позиционировании на этапе, справа выводится детализация товаров по периодам. Внизу отображается Баланс планов. Он выводит итоговые данные по периоду, показывает начальный и конечный остаток, позволяет оценить сбалансированность планов в заданном периоде.
  • Справа отображается детализация по товарам: остатки, количество прихода и расхода, потребность и её корректировка, количество к изменению. Для каждого вида плана (этапа) отображаются колонки в соответствии с его прикладным значением. Выбор в полях Приход и Расход откроет детализацию планов по количеству этого вида плана и периода. При необходимости можно скрыть детализацию и работать только с видами планов по периодам.
  • Помощник планирования: подсказывает какие данные следует обновить или актуализировать для сбалансирования планов. Информирует пользователя о вводе планов остатков, планов формирующих потребность и вычисляет очередь пересчета планов. Пересчитать все планы по очереди можно командой Пересчитать по дефицитам все планы. При необходимости можно скрыть помощник планирования.

Процесс ввода и пересчета планов
  • Выбор в пустом поле приведет к созданию плана.
  • Выбор в раскрашенном поле откроет детализацию планов по виду плана и периоду. На форме детализации есть возможность создать новый план и изменить статусы планов.
  • Выбор в поле с индикатором пересчета приведет к пересчету плана по дефицитам. Дефициты рассчитываются при изменении потребности при пересчете или вводе исходного или обеспечивающего плана.
