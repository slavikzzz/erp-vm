Получение и возврат материалов

При оперативной работе основная задача заключается в том чтобы обеспечить необходимый уровень запасов материалов в производстве: чтобы материалов хватало для выполнения маршрутных листов и чтобы не было излишков.

Если передача материалов осуществляется не оперативно, например, в конце месяца создается один документ по всем материалам, которые были израсходованы, то нужно:
- для материалов указать способ получения По заказу на передачу в производство
- в конце месяца оформить накладную на отрицательный остаток в подразделении (есть соответствующая команда).

Если для материала указан способ получения По заказу на передачу в производство, то он будет всегда отображаться в этом списке.
Если для материала указан способ получения По заказу на производство, то он будет отображаться в этом списке, если следующие колонки не пустые: Остаток в подразделении, Отгрузить со склада, Текущий излишек (дефицит), Планируемый излишек (дефицит).

По материалам отображаются следующие данные:

  • Остаток в подразделении - остаток материалов в производстве. Положительное значение указывает на объем полученных, но не списанных на выпуски материалов. Отрицательное значение показывает ситуацию, когда материал списали на выпуски, но не было оформлено получение со склада;
  • Остаток на складе - свободный остаток материала на складе;
  • Отгрузить со склада - количество материалов, которое нужно получить на основании распоряжений (на основании заказов материалов в производство и заказов на производство);
  • Текущий излишек (дефицит) - содержит расчетное значение, которое определяется по формуле + - . Положительное значение говорит об излишке материалов - скорее всего, такие материалы можно вернуть на склад или отменить заказы. Отрицательное количество говорит о дефиците материалов - для таких материалов требуется разрешить отгрузку со склада (в заказах установить действие Отгрузить);
  • Запланировано к получению - запланированное к получению количество материалов по заказам на производство и по заказам на передачу материалов;
  • В резерве - Зарезервированное количество материалов по заказам на производство и по заказам на передачу материалов;
  • Планируемый излишек (дефицит) - содержит расчетное значение, которое определяется по формуле + - . Положительное значение говорит о планируемом излишке материалов - скорее всего, запланировано к получению больше чем требуется и требуется отменить заказы. Отрицательное количество говорит о планируемом дефиците материалов - для таких материалов требуется оформить заказы на передачу со склада, чтобы покрыть планируемую потребность;
  • Подразделение - подразделение, в которое передаются или переданы материалы;
  • Получать со склада - склад, с которого материал передается в производство;
  • Основание для получения - на основании чего материал передается в производство.

Команды списка:

  • Зарезервировать - позволяет зарезервировать материалы, которые запланированы к получению. При выполнении команды открывается форма корректировки заказа в которой автоматически устанавливается действие Зарезервировать по выбранным материалам;
  • Отгрузить со склада - позволяет установить действие по обеспечению Отгрузить по тем материалам, по которым есть текущий дефицит. При выполнении команды открывается форма корректировки заказа в которой автоматически устанавливается действие Отгрузить по выбранным материалам. Если единица хранения материала штучная, а дефицит не кратен одной штуке, то исходный дефицит округляется до целого;
  • Оформить накладную - позволяет оформить передачу материалов в производство. При выполнении команды создается новый документ, который заполняется выбранными материалами. Если для выбранных материалов есть заказы в которых установлено действие Отгрузить, то перед созданием документа необходимо выбрать нужные заказы, по которым будет создан документ;
  • Оформить накладную на отрицательный остаток в подразделении - позволяет оформить передачу материалов для покрытия отрицательного остатка в подразделении. При выполнении команды создается новый документ, который заполняется выбранными материалами с количеством равным отрицательному остатку в подразделении;
  • Оформить возврат на текущий излишек - позволяет оформить возврат материалов на склад для устранения излишка материалов в подразделении. При выполнении команды создается новый документ, который заполняется выбранными материалами с количеством равным текущему дефициту;
  • Оформить заказ материалов - позволяет оформить заказ материалов в производство. При выполнении команды создается новый документ, который заполняется выбранными материалами с количеством равным планируемому дефициту;
  • Корректировка заказа материала - позволяет открыть форму для ручной корректировки заказа материалов.

Заказы на передачу

На закладке отображаются документы Заказы материалов в производство.

К оформлению накладных

На закладке отображается список распоряжений к оформлению документов на передачу материалов.
Передачу материалов можно оформить по заказам или по отгрузке. По каждому распоряжению отображаются индикаторы, показывающие расхождения Заказ-Накладная и Накладная-Ордер.
По команде Оформить по отгрузке выбранная передача материалов перезаполняется отгруженным количеством, а если выбран заказ - создается передача материалов на отгруженное ордером количество.
Если по заказу оформлено несколько передач материалов и существуют расхождения между ордером и накладной, то при заполнении по отгрузке откроется форма Переоформление документов передачи материалов. Форма предназначена для устранения расхождений Ордер-Накладная, путем перезаполнения созданных накладных, либо созданием новых накладных.

Передачи в производство, перемещения и возвраты

На закладке отображается список документов Передача в производство, Перемещение материалов в производстве, Возврат из производства.
