Помощник перехода на учет внеоборотных активов версии 2.4

Помощник используется для перехода с версии 2.2 на версию 2.4.
Если при переходе помощник не обнаружит ошибок, то переход считается завершенным, иначе - потребуется участие пользователя.

Помощник открывается с помощью команды, расположенной в панели настроек НСИ и администрирование - Настройка НСИ и разделов - Внеоборотные активы.
Команда доступна, если выполняется одно из условий:

  • Помощник перехода не использовался,
  • При переходе зарегистрированы ошибки,
  • Включена опция Отображать команды для ведения учета внеоборотных активов версии 2.2.

Если переход был выполнен без использования помощника, то рекомендуется воспользоваться помощником для проверки корректности перехода.

Описание версии учета 2.4

Новые возможности и изменения версии:

  • Полноценный учет внеоборотных активов в управленческом учете;
  • Документы поддерживают отражение в регламентированном и в управленческом учетах одновременно или только в одном из них;
  • Учет активов в международном учете ведется по данным управленческого или регламентированного учетов (отдельные документы не используются);
  • Оптимизировано начисление амортизации по большому количеству ОС и НМА.

Что нужно для перехода на учет 2.4

Помощник перехода позволяет автоматически выполнить действия, необходимые для перехода на версию 2.4:

  • Заполнить НСИ
  • Перенести параметры учета внеоборотных активов в новые регистры
  • Заполнить суммы вложений во внеоборотные активы в оперативных регистрах

Помощник состоит из нескольких шагов после выполнения, которых будут перенесены данные и зарегистрированы ошибки, обнаруженные во время переноса.

Выполнение шагов помощника

Предварительные настройки
  • Установить опции Партионный учет значение Версия 2.2
  • Включить опцию Управленческий учет по правилам международного финансового учета
  • Включить опцию Прочие доходы и расходы в валюте регл. учета, если включена опция Регламентированный учет
  • Если ведется учет внеоборотных активов в международном учете, то функциональная валюта должна совпадать с валютой управленческого учета

Перед тем как включить учет версии 2.4 необходимо:

Перенос данных

На этом шаге помощник переносит необходимые данные.

Если перенос данных уже выполнялся, то на этом шаге появляется возможность выбрать один из режимов:

  • Проверить - будет выполнение сравнение данных версии 2.2 и версии 2.4 и зарегистрированы ошибки, если они отличаются;
  • Исправить ошибки - будут перенесены данные только по внеоборотным активам, для которых зарегистрированы ошибки;
  • Перенести повторно - все ранее перенесенные данные будут заполнены повторно.


Данные переносятся по внеоборотным активам, которые приняты к учету на дату перехода.

Помощник создает и отражает в учете следующие документы:

  • Ввод остатков внеоборотных активов - по активам уже принятым на баланс;
  • Ввод остатков вложений во внеоборотные активы - по активам еще не принятым на баланс или при наличии модернизации;
  • Изменение параметров ОС - если способы отражения расходов заданы документом Изменение параметров ОС;
  • Изменение параметров НМА - если способы отражения расходов заданы документом Изменение параметров НМА;
  • Ввод начальных остатков активов и пассивов - при использовании функциональной опции Отражение начисленной амортизации регламентированного учета в управленческом учете.

Документы создаются на день раньше, чем дата начала учета 2.4 и не формируют проводок.
В форме документа отображается информационная надпись Документ введен для перехода на учет внеоборотных активов версии 2.4.

Параметры учета переносятся по данным регистров как есть - если в регистре данные неверные, то и после переноса они будут неверные, но будет зарегистрирована ошибка. В этих случаях рекомендуется исправить ошибки до даты начала перехода и после этого выполнять переход.
Документ проводится только если он прошел проверку заполнения.
Перед проведением из документа переносятся строки, которые не прошли проверку в отдельный документ.
В каждом документе будет не более 100 строк.

Завершение

На этом шаге отображаются результаты переноса данных:
- Перечисляются какие именно данные были перенесены;
- Наличие ошибок и предупреждений.

Для версии 2.4 в панели настроек раздела Внеоборотные активы доступные новые опции:
- Ведение списка объектов строительства;
- Правила учета внеоборотных активов в управленческом учете.

В закрытии месяца отображается состояние перехода в виде операции Переход на учет внеоборотных активов версии 2.4.

После перехода не рекомендуется изменять данные до даты начала учета.
Если данные будут изменены, то может потребоваться повторный переход.

Проверка перехода и устранение ошибок

Проверка перехода требуется в следующих случаях:
- переход выполнен без использования помощника
- после перехода были изменены данные в прошлом периоде (программа не отслеживает изменения в прошлом периоде, которые влияют на корректность перехода).

Для проверки перехода нужно открыть помощник, выбрать режим Проверить и запустить проверку.
Для устранения ошибок используется рабочее место Результаты перехода на учет версии 2.4.

Изменение данных после перехода

Если после перехода были изменены данные в прошлом периоде, то необходимо:

  • Открыть помощник и выполнить проверку данных;
  • Устранить найденные ошибки с помощью режима Исправить ошибки, либо вручную.

Как создаются группы финансового учета внеоборотных активов

В учете версии 2.4 используются Группы финансового учета внеоборотных активов для указания счетов учета.
Помощник перехода создает элементы в соответствии с имеющимися комбинациями счетов, заданных в учете версии 2.2.
Наименование формируется в зависимости от счета учета, например, Основные средства в организации, Арендованное имущество, Нематериальные активы организации.
К наименованию может добавляться номер комбинации, например, если у разных активов счет учета совпадает, но счет начисления амортизации различается.

Настройка счетов для шаблонов проводок в МФУ не генерируется.
Но ГФУ создаются с учетом счетов учета в МФУ - если в регламентированном учете счета совпадают, но в международном учете они разные, то будет создано несколько ГФУ, а в описании ГФУ будет информация о счетах международного учета.

Как заполняются параметры учета в управленческом учете

В версии 2.2 для управленческого учета не вводятся параметры учета, поэтому при переходе параметры определяются так:

  • Совпадают с параметрами международного учета - если ведется учет внеоборотных активов в международном учете и функциональная валюта в валюте упр. учета;
  • Совпадают с параметра бухгалтерского учета - если включена опция Отражение начисленной амортизации регламентированного учета в управленческом учете;
  • Не принимается в управленческом учете - в остальных случаях.

Если в управленческом учете расходы, отнесенные на актив, полностью списаны, то актив будет принят к учету как полностью самортизированный.
Стоимость и амортизация такого актива определяется как сумма всех затрат, отнесенных на данный актив.
Если затрат не было, то стоимость будет нулевой, при необходимости ее можно уточнить (нулевая стоимость допускается).

Как переносятся вложения во внеоборотные активы

В учете 2.4 для получения данных о вложениях во внеоборотные активы используется только регистр Прочие расходы.
На момент перехода в регистре Прочие расходы не заполнены суммы вложений в валютах регламентированного и управленческого учетов.

Помощник перехода переносит данные о вложениях с помощью документов Ввод остатков вложений во внеоборотные активы.
Документ заполняет регистры Прочие расходы, Активы и пассивы.
В регистре Прочие расходы заполняются ресурсы Сумма (упр), Сумма (регл), Постоянная разница, Временная разница.
В регистре Активы и пассивы вложения отражаются по предопределенным статьям активов Капитализация ОС, Капитализация НМА и НИОКР.

Порядок заполнения зависит от того принят ли к учету актив:
- если актив принят к учету, то данные берутся начиная с даты принятия к учету или начиная с даты самой поздней модернизации;
- если актив не принят к учету, то данные берутся за весь период.

Сумма вложений в регл. учете определяется следующим образом:
- берутся обороты по регистру Хозрасчетный по группе счетов 08 "Вложения во внеоборотные активы", по статье расходов с вариантом распределения Формирование стоимости внеоборотных активов или по пустой статье. При наличии оборотов по пустой статье необходимо самостоятельно во вводе остатков указать нужную статью или статьи.

Сумма вложений в упр. учете определяется следующим образом:
- берутся обороты регистров Прочие расходы, Активы и пассивы.

При переносе данных из регистра Активы и пассивы не заполняется статья расходов, ее нужно заполнить самостоятельно. После перехода остаток в регистре зануляется с помощью документа Ввод начальных остатков Активы и пассивы.
