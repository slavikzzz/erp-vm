Производство

Управление производством (версия 2.1) - включение функциональности управления производством предыдущей версии. Используется для совместимости. Для новых баз или для баз, в которых ранее не применялось управление производством, рекомендуется сразу использовать управление производством версии 2.2.
Дополнительная информация на ИТС

Управление производством (версия 2.2). Включение новых механизмов управления производством. Совместное применение с опцией Управление производством (версия 2.2) рекомендуется только на переходный период при переводе управления производством с механизмов 2.1 на 2.2.

Методика управления производством - настройка использования графика производства в управлении производством 2.2.

Планировать график производства регламентным заданием - позволяет организовать автоматический расчет графика производства по заданному расписанию.

Контроль выполнения при закрытии заказов на производство. Перевести заказ на производство в статус Закрыт невозможно, если по нему есть невыполненные этапы и/или неоформленные распоряжения на выпуск продукции.

Контроль отгрузки при закрытии этапов производства - не закрывать этапы по которым не исполнены распоряжения на отгрузку материалов (продукция).

Обособление материалов в производстве при заказе в производство - определяется объект обособления, под который будет выполняться обеспечение этапа производства материалами и работами. Доступные варианты:

  • по направлению деятельности - потребность обеспечивается в целом по направлению деятельности без детализации до заказов.

  • по назначению продукции - рекомендуется, если приобретение материалов для целевого производства продукции происходит до создания заказов на производство.

  • по заказу на производство - потребность обеспечивается по заказу на производство без детализации до этапов.

  • по этапу производства - рекомендуется использовать при необходимости детального управления обеспечением каждого этапа и автоматического контроля сроков поставки материалов в графике производства.

Обособление материалов в производстве при заказе в кладовую - определяется объект обособления, под который будет выполняться обеспечение заказа материалов в производство.

Разрешить использование интервала планирования "Час" - при включении этой функциональной опции появляется возможность составлять график производства с использованием интервала планирования "Час". По-умолчанию выключено и доступны только интервалы День, Неделя, Месяц. Включать эту функциональную опцию следует только в том случае, когда действительно необходимо на уровне главного диспетчера управлять получением результата выполнения этапа с точностью до часа. Дополнительно рекомендуется ознакомиться со статьей Интервальное планирование.

Использовать параметры назначения спецификаций - включение параметризации автоматического подбора спецификаций. Позволяет настроить состав используемых при автоподборе параметров.

Виды технологических операций - возможность использования наборов дополнительных реквизитов и сведений в производственных операциях.
Справочник Виды технологических операций, определяет состав наборов дополнительных свойств (реквизитов и сведений).

Контроль выполнения операций при закрытии этапов производства - не закрывать этапы у которых имеются незавершенные операции.

Опция Оформлять выработку сотрудников на основании производственных операций включает возможность детализация выработки сотрудников до конкретных производственных операций.

Переработка на стороне

Производство на стороне (версия 2.4) - позволяет вести учет операций по переработке сырья и материалов силами сторонних организаций. Оформление приобретения услуг по переработке.

Обособление сырья и материалов - определяется объект обособления, под который будет выполняться обеспечение сырьем и материалами. Доступные варианты:

  • по направлению деятельности - потребность обеспечивается в целом по направлению деятельности без детализации до заказов.

  • по переработчику / договору - используется при необходимости обобщенного управления обеспечением по переработчику или отдельному договору в целом. Так же рекомендуется, если приобретение материалов для переработки происходит до создания заказов переработчику.

  • по назначению продукции - позволяет использовать в качестве объекта обособления любое целевое назначение продукции, например: Направление деятельности или Заказ клиента. Рекомендуется, если приобретение материалов для переработки происходит до создания заказов переработчику.

  • по заказу переработчику - потребность обеспечивается по заказу переработчику.

Не закрывать частично отработанные заказы переработчикам - установка запрета на закрытие заказов переработчикам, поступивших и/или отгруженных не полностью.

Не закрывать частично оплаченные заказы переработчикам - установка запрета на закрытие заказов переработчикам, оплаченных не полностью.

Производство на стороне (версия 2.5) - позволяет вести учет операций по переработке сырья и материалов силами сторонних организаций. Оформление приобретения услуг по переработке.

Акты о расхождениях после передачи переработчику, Акты о расхождениях после поступления от переработчика - позволяет вводить акты о расхождениях.

Заказы переработчикам - позволяет планировать передачу в переработку.

Статусы заказов переработчикам - позволяет отслеживать ход выполнения заказов переработчикам

Не закрывать частично отработанные заказы переработчикам - установка запрета на закрытие заказов переработчикам, поступивших и/или отгруженных не полностью.

Не закрывать частично оплаченные заказы переработчикам - установка запрета на закрытие заказов переработчикам, оплаченных не полностью.

Обособление материалов:

При заказе - определяется объект обособления, под который будет выполняться обеспечение материалами. Доступные варианты:

  • по направлению деятельности - потребность обеспечивается в целом по направлению деятельности без детализации до заказов.

  • по переработчику / договору - используется при необходимости обобщенного управления обеспечением по переработчику или отдельному договору в целом. Так же рекомендуется, если приобретение материалов для переработки происходит до создания заказов переработчику.

  • по назначению продукции - позволяет использовать в качестве объекта обособления любое целевое назначение продукции, например: Направление деятельности или Заказ клиента. Рекомендуется, если приобретение материалов для переработки происходит до создания заказов переработчику.

  • по заказу переработчику - потребность обеспечивается по заказу переработчику.

У переработчика - определяется объект обособления, по которому будет вестись учет материалов, переданных переработчику. Доступные варианты:

  • не обособлять - при учете у переработчика материалы не обосабливаются.

  • по заказу переработчику - при учете у переработчика обособление выполняется по заказу переработчику.

Производственные затраты

Валюта расценок видов работ определяет валюту расценок видов работ сотрудников. Расценки задаются в карточке вида работ и используются для расчета сумм в документах Выработка сотрудников.

Валюта плановых цен материалов, продукции и отходов определяет валюту, в которой могут быть заданы плановые цены. Настройка доступна только при управлении производством 2.1.

Плановые цены материалов, продукции и отходов это вид цены, который используется для заполнения цен побочного и промежуточного выпуска (по фиксированной стоимости). Определяет стоимость продукции при выпуске по плановой стоимости (если в учетной политике организации включена соответствующая опция). Определяет цены материалов при расчете плановых калькуляций.

Плановые калькуляции при выпуске продукции. Настройка определяет доступность плановых калькуляций в документах выпуска продукции для установки плановой стоимости продукции. Цены в документах выпуска устанавливаются для побочного и промежуточного выпуска (по фиксированной стоимости) и при использовании выпуска по плановой стоимости (настраивается в учетной политике организации). Настройка доступна только при управлении производством 2.1.

Аналитический учет по группам (видам) продукции включает возможность разделения прямых производственных затрат (материалов и трудозатрат производства) по группам (видам) продукции.
