Международный финансовый учет

Раздел предназначен для управления параметрами ведения международного финансового учета.

Подсистема предназначена для решения следующих задач:

  • Отражение документов оперативного учета в финансовом учете (регулируется функциональной опцией Проводки по данным оперативного учета), в том числе на нескольких планах счетов;
  • Отражение документов в международном финансовом учете методом трансляции проводок регламентированного учета (регулируется функциональной опцией Проводки по данным регламентированного учета). При этом должна быть включена опция Регламентированный учет (Настройка НСИ и администрирование - Настройка разделов - Регламентированный учет);
  • Подготовка отчетности в соответствии с принципами международных стандартов финансовой отчетности (МСФО), включая возможность настройки дополнительных отчетов для проверки и анализа учетных данных;
  • Подготовка отчетности по правилам управленческого учета, построенного на основе МСФО;
  • Гибкая настройка форм финансовой отчетности.


Опция Несколько планов счетов позволяет вести учет на нескольких различных планах счетов, с разными источниками функциональной валюты (по данным управленческого и регламентированного учетов).

Выбор варианта настройки трансляции в Проводки по данным оперативного учета:
■ Настройка по объектам учета и их корреспонденциям – режим по умолчанию для новых баз, трансляция данных оперативного учета в МФУ балансовым методом от объектов учета;
■ Настройка по хозяйственным операциям – трансляция оборотов по хозяйственным операциям в МФУ.

Для новых внедрений рекомендуется использовать Настройка по объектам учета и их корреспонденциям.
При инициализации новой информационной базы доступен только режим Настройка по объектам учета и их корреспонденциям. В базах, где ранее велся МФУ по данным оперативного учета, опция Настройка по хозяйственным операциям включена и доступна. Загрузка в базу модели МФУ, содержащий в себе настроенные шаблоны проводок по хоз. операциям, так же сделает опцию доступной.

Совместное использование с трансляцией из регламентированного учета.
Режимы Проводки по данным оперативного учета и Проводки по данным регламентированного учета могут использоваться совместно.
Трансляции будут работать одновременно, это приведет к задвоению проводок, поэтому нужно использовать настройки исключения и документ отражать в МФУ по данным только одного учета.
В случае применения Настройка по хозяйственным операциям есть возможность добавить в исключения (игнорирование при формирования проводок) операцию целиком или шаблона по ней. Для этого используются команды Исключить хозяйственную операцию и Исключить шаблон проводки.
Трансляция из регламентированного учета имеет настройку Исключить соответствие счетов и Исключить соответствие оборотов.
Использование исключений позволяет часть операций отражать по данным оперативного учета, а часть по данным регламентированного.
В случае применения Настройка по объектам учета и их корреспонденциям объекты учета отражаются в учете всегда, исключить из отражения можно только корреспонденции. Для этого в корреспонденции объектов учета используется флаг Не отражается на счетах.
В случае совместного использования отражения в МФУ по объектам учета и по настройки данным регламентированного учета нужно настроить в исключения все счета регламентированного учета на закладке Соответствие счетов и использовать только настройки корреспонденций на закладке Соответствие оборотов, настроив исключения для таких проводок по корреспонденции объектов учета.

Настроенную модель МФУ можно выгрузить и загрузить используя команды Загрузить модель учета и Выгрузить модель учета.

По ссылке Настроить вариант проводок и валюты учета на плане счетов выполняются настройки:
Функциональной валюты и Валюты представления. При включенной опции Несколько планов счетов настройка валют выполняется для каждого плана счетов отдельно. Настройка выполняется в карточке плана счетов;
Варианта проводок: С корреспонденцией или Без корреспонденции.

Функциональная валюта - определяется валютой от выбранного режима ведения учета, какой учет лежит в основе МФУ управленческий или регламентированный.
От этого зависит, какие данные из оперативного учета будут браться для формирования проводок по МФУ: суммы в упр. валюте или в регл. валюте.
В случае настройки трансляции операций по данным регл. учета за основу будут всегда браться данные регл. учета по плану счетов, не смотря на настройку режима функциональной валюты.

При использовании международного учета и возможности независимой от регл. учета трактовки расходов, финансового результата и независимых настроек ведения внеоборотных активов (например, иной СПИ или дата принятия к учету) рекомендуется использовать настройку функциональной валюты по данным упр. учета.

Это накладывает ограничение на выбор произвольной валюты упр. учета, она должна быть национальной валютой, согласно правилам МСФО, в которой совершается большинство хозяйственных операций в учете. Для получения отчетности в произвольной иной валюте нужно использовать валюту представления.

Валюта представления отчетности - это валюта, в которой финансовая отчетность представляется пользователям.

В соответствии с п.8 МСФО (IAS) 21 «Влияние изменений обменных курсов валют» функциональной валютой является валюта, используемая в основной экономической среде, в которой предприятие осуществляет свою деятельность.

При выборе варианта проводок С корреспонденцией в каждой проводке указывается и дебетуемый и кредитуемый счет. При выборе варианта Без корреспонденции проводки по дебетуемым и кредитуемым счетам формируются отдельными записями, сохранение баланса контролируется на уровне документа.

Отражать в учете только проверенные документы - при групповом отражении документов в международном финансовом учете обрабатывать только документы со статусом Проверен. При отражении документа из формы просмотра проводок статус проверки не анализируется.
