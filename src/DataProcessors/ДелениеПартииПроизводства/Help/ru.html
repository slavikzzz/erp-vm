Деление партии производства

Рабочее место предназначено для внесения различных изменений в партии производства. Изменения могут быть сделаны только начиная с тех этапов, которые еще не выполнялись.
В поле "После этапа" указывается этап производства, после которого должны быть внесены изменения.
Поддерживаются следующие действия: Сокращение (увеличение) производства, прекращение производства партии, разделение на несколько партий, оформление промежуточного выпуска.

Сократить (увеличить) производство

Это действие применяется в тех случаях, когда дальнейшее выполнение партии должно продолжиться уже над меньшим количеством. Такое может потребоваться, например, при выявлении неисправимого брака обрабатываемых изделий, при изменении потребности.
Необходимо указать новый размер партии. При этом изменяемые этапы могут быть либо перезаполнены по исходной спецификации, либо пересчитаны по коэффициенту.

Отменить дальнейшее производство, оформить выпуск

Применяется при необходимости прекращения производства партии. В этапе, на котором прекращается производство, оформляется выпуск. Для этого необходимо выбрать номенклатуру, получателя и указать количество.
Последующие этапы будут удалены.

Разделить на партии

Используется для разделения дальнейшего выполнения на несколько независимых партии. В этом случае оформляется промежуточный выпуск, для которого надо указать номенклатуру. Этот выпуск будет разбит на части с заданным способом распределения затрат между ними.
По каждой из образовавшихся частей можно выбрать дальнейшую судьбу: продолжить обработку, положить на склад или списать на расходы.
При продолжении обработки будут сформированы этапы оставшейся обработки. При этом для продолжения обработки можно указать другую спецификацию.
Может применяться в различных случаях. Например, если забракована часть партии. Или если от большой партии, которая будет долго обрабатываться, надо отделить небольшое количество, которое срочно надо обработать до конца.

Оформить промежуточный выпуск и продолжить обработку

Вариант может применяться в том случае, когда процесс описан бесполуфабрикатным многоэтапным способом, но вдруг потребовалось, например, временно положить текущее состояние на склад.
Или если надо продолжить дальнейшую обработку партии уже по другой спецификации.
