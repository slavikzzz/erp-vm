Проводки международного учета

Форма для просмотра, формирования и редактирования проводок международного учета.
Если документ проведен, а проводки по нему еще не сформированы из рабочего места Отражение документов в международном учете, то их можно сформировать по кнопке Отразить в международном учете.
Для ручного редактирования проводок нужно включить отметку Ручная корректировка проводок.
В поле комментарий будет выводится информация о причинах, по которым проводка не может быть сформирована автоматически с возможностью перехода к настройке отражений по данным оперативного или регламентированного учетов.

Если документ отражается в учете по нескольким планам счетов, то проводки по каждому из планов счетов отображаются на отдельной вкладке. Флаг Ручная корректировка проводок включается независимо по каждому из планов счетов.
