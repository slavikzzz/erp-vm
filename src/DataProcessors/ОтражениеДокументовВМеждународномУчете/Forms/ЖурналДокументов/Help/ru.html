Журнал документов регл. учета

Данное рабочее место представляет собой журнал отражения документов в регламентированном учете.

В нем представлены все отраженные или проверяемые (если включена соответствующая функциональная опция) документы регламентированного учета.

Для более быстрой и удобной работы с журналом документов на форму выведены быстрые отборы, которые позволяют:

  • Задать период просмотра журнала документов;
  • Установить отбор по организации;
  • Просмотреть отражение документов: отраженные, ожидающие автоматического отражения, отраженные вручную, ожидающие подтверждения при ручном отражении, не отраженные (не указаны счета учета);
  • Просмотреть проверку документов (если соответствующая функциональная опция включена): проверенные, непроверенные, ожидающие повторной проверке (если в настроках разрешено изменение проверенных документов);
  • Выбрать произвольный набор типов документов для просмотра: так, например, можно выбрать только Операции регламентированного учета или выбрать для просмотра только документы поступления и реализации товаров.

При работе с данным рабочим местом можно производить следующие действия с документами:

  • Отразить в учете - для документов ожидающих отражения в учете или для которых не были указаны счета учета будет произведено автоматическое отражение в учете, и подтверждение актуальности проводок для документов отраженных в учете автоматически, но в последствии перепроведенных;
  • Снять отражение в учете - предполагает работу только с автоматически отраженными документами, в этом случае выделенные документы приобретят статус "ожидающие отражения в регл. учете";
  • Открыть форму проводок регламентированного учета;
  • Проверить - все непроверенные документы пометятся как проверенные;
  • Отменить проверку - все проверенные документы пометятся как непроверенные (или к повторной проверке - если настройка проверки не предполагает блокировку проверенных документов);
  • Создать операцию регл. учета - будет создан документ Операция (регл. учета) (бухгалтерская операция), в списке корректируемых документов которого будут находиться выделенные документы данной формы.

Если данная форма открывается из основного рабочего места отражения документов в регламентированном учете, то она откроется с предустановленными отборами, в зависимости от того, какая группа документов открывалась для просмотра.

На форму возможно вывести некоторые реквизиты документов (такие как контрагент, договор и т.д.), для этого необходимо в командной панели выбрать Изменить форму и в группе Группа дополнительных данных по документам списка выбрать необходимые реквизиты.
