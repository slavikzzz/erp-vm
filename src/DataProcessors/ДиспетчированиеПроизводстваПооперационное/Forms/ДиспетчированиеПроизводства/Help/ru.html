Создание маршрутных листов


Для создания маршрутных листов используется команда Сформировать маршрутные листы.
Команда становится доступной, если есть необходимость в создании.

Маршрутные листы создаются на каждый этап производства. Количество маршрутных листов соответствующих одному этапу определяется в настройке этапа Разбивать маршрутные листы. При необходимости маршрутный лист можно разбить на несколько с помощью команды Разбить маршрутный лист.

Планирование расписания (доступно только для методики Пооперационное планирование)


Для планирования расписания используется команда Планировать расписание, которая открывает специальное рабочее место Пооперационное планирование.

Оформление связанных документов


При выполнении маршрутных листов необходимо оформить следующие документы:

  • Выработка сотрудников - оформляется при выполнении маршрутного листа, если указано какие бригады выполняют работы;
  • Выпуск продукции и выполнение работ - оформляется после выполнения маршрутного листа.

Если на основании маршрутного листа нужно оформить указанные документы, то в соответствующей колонке будет выведен индикатор.
Создать необходимый документ можно щелчком на индикаторе или при помощи команды расположенной в подменю Задачи.

Индикаторы


Для маршрутных листов выводятся различные индикаторы, которые позволяют определить наличие проблем или показывают справочную информацию.

  • Предупреждения - для расшифровки текста предупреждения необходимо выполнить двойной клик на индикаторе;
  • Не сформировано расписание (доступно только для методики Пооперационное планирование);
  • Предыдущие этапы не завершены:
  • Требуется оформить выработку сотрудников;
  • Требуется оформить выпуск продукции или выполнение работ.

Просмотр расписания (доступно только для методики Пооперационное планирование)


Для просмотра пооперационного расписания используется команда Расписание. Расписание может быть отображено в виде списка или в виде диаграммы Ганта.
Имеется возможность индикации расписания выделенных маршрутных листов в одном из двух режимов:

  • Режим отбора – отображается расписание только выделенных документов;
  • Режим подсветки – для индикации расписания выделенных маршрутных листов используется цветовая подсветка.

Печать сменно-суточных заданий


Для печати сменно-суточных заданий используется команда Печать сменно-суточных заданий. Для выполнения печати в маршрутных листах должны быть назначены бригады по видам работ.
