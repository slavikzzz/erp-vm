Выполнение маршрутных листов

Рабочее место позволяет определить какие действия необходимо выполнить для выполнения маршрутных листов.

Ввод данных о выполнении

Выполнение операций

Выработка сотрудников

Выпуск продукции

Выполнение работ и услуг

Ввод данных о выполнении

Данный блок показывает сколько выполняется маршрутных листов.

Блок доступен при использовании в подразделении методики ББВ/УББВ для управления маршрутными листами.

Выполнение операций

Данный блок показывает сколько требуется выполнить операций сегодня и в последующие дни, а также позволяет отразить выполнение операций, ввести фактический расход материалов, выпуск изделий и выполнение работ, сформировать сменно-суточные задания.

Блок доступен при использовании в подразделении пооперационного планирования для управления маршрутными листами.

Выработка сотрудников

Данный блок позволяет определить необходимость оформления выработки сотрудников по выполненным работам.
Если требуется оформить выработку, то будет отображаться индикатор Оформить.
Если оформление выработки не требуется, то будет отображаться индикатор Нет задач.

Выпуск продукции

Данный блок позволяет определить необходимость оформления выпуска продукции.
Если требуется оформить выпуск, то будет отображаться индикатор Оформить.
Если оформление выпуска не требуется, то будет отображаться индикатор Нет задач.

Выполнение работ и услуг

Данный блок позволяет определить необходимость оформления выполнения работ (услуг).
Если требуется оформить выполнение работ (услуг), то будет отображаться индикатор Оформить.
Если оформление выполнения работ (услуг) не требуется, то будет отображаться индикатор Нет задач.
