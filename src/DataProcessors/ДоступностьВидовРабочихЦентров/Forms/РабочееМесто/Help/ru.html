Доступность видов рабочих центров

Рабочее место позволяет определить доступность видов рабочих центров в подразделении.
Рабочее место доступно в разделе Производство - Нормативно-справочная информация - Доступность видов рабочих центров.

Внешний вид рабочего места

На форме расположен список видов РЦ, для которых отображается информация:

  • доступность введенная для вида РЦ - показывает доступность вида рабочих центров (отображается, если доступность вводится для вида РЦ);
  • доступность определяемая по графикам работы рабочих центров - показывает доступность вида рабочих центров по данным графиков работы его рабочих центров;
  • доступность занятая графиком производства - показывает сколько по графику производства занято доступности вида рабочих центров.

Ввод доступности видов РЦ

Если работа видов рабочих центров планируется в графике производства, то для них требуется указать доступность в интервалах планирования.
Ввод доступности выполняется в соответствии с параметрами планирования вида РЦ в графике производства - вводится либо доступность вида РЦ, либо график работы РЦ.

Ввод графиков работы РЦ

Если требуется планировать работу рабочих центров, то для них требуется ввести графики работы.
Ввод графиков работы РЦ выполняется в отдельной форме Графики работы рабочих центров. Открытие формы выполняется при выборе нужной ячейки в текущем рабочем месте.
