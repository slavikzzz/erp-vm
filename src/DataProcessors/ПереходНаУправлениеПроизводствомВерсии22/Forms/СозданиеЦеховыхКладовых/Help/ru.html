Создание цеховых кладовых

Форма предназначена для создания цеховых кладовых, относящихся к выбранным производственным подразделениям. В основе формы иерархический список производственных подразделений.
В колонке Цеховая кладовая можно отметить флажком создание новой кладовой или выбрать уже существующий склад для назначения его в качестве кладовой для выбранного подразделения. В этой же форме для создаваемых кладовых могут быть настроены способы обеспечения перемещением со склада.
После задания всех необходимых настроек в форме создание кладовых осуществляется по нажатию кнопки Создать цеховые кладовые.
