Помощник по переходу на ЕНС

Помощник позволяет произвести все действия, необходимые для перехода на учет ЕНС на 01.01.2023 г., и проконтролировать результат их выполнения. 

1. Перед переходом на ЕНС рекомендуется запросить сверку с ФНС. Данные, которые будут переносится на ЕНС должны совпадать с данными в акте сверки.

2. Рекомендуется контролировать сведения о налогах, переносимых на ЕНС контролировать при помощи стандартных отчетов. В частности по команде "Остатки расчетов по налогам и взносам" формируется оборотно-сальдовая ведомость по счетам 68 и 69.

3. Переход на учет ЕНС состоит из следующих этапов:

  • Перенос остатков расчетов по налогам и взносам. Программа автоматически формирует документ "Операция по ЕНС" и заполняет его. При переносе остатков нужно указать сведения о налоге и срок уплаты, на который образовалась задолженность. Рекомендуется проверить корректность определения программой сроков уплаты.
  • Перенос остатков расчетов по пеням, штрафам, процентам. Программа автоматически формирует документ "Операция по ЕНС" и заполняет его. 
  • Перенос переплат, которые образовались при расчетах по налогам, а также по пеням и т.п. Эти сведения формируют положительное сальдо ЕНП.
  • Формирование уведомления об исчисленных налогах по НДФЛ, уплаченному в последний рабочий день 2022 года.
  • Зачет зарегистрированных переплат в счет задолженности, перенесенной на ЕНС предыдущим операциями. Формируется документ "Операция (БУ и НУ)".

После выполнения всех этапов состояние ЕНС в программе должно совпасть с состоянием ЕНС в личном кабинете ФСН.
