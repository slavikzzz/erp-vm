Оформление производства без заказов

Рабочее место предназначено для автоматического и ручного оформления производства без заказа и распределения возвратных отходов.
Для рабочего места доступна гибкая настройка фильтров по Организации, Подразделению и т.д.

Продукция

В списке отображается информация:

  • о выпущенной продукции, на которую необходимо отнести затраты;
  • о настройках модели формирования стоимости продукции.

Команды:

  • Оформить по настройкам - предназначена для автоматического формирования документов для выделенных строк, согласно настройкам модели формирования стоимости;
  • Оформить вручную - Производство без заказа - предназначена для пакетного формирования документов Производство без заказа для выделенных строк;
  • Оформить вручную - Распределение возвратных отходов - предназначена для пакетного формирования документов Распределение возвратных отходов для выделенных строк;
  • Настройка модели формирования стоимости - Настроить - предназначена для пакетной настройки модели формирования стоимости для выделенных строк посредством формы настройки модели формирования стоимости;
  • Настройка модели формирования стоимости - Очистить - предназначена для пакетной очистки настроек модели формирования стоимости для выделенных строк;
