Установка драйвера токена Рутокен для macOS


Начиная с версии macOS 10.7, драйверы для  Рутокен Lite и ЭЦП не требуются. Для определения модели подключите Рутокен и откройте программу Launchpad. Далее в строке поиска введите слово "информация" и откройте программу Информация о системе (System Information). В окне программы в разделе USB отобразится модель носителя и информация о нем (рис.1). 

Рис.1

 


Для Рутокен S установите драйвер носителя. Для этого скачайте установочный файл с официального сайта. Далее запустите файл *.pkg и следуйте указаниям установщика. 


Установка драйвера токена JaCarta для macOS


Чтобы работать с носителем JaCarta установите приложение Единый клиент JaCarta с официального сайта. Затем распакуйте архив, запустите установочный файл и дождитесь завершения установки.
