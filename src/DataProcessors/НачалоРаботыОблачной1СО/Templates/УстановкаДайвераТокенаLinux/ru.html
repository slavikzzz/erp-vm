Установка драйвера токена Рутокен для Linux


Настройка драйвера токена Рутокен на Linux зависит от модели носителя. Для определения модели подключите Рутокен и откройте приложение Терминал. Далее введите команду lsusb. В окне терминала отобразится модель носителя (рис.1).

Рис.1

 


Если используется Рутокен S, то необходимо скачать соответствующий драйвер с официального сайта, запустить его и следовать указаниям мастера. 

Если используется Рутокен Lite, Рутокен ЭЦП, то драйвер не требуется, если установлена библиотека libccid версии 1.4.2 и выше. Проверить версию библиотеки можно в терминале используя команду:

• dpkg -s libccid libpcsclite1 pcscd pcsc-tools opensc для deb-based систем (Debian, Ubuntu, Linux Mint, Astra Linux)

• команду sudo rpm -q ccid pcsc-lite pcsc-tools opensc для rpm-based систем (RedHat, CentOS, Fedora, ALT Linux, ROSA Linux, MCBC, ГосЛинукс).



Установка драйвера токена JaCarta для Linux


Чтобы работать с носителем JaCarta установите приложение Единый клиент JaCarta с официального сайта. Затем распакуйте архив, установите файл install.sh и дождитесь завершения установки.
