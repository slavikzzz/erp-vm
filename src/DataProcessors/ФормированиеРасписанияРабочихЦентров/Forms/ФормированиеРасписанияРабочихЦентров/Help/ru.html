Формирование расписания рабочих центров

Рабочее место предназначено для формирования расписания ключевых рабочих центров.
Расписание отражает в какое время ключевой рабочий центр должен выполнить маршрутный лист производства.

Начало работы

Для начала работы необходимо выбрать подразделение, вид рабочего центра и период.

Период, на который составляется расписание, может быть произвольным или определенный день.
При выборе произвольного периода нельзя указать период, который охватывает несколько интервалов планирования. Например, если интервал планирования день, то нельзя указать период, в который входит несколько дней.

Работа со списком маршрутных листов, по которым требуется составить расписание

Уменьшение запланированного количества единиц/партий изделий

Количество единиц/партий изделия в маршрутном листе определяется настройкой Разбивать маршрутные листы, указываемой в спецификации или в маршрутной карте.
При необходимости можно уменьшить запланированное количество единиц/партий изделия в маршрутном листе с помощью команды Разбить маршрутный лист.
В форме нужно указать количество единиц/партий изделия в новом маршрутном листе, который будет создан в результате разбиения выбранного маршрутного листа. При этом в выбранном маршрутном листе количество будет уменьшено.

Как назначить рабочему центру выполнение маршрутных листов

Для этого нужно выбрать маршрутные листы и назначить выполнение рабочему центру с помощью команды ">".
При этом они перейдут в список Расписание. Чтобы вернуть их обратно в список нужно воспользоваться командой Удалить расписание.

Под списком маршрутных листов отображается справочная информация, которая помогает при формировании расписания.

Если для вида рабочего центра не доступна параллельная загрузка:

  • Всего - показывает объем работ по всем маршрутным листам, расположенным в списке;
  • Выбрано - показывает объем работ по выбранным маршрутным листам;
  • Переналадка - показывает время переналадки, затрачиваемое при выполнении выбранных маршрутных листов;
  • После назначения (свободно) - показывает сколько будет свободного времени у рабочего центра после назначения выбранных маршрутных листов;
  • После назначения (превышено) - показывает насколько будет превышено доступное время у рабочего центра после назначения выбранных маршрутных листов.

Если для вида рабочего центра доступна параллельная загрузка:

  • Всего - показывает процент загрузки рабочего центра по всем маршрутным листам;
  • Выбрано - показывает загрузки рабочего центра по выбранным маршрутным листам.

Оптимизация работы рабочих центров

Для оптимизации работы рабочих центров можно упорядочить маршрутные листы по изделиям или вариантам наладки (команды расположены в подменю Упорядочить).

Отклонение от графика производства

Если работы по маршрутному листу не удается выполнить в период определенный графиком производства то можно:

  • Изменить окончание работ - можно указать дату окончания работы РЦ позже, чем окончание интервала планирования.
  • Перенести выполнение - выполнение маршрутного листа можно перенести на другой интервал. Для этого нужно выбрать дату формирования расписания и в списке незапланированных маршрутных листов выбрать нужный маршрутный лист.

Работа с расписанием маршрутных листов

Маршрутные листы, для которых указано какой рабочий центр их будет выполнять, отображаются в списке Расписание.

В этом списке можно увидеть в какое время будет выполняться маршрутный лист. По умолчанию время определяется автоматически на основании очереди маршрутных листов. Но его можно изменить с помощью полей Начало и Окончание.

Управление объемом работ

Для каждого маршрутного листа определяется нормативный объем работ - количество часов, которое нужно потратить рабочему центру, для выполнения маршрутного листа.
Норматив определяется в спецификации, но может быть изменен при формировании расписания следующими способами:

  • Вручную - в поле Объем работ можно указать нужное количество часов;
  • Автоматически - с помощью команды Уместить в график работы, которая уменьшает нормативный объем работ так чтобы он не превышал доступное время рабочего центра. Эту возможность можно использовать для того чтобы "уместить" все маршрутные листы в график работы рабочего центра.

Работа с графиком занятости

Что отображается на графике

На графике отображается занятость рабочего центра на протяжении указанного периода:

  • Горизонтальная шкала - отображает периоды времени, в выбранном масштабе (5 минут, 15 минут, 30 минут, 1 час);
  • Вертикальная шкала - отображает загрузку рабочего центра в определенный период. Шкала отображается, если доступна параллельная загрузка;
  • Область, закрашенная сплошным цветом - отображает выполнение маршрутного листа в промежуток времени, относящийся к этой области. Если в этот же промежуток времени запланировано выполнение другого маршрутного листа, то область будет закрашена шахматным узором;
  • Область, закрашенная наклонными линиями слева направо - отображает период в который рабочий центр не доступен (нет расписания);
  • Область, закрашенная наклонными линиями справа налево - отображает операцию переналадки, цвет соответствует цвету варианта наладки, на который выполняется переналадка.

Перемещение маршрутного листа

Для перемещения маршрутного листа нужно выбрать его, вызвать контекстное меню и выбрать команду Выбрать для перемещения.
Далее нужно указать как его переместить:

  • Переместить перед определенным маршрутным листом - выбрать маршрутный лист, вызвать контекстное меню и воспользоваться командой Поместить перед. Если в выбранной области расположено несколько маршрутных листов, то будет предложено выбрать маршрутный лист.
  • Переместить после определенного маршрутного листа - выбрать маршрутный лист, вызвать контекстное меню и воспользоваться командой Поместить после. Если в выбранной области расположено несколько маршрутных листов, то будет предложено выбрать маршрутный лист.
  • Начать в определенное время - выбрать свободный или нерабочий период времени, вызвать контекстное меню и воспользоваться командой Начать. В форме ввода периода можно уточнить время начала.
  • Завершить в определенное время - выбрать свободный или нерабочий период времени, вызвать контекстное меню и воспользоваться командой Завершить. В форме ввода периода можно уточнить время окончания.

Другие интерактивные действия
  • Удалить из расписания - позволяет удалить из расписания маршрутный лист;
  • Ввести период выполнения - позволяет вручную ввести период выполнения;
  • Открыть - позволяет открыть форму маршрутного листа.
