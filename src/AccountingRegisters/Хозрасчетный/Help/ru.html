Журнал проводок (регламентированный учет)

Форма для просмотра и формирования проводок регламентированного учета.
Если документ проведен, а проводки по нему еще не сформированы из рабочего места Отражение документов в регламентированном учете, то их можно сформировать по кнопке Отразить в регл. учете.
В поле комментарий будет выводится информация о причинах, по которым проводка не может быть сформирована автоматически с возможностью перехода к настройке счетов учета.
