Distributed infobase is the information system consisting of several infobases of 1C:Enterprise (hereinafter referred to as nodes) with the same configurations and exchanging data in an offline mode. Characterized by the rigid hierarchic structure of nodes and by the presence of the single master node.

The method of distributed infobases (hereinafter referred to asDIB) is used to create locally distributed systems based on identical configurations of 1C:Enterprise 8. The method allows to transfer both data changes and changes of the infobase configuration. This option may be used to arrange the joint work in the configuration 1C:ERP Company management 2, when any other options of joint work, such as a client/server mode or work with the infobase in a web application mode cannot be used.

For more flexible exchange schemes, including between various systems, the universal data exchange method is used, the exchange via the universal format "Enterprise Data" etc.
