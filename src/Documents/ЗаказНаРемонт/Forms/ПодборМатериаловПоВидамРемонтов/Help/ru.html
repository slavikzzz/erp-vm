Подбор материалов по видам ремонтов

Форма предназначена для подбора материалов по видам ремонтов, указанных в заказе.
В списке отображаются материалы, количество которых в заказе, не совпадает с нормативами, указанными в виде ремонта.
Для добавления материала в заказ нужно указать количество в колонке После изменения.
