Формирование записей раздела 7 декларации по НДС

Документ предназначен для отражения операций в разделе 7 декларации по НДС.

По кнопке Заполнить документ заполняется автоматически по необлагаемым НДС операциям, доступным к отражению в разделе 7 декларации по НДС на дату ввода документа. Код операции заполняется автоматически на основании справочных данных в договорах и номенклатуре. Типы подтверждающих документов определяются на основании метаданных документов реализации.

Документ можно заполнить вручную, при этом дополнительно требуется распределить включенный в стоимость НДС по указанным в документе операциям, нажав на кнопку Распределить НДС.
За один налоговый период можно создавать несколько документов, их данные будут просуммированы при заполнении декларации и реестра подтверждающих документов.
