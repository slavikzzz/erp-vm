Снятие с регистрации транспортных средств

Документ предназначен для приостановки регистрации транспортных средств для целей налогообложения транспортным налогом.

  • Организация - организация, в которой зарегистрированы транспортные средства;
  • Снятие с регистрации с - режим С даты документа используется для снятия с регистрации с даты документа. Режим С прошлой даты используется для снятия с регистрации в прошлом и для последующего пересчета налога.
