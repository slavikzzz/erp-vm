Заполнение документа

Порядок учета затрат
  • По месту возникновения - используется, когда затраты во внешней Системе, до момента включения их в себестоимость продукции, учитываются по месту (подразделению) их возникновения.
  • По месту выпуска - используется, когда затраты во внешней Системе, с момента их возникновения, учитываются по месту (подразделению) выпуска продукции.
  • По текущему местонахождению партии - используется, когда затраты во внешней Системе перемещаются между местами их учета параллельно производственному процессу.

Затраты

Табличная часть предназначена для фиксации агрегированных остатков НЗП по данным внешней Системы, а также установки соответствия их аналитик внутренним аналитикам Системы.
Для работы с табличной частью предусмотрены следующие команды:

  • Команда Загрузить из файла... - предназначена для заполнения табличной части, посредством вызова стандартного диалога загрузки данных табличных документов.
  • Команды Заполнить тип затрат, Заполнить статью калькуляции, Заполнить правило отнесения на выпуск - предназначены для заполнения соответствующих реквизитов в выделенных строках табличной части.

По мере заполнения документа, в заголовке закладки, показывается информация о количестве строк табличной части.

Дополнительно

Направление деятельности - используется в качестве отбора при распределении остатков НЗП на партии производства.
Вид цены - используется для получения плановых цен материалов при распределении остатков НЗП на партии производства.

Ввод на основании

Распределение агрегированных остатков НЗП на партии производства выполняется посредством ввода документа "Начальные остатки НЗП по партиям производства" на основании документа "Начальные остатки НЗП по подразделениям".

Контекстные отчеты

Для анализа распределения остатков НЗП по партиям производства предназначены конекстные отчеты:

  • Анализ распределения начальных остатков НЗП - при построении выполняется предварительное распределение остатков НЗП по партиям производства, результат которого выводится в отчет.
  • Ошибки в данных распределения остатков НЗП - при построении выполняется предварительное распределение остатков НЗП по партиям производства. В отчет выводятся ошибки в данных базы распределения.
