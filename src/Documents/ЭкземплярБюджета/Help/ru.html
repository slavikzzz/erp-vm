Экземпляр бюджета

Документ предназначен для ввода плановых значений по выбранному Виду бюджета за определенный период в разрезе Сценария, а так же Организации и Подразделения, если они настроены в Модели бюджетирования.

Документ является основным инструментом для установки значений по Статьям бюджетов с детализацией до аналитики. Документ используется для первичного ввода плановых данных по бюджетам и для корректировок.

Дата документа, введенные планы (или изменения планов) используются в бюджетных отчетах, позволяя получать различные версии бюджетов на дату актуальности отчета.

Основные параметры планирования

На закладке Основное указываются Вид бюджета (бюджетная форма для планирования, согласно которой будут вводиться или рассчитываться данные планов), сценарий планирования, период планирования, основные аналитические разрезы планирования (если в бюджетной форме для планирования они предусмотрены).

Также есть возможность добавления дополнительных реквизитов в документ.

Граница фактических данных


Граница фактических данных - данная настройка доступна в документе, если в виде бюджета настроено смещение при заполнении статей бюджетов с использованием границы фактических данных. Дата заполняется по умолчанию из настроек вида бюджета, но может быть указана вручную для иного учета периода выборки фактических данных при расчетах.

Граница фактических данных применяется при предварительном планировании за несколько периодов до периода планирования. Например, когда нужно составить бюджет на год в сентябре предыдущего года.

Статусы бюджетов


Документ может находиться в различных статусах: Отклонен, В подготовке, На утверждении, Утвержден.

Статусы используются, если в модели бюджетирования включена настройка Утверждение бюджетов. Наличие статуса позволяет организовать работу над бюджетом (планом) в два этапа: Подготовка и Утверждение. Если используются Бюджетные процессы, то сотрудник, получивший задачу Ввод экземпляра бюджета (или плана), создает документ Экземпляр бюджета или документы планов. После заполнения документов в рамках процесса создается задача Утверждение бюджетов.

Редактирование документа возможно во всех статусах, кроме Утвержден. Плановые данные для бюджетных отчетов формируются во всех статусах, кроме Отменен.

В бюджетных отчетах существует настройка отбора данных с учетом статуса документов, что позволяет анализировать влияние еще не утвержденных документов на сводный план.

Проведение планирования

Выполнение планирования производится на закладке Бюджет.

Заполнение, расчет и редактирование документа


Ввод и редактирование документа осуществляется в двух режимах:

  • Редактировать в таблице - ввод данных в виде бюджетного отчета. В данном режиме возможно добавление аналитических измерений (если в структуре бюджетной формы предусмотрены такие аналитики) с использованием команд Добавить строку и Удалить строку;
  • Редактирование списком - ввод строк с данными по структуре бюджета с возможностью указания произвольной аналитики (если это разрешено в виде бюджета) и комментариев к вводимым значениям.


Автоматический пересчет - опция предназначена для включения/выключения пересчета на лету данных, зависимых от вводимой информации (например расчет граф Итого, Производные показатели) внутри Экземпляра бюджета. На больших объемах данных рекомендуется отключить автоматический пересчет и использовать кнопку Обновить, при нажатии на которую происходит обновление связанных данных.

Обновить осуществляет только пересчет связанных данных, не меняя при этом исходную информацию введенную в в Экземпляре Бюджета.

Заполнить по данным бюджетирования - осуществляет заполнение строк текущего Экземпляра бюджета по связям, установленным между статьями с другими сценариями и статьями вне текущего Экземпляра бюджета.
Заполнение перезаписывает информацию внутри текущего Экземпляра бюджета.

Выгрузка и загрузка из Excel


Возможна выгрузка и загрузка данных через Excel.
Экспорт осуществляется в присоединенные файлы, откуда при импорте выбирается файл с данными.
Выгрузка в Excel возможна только для однородных по используемой аналитике и уровням иерархии таблиц. Сложные по структуре таблицы, требующие загрузки из Excel, нужно настроить в виде нескольких таблиц.
Каждая настроенная в виде бюджета таблица выгружается и загружается отдельно, для этого нужно поставить на нее курсор в режиме табличного ввода или выбрать элемент в дереве структуры бюджета при режиме редактирования списком.
При невозможности автоматического опознания данных при импорте из Excel в открывшейся форме уточнения можно определить такие строки вручную.
