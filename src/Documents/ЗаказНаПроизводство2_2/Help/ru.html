Заказ на производство

Заказ на производство предназначен для оформления в системе потребности к выполнению производственных процессов разных типов и управления исполнением этой потребности. Документ доступен при включении функциональной опции НСИ и администрирование - Производство - Управление производством (версия 2.2).
Подробное описание приведено в разделе Заказы на производство в документации на сайте ИТС: https://its.1c.ru/db/erp25doc#bookmark:ProductionDecisionMaking:Version22ManufacturingOrders

Жизненный цикл заказа

Заказ проходит через формирование, передачу к производству, выполнение, закрытие. Для управления жизненным циклом используется статус.

Формирование заказов

Заказ на производство может формироваться различными способами:

  • Вручную
  • По плану
  • По потребности
  • На основании

Формирующийся заказ получает статус Формируется. В этом статусе с заказом могут работать пользователи, не относящиеся к производству. Заказ в статусе Формируется не вызывает никаких действий в производственной подсистеме.

Передача к производству

Сформированный заказ, по которому принято решение производить, передается к производству. Для этого заказу присваивается статус К производству. С заказами в статусе К производству дальше работает главный диспетчер производства, обеспечивая их исполнение. Заказ на производство исполняется с помощью этапов производства, описывающих производственный процесс.

Закрытие

Заказ, по которому больше ничего не будет производиться, переводится в статус Закрыт. Статус Закрыт ставится вручную, служит для индикации того, что по заказу все сделано и для защиты заказа от дальнейших действий. При проведении заказа на производство в статусе Закрыт выполняется контроль наличия невыполненных этапов производства (при включенной функциональной опции Не закрывать частично отработанные заказы на производство в разделе НСИ и администрирование - Производство). Для закрытия незавершенного заказ при включенном контроле необходимо откорректировать продукцию заказа и этапы производства по заказу, сократив и убрав то, что не будет производиться.
