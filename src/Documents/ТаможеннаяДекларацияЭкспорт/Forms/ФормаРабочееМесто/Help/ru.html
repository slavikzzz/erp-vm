Форма рабочее место

Рабочее место предназначено для работы с таможенными декларациями на экспорт.

На вкладке Таможенные декларации на экспорт отображается список зарегистрированных документов. Доступен отбор документов по статусу сбора сопроводительных документов.

На вкладке К оформлению выводятся распоряжения на регистрацию таможенных деклараций на экспорт. На вкладку попадают экспортные реализации, по которым еще не были зарегистрированы таможенный декларации.
С помощью команды Сформировать таможенную декларацию осуществляется создание документа по распоряжению. Для регистрации таможенной декларации по нескольким реализациям необходимо выделить несколько строк в таблице.
