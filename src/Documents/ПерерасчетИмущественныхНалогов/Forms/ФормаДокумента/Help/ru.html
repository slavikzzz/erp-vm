Перерасчет имущественных налогов

Документ предназначен для корректировки (доначисления, возмещения) имущественных налогов при обнаружении ошибок.
Сумму корректировки можно ввести вручную или рассчитать.

Отражение доходов

Основное
  • Вид налога - вид налога, для которого выполняется перерасчет;
  • Перерасчет с .. по .. - периода, за который выполняется перерасчет;
  • Основное средство - основное средство, для которого выполняется перерасчет;
  • Операция - допускается доначисление или возмещение налога. Операция может быть рассчитана автоматически на основании сделанных исправлений, для этого используется команда Рассчитать. Детальные сведения о расчете открываются с помощью команды Подробнее;
  • Вариант возмещения - доступно при возмещении налога и позволяет возмещаемую сумму зачесть в счет налога одного из налогов;
  • Сумма - сумма доначисление или возмещение налога.

Отражение доходов

На закладке указывается способ отражения корректировки в доходах или расходах.
