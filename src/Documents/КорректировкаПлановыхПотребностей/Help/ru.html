Назначение документа

Документ предназначен для уменьшения и увеличения плановых потребностей:

  • Для товара по которому уменьшается потребность, выводится текущая потребность. Можно редактировать количество на которое будет уменьшена потребность и конечную потребность.
  • Товар для которого увеличивается потребность отображает остаток или потребность, при её наличии (необязателен для заполнения).

Статусы корректировок

Корректировки вводятся в статусе Утверждена. Также для корректировки доступны статусы К проверке и Отменена.
При этом корректировки в статусе К проверке влияют на потребность. Чтобы корректировка перестала действовать её нужно перевести в статус Отменена. Пользователь принимает решение, что делать с корректировками на основании новой потребности: отменить/отредактировать и утвердить.

Режимы ввода

Режим ввода может быть Один к одному (одна номенклатура заменяется на другую), либо Многострочный (заменяются наборы).
