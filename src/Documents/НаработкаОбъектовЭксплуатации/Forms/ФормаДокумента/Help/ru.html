Наработка объектов эксплуатации

Документ предназначен для отражения текущих значений показателей эксплуатации, определяющих натуральное выражение степени износа объекта эксплуатации или узла объекта (машино-часы, километры и т.д.)

Перечень показателей наработки, предназначенный для заданного набора объектов эксплуатации или их узлов, указывается в карточке класса.

Для правильного заполнения документа в строках таблицы на закладке Наработки должны быть указаны:

  • Объект эксплуатации - объект, для которого регистрируется текущее значение наработки;
  • При необходимости может быть указан узел объекта - конструктивная, логическая или любая другая заданная часть указанного объекта эксплуатации, для которой регистрируется текущее значение наработки;
  • Показатель наработки - показатель текущего значения наработки;
  • Значение - текущее значение наработки. Например, если регистрируется пробег автомобиля, то нужно указать полный пробег, а не дельту с даты последней регистрации.

В качестве объекта эксплуатации или узла заданного объекта в соответствующих колонках могут быть указаны элементы, классификация которых предопределяет один или несколько показателей наработки регистрируемых независимо.

При проведении документа производится расчет текущих значений показателей наработок для объектов и узлов потребителей значения наработки. А так же производится расчет среднесуточных значений наработки на основе заданных классификацией параметров.
