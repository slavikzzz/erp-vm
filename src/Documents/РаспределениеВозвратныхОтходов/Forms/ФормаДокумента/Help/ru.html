Распределение возвратных отходов

Документ предназначен для распределения возвратных отходов, возникших при производстве продукции. При проведении документа производится распределение возвратных отходов, возникших при производстве продукции, на себестоимость производимой продукции.

Для настройки распределения доступны следующие закладки документа:

Основное

По правилу

По партиям производства

По расходам

Дополнительно

Основное

Содержит основные реквизиты документа, такие как номер, дата, комментарий, информацию о распределяемом возвратном отходе.

По правилу

Содержит настройки распределения указанного количества возвратного отхода согласно выбранному правилу распределения. При выборе правила распределения, его настройки копируются в документ и далее пользователь может их самостоятельно изменить в случае необходимости. Также есть возможность перенести введенные в документе настройки в выбранное правило распределения с помощью соответствующей кнопки на закладке По правилам. В зависимости от выбранного правила распределения, есть возможность сформировать контекстный отчет, вызываемый по гиперссылке Проверить наличие данных для базы распределения. Отчет показывает наличие данных, необходимых для формирования базы распределения.

По партиям производства

Содержит информацию о партиях производства, на которые нужно отнести указанное количество распределяемого возвратного отхода.
Команда Заполнить статью калькуляции - предназначена для выбора и заполнения статьи калькуляции в выделенных строках.

По расходам

Содержит информацию о статьях расходов, за счет которых формируется стоимость распределяемого количества возвратного отхода.

  • Команда Подразделение - предназначена для выбора и заполнения подразделения в выделенных строках.
  • Команда Заполнить статью расходов - предназначена для выбора и заполнения статьи расходов в выделенных строках.
  • Команда Заполнить статью активов и пассивов - предназначена для выбора и заполнения статьи расходов (статьей активов/пассивов) в выделенных строках.

Дополнительно

Содержит дополнительные реквизиты документа, такие как тип налогооблажения НДС, вид цен, направление деятельности, ответственного и другие.
