Документ предназначен для отражения принятия к учету узлов и компонентов амортизации основных средств.

Параметры учета

Основное
  • Организация и Подразделение - организация и подразделение, в которые принимаются основные средства;
  • НСИ и администрирование - Финансовый результат и контролинг - Финансовый результат - Финансовый результат по направлениям деятельности и дополнительной настройки в справочнике Направления деятельности объекта аналитического учета Учет внеоборотных активов;
  • Направление деятельности - Направление деятельности, для которого используются активы. Учет внеоборотных активов по направлениям деятельности может быть использован не только для фиксации целевого финансирования, но и для расширенной детализации учета затрат в рамках проектного учета. В этом случае аналитика Направление деятельности соответствует управленческой аналитике для раздельного учета затрат по проектам, в рамках которых формируется стоимость внеоборотных активов и ведется учет иных затрат и доходов;
  • Событие - событие, связанное с принятием к учету (доступны виды событий: "принятие к учету, принятие к учету с вводом в эксплуатацию").

Компоненты амортизации

На закладке указывается список принимаемых к учету узлов и компонентов основных средств и их ОС-владельцы.

С помощью команды Заполнить - Подобрать можно добавить несколько узлов и компонентов амортизации. В форме выбора имеется возможность выделить несколько строк списка и добавить их в документ с помощью команды Выбрать.

  • Предварительная стоимость - для каждого узла и компонента амортизации требуется заполнить первоначальную стоимость. Фактическая стоимость будет рассчитана при выполнении операции закрытия месяца Расчет стоимости ОС и НМА. При этом стоимость, указанная в документе, не изменяется, но отображается информационная надпись: "Предварительная стоимость отличается от фактической". При закрытии месяца проверяется, что предварительная стоимость не превышает фактическую;
  • Ликвидационная стоимость - сумма, которую организация ожидает получить за актив в конце срока его полезной службы за вычетом ожидаемых затрат на его выбытие.

Параметры учета


Бухгалтерский учет:

  • Порядок учета - доступны следующие варианты учета: Начислять амортизацию, Не начислять амортизацию, Не принимать к учету, включить в расходы;
  • Метод начисления амортизации - доступны следующие способы: Линейный, Пропорционально объему продукции (работ), Уменьшаемого остатка;
  • Срок использования - срок полезного использования в месяцах;
  • Коэффициент ускорения - поле доступно, если используется способ начисления Уменьшаемого остатка.


Управленческий учет:

  • Порядок учета - доступны следующие варианты учета: Начислять амортизацию, Не начислять амортизацию, Не принимать к учету, включить в расходы;
  • Метод начисления амортизации - доступны следующие способы: Линейный, Уменьшаемого остатка, Пропорционально объему продукции (работ);
  • Срок использования - срок полезного использования в месяцах;
  • Коэффициент ускорения - поле доступно, если используется метод начисления Уменьшаемого остатка.


Общее:

  • Группа финансового учета - определяет правила отражения в бухгалтерском (финансовом) учете операций с активом.
