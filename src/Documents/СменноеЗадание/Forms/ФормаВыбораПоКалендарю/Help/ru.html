Смены

Форма предназначена для выбора смен, доступных для формирования сменных заданий. В списке с возможностью выбора нескольких отображаются смены с указанием статуса соответствующего задания. Доступны для выбора только смены с формирующимися заданиями либо с пока еще не созданными заданиями. При выборе смены, не имеющей пока задания, сменное задание создастся автоматически.
