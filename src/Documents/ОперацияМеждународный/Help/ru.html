Операция (международный учет)

Документ предназначен для отражения в Международном финансовом учете проводок по хозяйственным операциям, для которых в конфигурации не создано документов оперативного учета.

При заполнении документа хозяйственная операция отражается в нем в виде бухгалтерской проводки по плану счетов международного учета.

Возможны следующие способы заполнения документа:

  • ручной ввод корреспонденции счетов и суммы проводки;
  • загрузка данных из табличного документа, например из таблицы Excel;
  • формирование исправительных сторно или реверсивных проводок по документам;
  • созданием на основании другой операции международного учета, как его исправления;
  • созданием на основании из проводок международного учета в запрещенном для изменений периоде;
  • загрузка из ранее сохраненной типовой операции.


Типовая операция представляет собой настроенную и сохраненную для повторного использования Операцию (международный учет).

При загрузке данных из одного табличного документа в несколько документов Операция (международный учет (команда Загрузить из формы списка документов) - проводки в табличном документе должны быть отсортированы по организациям и датам.
