Основные средства

Форма предназначена для отражения начальных остатков основных средств и полученных предметов лизинга, учитываемых на балансе.

В форме документа указывается Организация, Местонахождение к которым относятся основные средства и предметы лизинга.

Ввод детальных данных об активах осуществляется в специальной форме, которая автоматически открывается при добавлении новой или редактировании существующей строки табличной части документа.
Детальные данные рекомендуется заполнять после правильного заполнения организации и даты, т.к. их заполнение зависит от применяемой учетной политики.
Если после ввода детальных данных изменить дату или организацию, то данные могут быть очищены, если они не соответствуют выбранной учетной политике.

Ввод остатков возможен одновременно по оперативному, бухгалтерскому и налоговому учетам, управленческому учету на едином плане счетов регламентированного учета или по каждому из учетов отдельно.
В случае ввода остатков по оперативному учету, документ отражается только по регистрам оперативного учета.
При вводе остатков только по бухгалтерскому и налоговому учетам или по управленческому учету, документ отражается только по соответствующим ресурсам регистра бухгалтерии.

При вводе остатков в оперативном учете доступен выбор в каком из учетов числятся основные средства: Управленческом или Регламентированном.

Примечание. Если документ вводится до начала учета внеоборотных активов версии 2.4, то он не отражается в регламентированном учете и используется для перехода с версии 2.2 на учет версии 2.4 (т.е. используется для переноса данных).
