Предметы лизинга за балансом

Форма предназначена для отражения начальных остатков полученных предметов лизинга, учитываемых за балансом.

Ввод остатков возможен одновременно по оперативному, бухгалтерскому и налоговому учетам или по каждому из учетов отдельно.
В случае ввода остатков по оперативному учету, документ отражается только по регистрам оперативного учета.
При вводе остатков только по бухгалтерскому и налоговому учетам, документ отражается только по соответствующим ресурсам регистра бухгалтерии.

Примечание. Если документ вводится до начала учета внеоборотных активов версии 2.4, то он не отражается в регламентированном учете и используется для перехода с версии 2.2 на учет версии 2.4 (т.е. используется для переноса данных).

Основное

На закладке указываются Организация, Арендодатель, Местонахождение к которым относятся предметы лизинга.
Группа финансового учета определяет правила отражения в бухгалтерском (финансовом) учете операций с основными средствами.

Основные средства
  • Основное средство - предмета лизинга;
  • Дата поступления объекта основного средства в учет;
  • Договор лизинга - договор на основании, которого основное средство получено в лизинг;
  • Залоговая стоимость - залоговая стоимость предмета лизинга.

Остатки вводятся только в Регламентированном учете.
