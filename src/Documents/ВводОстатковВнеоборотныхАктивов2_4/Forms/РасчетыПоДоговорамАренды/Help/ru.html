Расчеты по договорам аренды

Форма предназначена для отражения начальных остатков по взаиморасчетам по договорам лизинга.

Ввод остатков возможен одновременно по оперативному, бухгалтерскому и налоговому учетам, управленческому учету на едином плане счетов регламентированного учета или по каждому из учетов отдельно.
В случае ввода остатков по оперативному учету, документ отражается только по регистрам оперативного учета.
При вводе остатков только по бухгалтерскому и налоговому учетам или по управленческому учету, документ отражается только по соответствующим ресурсам регистра бухгалтерии.

Основное

На закладке указываются Организация, Контрагент, по которым вводятся взаиморасчеты.

Расчеты
  • Договор - договор лизинга;
  • Тип задолженности - тип задолженности по лизингу;
  • Счет учета - счет учета остатка задолженности по взаиморасчетам. Автоматически заполняется по данным группы финансового учета, указанной в договоре;
  • Признак Сальдо обязателен для заполнения и ограничен значениями: Дебетовое и Кредитовое;
  • Сумма остатка по указанному типу задолженности в заданной валюте, а так же в валюте регламентированного учета, в зависимости от настроек указанного Договора лизинга.
