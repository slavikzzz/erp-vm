Начисление и списание резервов предстоящих расходов

Рабочее место предназначено для создания документов Начисления и списания резервов предстоящих расходов по распоряжениям. Распоряжения выводятся на вкладке К отражению. Распоряжения создаются по организациям и видам резервов, исходя наличия актуальных на дату Объектов учета резервов.

Для создания документа по организации и виду резерва необходимо выполнить команду Создать документ. При создании документ будет заполнен списком объектов учета резервов, для которых необходимо выполнить оценку резервов на дату.

На вкладке Список документов выводятся ранее сформированыне документы с отбором по Организации.
