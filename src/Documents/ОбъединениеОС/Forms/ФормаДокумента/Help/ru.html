Объединение ОС

Документ предназначен для отражения объединения основных средств, например, когда из нескольких земельных участков образуется один.
Исходные основные средства всегда снимаются с учета.

Параметры учета

Основное
  • Организация и Подразделение - Организация и подразделение, в которых числятся основное средство;
  • Основное средство - ОС, которое получено при объединении или к которому присоединяются выбранные основные средства;
  • Операция - доступны операции: Объединение в новое ОС - позволяет принять к учету новое ОС, объединив несколько исходных ОС, Присоединение к существующему ОС - позволяет к принятому к учету ОС присоединить одно или несколько других ОС;
  • Событие - событие, связанное с объединением (доступны виды событий: "реконструкция", "достройка", "дооборудование", "прочее");
  • Объединение однотипных ОС - позволяет упростить ввод параметров учета. Если опция установлена, то параметры будут совпадать с параметрами первого ОС. При этом будет ограничен выбор ОС - можно выбрать ОС, у которых совпадают: группа ОС, амортизационная группа, СПИ, налогообложение, направление деятельности. Если опция не установлена, то параметры учета указываются в документе. При этом выбор ОС не ограничен - можно выбрать ОС, у которых разные параметры учета. Присоединение к существующему ОС всегда выполняется для однотипных ОС.

Параметры учета

Если снят флаг Объединение однотипных ОС, то на закладке будут доступны параметры учета нового ОС.

Во всех случаях нужно указать Остаточный срок использования или Остаточный объем наработки, которые используются для начисления амортизации
Эти параметры не нужно указывать, если используется порядок учета Не начислять амортизацию.
Параметры не должны превышать нормативный срок использования (проверяется при проведении).

Если хотя бы для одного исходного ОС используется целевое финансирование, то требуется указать статью и аналитику доходов.
Статья доходов используется при начислении амортизации для формирования проводки "Признание доходов от целевого финансирования".

Дата принятия к учету нового ОС совпадает с датой документа.
При начислении амортизации используется остаточная стоимость.

ЕГДС - генерирующая единица, в состав которой включается объединенное основное средство.
При наличии остатков обесценения по объединяемым основным средствам:

  • для ОС относящихся к той же ЕГДС - остатки обесценения суммируются;
  • для ОС относящихся к другой ЕГДС - остатки обесценения восстанавливаются.
