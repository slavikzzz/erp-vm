Объединение ОС

Документ предназначен для отражения объединения основных средств, например, когда из нескольких земельных участков образуется один.
Исходные основные средства всегда снимаются с учета.

Ограничения

Ограничения

В месяце, в котором оформлено объединение ОС, не должно быть принятия к учету и модернизации ОС, которые объединяются
Ограничение действует, если амортизация начисляется по стандартам международного учета и документ отражен в упр. учете.
Для обхода нужно оформить документ в упр. учете в следующем месяце.
Проверяется при закрытии месяца.

Не допускается объединять ОС, полученные или переданные в аренду. Проверяется при проведении документа.

Если хотя бы у одного исходного ОС заполнено направление деятельности, то у остальных оно должно либо совпадать, либо быть пустым, а у нового ОС всегда должно совпадать.
Проверяется при проведении документа.

Не допускается объединять основные средства, если только для некоторых из них начисляется износ по ЕНАОФ.
Проверяется при проведении документа.
