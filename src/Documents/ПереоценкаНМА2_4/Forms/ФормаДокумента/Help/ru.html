Переоценка НМА

Документ предназначен для отражения переоценки нематериальных активов (переоценка расходов на НИОКР не поддерживается).

Основное

Нематериальные активы

Основное
  • Организация и Подразделение - организация и подразделение, в которых числятся нематериальные активы;
  • Уценка накопленной амортизации - задается статья, по которой будет отражена уценка накопленной амортизации;
  • Уценка восстановительной стоимости - указывается статья, по которой будет отражена уценка восстановительной стоимости.

Нематериальные активы

На закладке выбираются нематериальные активы, по которым выполняется переоценка, и их стоимость в регламентированном и управленческом учетах.

С помощью команды Подобрать можно добавить несколько нематериальных активов. Для быстрого заполнения списка группой однотипных нематериальных активов необходимо в форме выбора отобрать нематериальные активы по наименованию, выделить несколько строк списка и добавить их в документ с помощью команды Выбрать.
