Резервы по оплате труда

Документ предназначен для учета резервов (оценочных обязательств) по отпускам, вознаграждений по итогам года и за выслугу лет. Функционал используется для резервов с 2023 года.

Подробное описание приведено в статье Учет резервов (оценочных обязательств) по оплате труда в документации на сайте ИТС https://its.1c.ru/db/erpdoc#bookmark:EstimatedLiabilities:EstimatedLiabilities.
