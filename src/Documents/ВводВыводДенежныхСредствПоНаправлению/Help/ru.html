Ввод / вывод денежных средств по направлению

Документ предназначен для учета денежных средств по направлениям деятельности и позволяет перемещать выделенные средства между направлениями без их фактического движения по расчетным счетам и кассам (для фиксации факта вложения\вывода средств по направлениям деятельности в отчетах).

При включенной настройке Остатки денежных средств определяются по корреспонденции приходов и расходов этого направления, активируется режим косвенного определения денежных средств по кассам и банковским счетам, для которых направление деятельности не указывается.

Принадлежность денежных средств определяется по корреспонденции поступления и списания денежных средств по направлениям деятельности. Данный способ удобен для организаций, которые не могут явно разделить расчетные счета и кассы.

Разделение денежных средств по направлениям происходит виртуально, без реальных движений и перебросок между расчетными счетами и кассами. Денежные средства в направлении не делятся на наличные и безналичные и по кассам и счетам. Деньги в направлении могут поступать на один расчетный счет по объектам расчета из направления, а тратиться из другого расчетного счета или кассы по объектам расчета этого же направления.

Для ведения денежных средств в направлениях предусмотрена предопределенная статья активов и пассивов Денежные средства (в направлениях). Для анализа средств в направлениях используется отчет Управленческий баланс.

В случаях ввода и вывода средств из/в пустого направления используются предопределенные статьи активов и пассивов Вложения собственных средств и Вывод собственных средств.
Учет по служебным статьям ведется без разрезов по Организациям и Подразделениям, с детализацией только по Направлениям деятельности.
