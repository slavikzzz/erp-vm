Подготовка к передаче ОС

Документ предназначен для отражения подготовки к переходу права собственности основных средств.

Для оформления фактической передачи необходимо создать документ Реализация услуг и прочих активов с операцией Реализация внеоборотных активов или Реализация ОС с отложенным переходом прав. Документ можно создать, используя механизм ввода на основании.

Основные средства

Основное
  • Организация и Подразделение - организация и подразделение, в которых числятся основные средства;
  • Клиент, Контрагент - покупатель объектов основных средств.
  • Валюта - валюта, в которой указываются стоимости продажи основных средств;
  • Событие - событие, связанное с подготовкой к передаче (доступен только вид события "подготовка к передаче").

Основные средства

На закладке указываются основные средства, предназначенные для передачи (продажи).

С помощью команды Подобрать можно добавить несколько основных средств. Для быстрого заполнения списка группой однотипных основных средств необходимо в форме выбора отобрать основные средства по наименованию, выделить несколько строк списка и добавить их в документ с помощью команды Выбрать.
