Разукомплектация ОС

Документ предназначен для отражения разукомплектации основного средства.
Например, в следующих случаях: консервация части здания, передача в аренду части здания, разделение земельного участка на несколько земельных участков.

Амортизация

Основное
  • Организация и Подразделение - Организация и подразделение, в которых числится основное средство;
  • Основное средство - ОС, которое нужно разукомплектовать;
  • Операция - доступны операции: Разукомплектация ОС - позволяет полностью перенести стоимость и амортизацию исходного ОС на новые ОС (исходный ОС снимается с учета), Частичная разукомплектация ОС - позволяет частично перенести стоимость и амортизацию исходного ОС на новые ОС (исходный ОС остается в учете), Выделение узлов и компонентов амортизации - позволяет выделить из исходного ОС узел или компонент амортизации (исходный ОС остается в учете);
  • Событие - событие, связанное с разукомплектацией (доступны виды событий: "реконструкция", "прочее").

Принимаемые к учету ОС
  • Основное средство - Основное средство, которое получено при разукомплектации. Группа основного средства, амортизационная группа и направление деятельности должны быть аналогичными;
  • Стоимость в регламентированном и управленческом учетах - Указывается фактическая стоимость, которая не может превышать стоимость исходного основного средства (проверяется при закрытии месяца);
  • Ликвидационная стоимость - Сумма, которую организация ожидает получить за актив в конце срока его полезной службы за вычетом ожидаемых затрат на его выбытие.

Параметры учета полученных основных средств совпадают с параметрами учета исходного основного средства. Для их изменения можно оформить документ Изменение параметров ОС.

Накопленная амортизация для новых ОС рассчитывается при закрытии месяца по формуле:
[Накопленная амортизация исходного ОС] * [Стоимость нового ОС] / [Восстановительная стоимость исходного ОС]

Амортизация

Для хозяйственной операции Выделение узлов и компонентов амортизации необходимо указать данные, используемые при начислении амортизации.


Бухгалтерский учет:

  • Метод начисления - доступны следующие методы начисления: Линейный, Уменьшаемого остатка, Пропорционально объему продукции (работ);
  • Срок использования - срок полезного использования в месяцах;
  • Коэффициент ускорения - поле доступно, если используется метод начисления Уменьшаемого остатка;
  • Годовая норма - поле доступно, если используется метод начисления;
  • График амортизации - используется для начисления амортизации по заданному графику, например в организациях с сезонным характером производства.


Управленческий учет:

  • Метод начисления - доступны следующие методы начисления: Линейный, Способ уменьшаемого остатка, Пропорционально объему продукции (работ);
  • Срок использования - срок полезного использования в месяцах;
  • Коэффициент ускорения - поле доступно, если используется метод начисления Способ уменьшаемого остатка.


Общее

  • Группа финансового учета - определяет правила отражения в бухгалтерском (финансовом) учете операций с основным средством;
  • Показатель наработки - выбирается один из доступных показателей наработки. Поле доступно, если используется метод начисления Пропорционально объему продукции (работ);
  • Объем наработки - указывается предельный объем наработки в указанной единице измерения. Поле доступно если используется метод начисления Пропорционально объему продукции (работ).
