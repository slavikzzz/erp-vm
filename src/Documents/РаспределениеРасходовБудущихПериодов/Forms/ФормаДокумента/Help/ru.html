Распределение по периодам

Документ распределения предлагает два варианта указания периода распределения:

  • Произвольный - позволяет указать произвольные даты начала и окончания распределения РБП.
  • Количество месяцев - позволяет указать дату начала списания РБП и количество месяцев, на которые необходимо распределить расходы.


Автоматически распределить сумму расходов по месяцам можно выбрав одну из комманд группы Заполнить на закладке Распределение по месяцам документа распределения.
В зависимости от выбранного варианта указания периода распределения сумма будет распределена по месяцам между датой начала и датой окончания распределения, либо начиная с даты начала распределения на указанное количество месяцев. Распределение будет происходить в соответствии с количеством дней каждого из месяцев распределения. Статья расходов будет заполнена в соответствии с той статьей расходов, которая указана для распределения в статье расходов с видом распределения На расходы будущих периодов. Подразделение-получатель расхода будет соответствовать подразделению, в котором появились исходные распределяемые расходы.
