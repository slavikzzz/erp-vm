Списание ОС (2.2)

Документ предназначен для отражения в системе факта полной или частичной ликвидации перечня основных средств.

Основное


При оформлении документа уточняется:

  • Списание на основании - ссылка на документ, на основании которого создается списание объекта ОС;
  • Местонахождение ОС - подразделение организации, которое списывает основные средства;
  • Причина списания - причина списания основных средств (будет указана в печатных формах);
  • Событие ОС - наименование события жизненного цикла основного средства, которое отражается документом;
  • Признак Частичная ликвидация - отметка подтверждающая выполнение операции частичной ликвидации. При отсутствии признака выполняется обычная операция списания;
  • Статья и аналитика расходов - статья расходов и соответствующая ей аналитика, по которой отражаются расходы от списания основного средства.

Основные средства


На закладке уточняется список основных средств, предназначенных для списания. При указании события с видом Частичная ликвидация или одноименного флага, для каждого из основных средств заполняется Сумма частичной ликвидации.

Для быстрого заполнения документа группой однотипных основных средств необходимо ввести хотя бы один такой объект и автоматически заполнить табличное поле основными средствами с таким же наименованием с помощью команды Заполнить - По наименованию.

Приходуемые материальные ценности


На закладке указываются возникающие при списании (ликвидации) основного средства материальные ценности (например, драгоценные металлы). Поступление материальных ценностей осуществляется из группы команд Создать на основании при помощи документа Прочее оприходование товаров.

Печатные формы


Для документа доступны печатные формы:

  • Акт о списании объекта основных средств ОС-4 (кроме автотранспортных средств).
  • Акт о списании автотранспортных средств ОС-4а (формируется при списании основных средств с признаком Автотранспорт).
  • Акт о списании групп объектов основных средств ОС-4б (формируется при списании более, чем одного основного средства).
