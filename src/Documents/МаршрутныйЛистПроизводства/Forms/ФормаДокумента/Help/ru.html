Маршрутный лист производства

Документ отражает выполнение этапа графика производства и оформляется на определенное количество единиц или партий изделия.
Маршрутные листы оформляются в рабочих местах Диспетчирование (ББВ), Диспетчирование (регистрация операций) и Диспетчирование (MES), на основании графика производства (Заказов на производство, переданных в производство).

Использование статусов документа

В документе используются следующие статусы:

  • Создан - документ находится на стадии подготовки к передаче в производство: уточняются нормативы, назначаются бригады по видам работ, формируется расписание по рабочим центрам;
  • К выполнению - документ подготовлен и передан к выполнению в производство. В рабочем месте по выполнению маршрутных листов будет доступна информация о том, что нужно выполнить данный маршрутный лист. В рабочем месте по выполнению операций станут доступны для диспетчеризации операции данного маршрутного листа, если он принадлежит подразделению с пооперационной методикой управления;
  • Выполняется - показывает, что в производстве началось выполнение данного маршрутного листа. Статус не используется для маршрутных листов, принадлежащих подразделениям с методиками управления Пооперационное планирование и Регистрация операций (без планирования);
  • Выполнен - показывает, что выполнение завершено и документ может использоваться для формирования выпуска продукции и затрат на производство;
  • Отменен - показывает, что документ отменен в результате перепланирования графика производства. Недоступен для выбора, устанавливается автоматически при перепланировании графика.

Заполнение документа

Документ заполняется двумя способами.

  • По данным заказа на производство. Данный способ используется, если в этапе производства не указана Маршрутная карта.
  • По данным Маршрутной карты. Данный способ используется, если в этапе производства указана Маршрутная карта.

В списках выходных изделий доступно указание направления выпуска: на склад или в подразделение. При заполнении документа направление выпуска заполняется по данным спецификации заказа на производство. Полуфабрикаты, определенные в спецификации заказа как Производимые в процессе на закладках выходных изделий и материалов и услуг выделяются коричневым. Для таких позиций редактирование ключевых полей недоступно.

Заполнение маршрута производства

Вид маршрута зависит от выбранной методики управления маршрутными листами для подразделения.

При использовании методики ББВ отмечается прохождение контрольных следующих точек маршрута:

  • Начато выполнение маршрутного листа - отмечается, когда выполнение началось;
  • Готовность к работе ключевого рабочего центра - отмечается, когда все готово к работе ключевого рабочего центра. Прохождение этой контрольной точки влияет на статус предварительного буфера;
  • Завершена работа ключевого рабочего центра - отмечается, когда рабочий центр завершил выполнение данного Маршрутного листа. Прохождение этой контрольной точки влияет на статус работы рабочего центра (отставание или опережение графика работы);
  • Завершено выполнение маршрутного листа - отмечается, когда выполнение маршрутного листа завершено. Прохождение этой контрольной точки влияет на статус завершающего буфера.


При использовании методики УББВ отмечается прохождение следующих контрольных точек маршрута:

  • Начато выполнение маршрутного листа - отмечается, когда выполнение маршрутного листа началось;
  • Завершено выполнение маршрутного листа - отмечается, когда выполнение маршрутного листа завершено. Прохождение этой контрольной точки влияет на статус завершающего буфера.


При использовании методик Пооперационное планирование и Регистрация операций (без планирования) маршрут производства заполняется на основании данных маршрутной карты этапа производства, и содержит перечень выполняемых операций. Операции включают следующие реквизиты:

  • Приоритет (доступен только для методики Регистрация операций (без планирования)) - приоритет выполнения операции;
  • Выполнение - текущий статус выполнения операции. Изменить статус операций можно при помощи группы команд Выполнение, а так же в отдельном рабочем месте Выполнение операций. При переводе маршрутного листа в статус Выполнен, всем операциям документа автоматически будет установлен статус Завершено;
  • Начало и Окончание (доступны только для методики Пооперационное планирование) - плановые сроки выполнения операции, рассчитываются при планировании пооперационного расписания;
  • Вид рабочего центра и Рабочий центр - оборудование, на котором выполняется операция;
  • Вариант наладки - вариант наладки оборудования при выполнении операции;
  • Вспомогательные РЦ (доступен только для методики Пооперационное планирование) - список дополнительного оборудования, загрузка которого планируется совместно с основным рабочим центром;
  • Время штучное - время выполнения операции для одной единицы/партии изделий;
  • Время ПЗ - подготовительно-заключительное время при выполнении операции;
  • Время общее - совокупное время выполнения операции. Для параллельно загружаемых рабочих центров определяется как сумма штучного и подготовительно-заключительного времени. Для рабочих центров не загружаемых параллельно рассчитывается по формуле: (время штучное * количество единиц/партий изделий) + подготовительно-заключительное время;
  • Загрузка штучная - загрузка рабочего центра при выполнении операции для одной единицы/партии изделий, заполняется только для параллельно загружаемых рабочих центров;
  • Непрерывная (доступен только для методики Пооперационное планирование) - признак того, что выполнение операции не должно прерываться интервалами недоступности (нерабочее время).

Ввод фактического расхода материалов, выпуска изделий и выполнения работ

При передаче маршрутного листа к выполнению данные о фактическом расходе материалов и выполнении работ автоматически заполняются по указанным нормативам.
Если при выполнении маршрутного листа возникли отклонения, то их нужно отразить используя колонки Факт, Отклонение, доступные для каждой табличной части.

Если для изделий или материалов требуется указывать серии, то нужно использовать команду Указать серии.
Необходимость указания серий определяется настройкой ведения учета серий в производстве заданной в виде номенклатуры.
Если для материала применяется тип политики учета серий Управление по FEFO остатками серий, то для заполнения серий доступна команда Заполнить серии по FEFO (из группы команд Ещё).

Для маршрутных листов, находящихся в процессе выполнения, с помощью поля Израсходован можно отметить, какие материалы уже израсходованы и указать дату расхода. По дате расхода определяется отчетный период, в котором произошло потребление материала (работы).
Отметка и дата расхода заполняются автоматически при отметке выполнения маршрутного листа или при отметке выполнения операций.

Печатные формы

Для документа Маршрутный лист производства можно вывести на печать следующие печатные формы:

  • Маршрутный лист - форма маршрутного листа, которая может сопровождать партию изделий в процессе производства. Форма содержит изделия, требуемые материалы и виды работ. В форме можно отразить фактический расход материалов, выпуск изделий и выполнение работ.
