Перемещение ОС

Документ предназначен для отражения перемещения основных средств в другое подразделение или в обособленное подразделение, выделенное на отдельный баланс, а также передачи основных средств от одного материально-ответственного лица к другому.

Основное
  • Операция - позволяет определить порядок перемещения.
  • Сдатчик и Получатель - в полях этих группы указывается откуда и куда перемещаются основные средства.
  • Событие - событие, связанное с перемещением (доступен только вид события "внутреннее перемещение").
  • ЕГДС - генерирующая единица, в состав активов которой должны быть включены перемещаемые основные средства.

Основные средства

На закладке указываются перемещаемые основные средства.

Если выбрана операция Перемещение узлов основных средств, то необходимо указать новое ОС-владелец.

С помощью команды Подобрать можно добавить несколько основных средств. Для быстрого заполнения списка группой однотипных основных средств необходимо в форме выбора отобрать основные средства по наименованию, выделить несколько строк списка и добавить их в документ с помощью команды Выбрать.

Отражение расходов

На закладке указываются параметры учета у получателя.
