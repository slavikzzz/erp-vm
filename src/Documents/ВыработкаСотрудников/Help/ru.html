Выработка сотрудников

Документ предназначен для оформления выполненных производственных работ бригадами и работниками. Если исполнителем работ является бригада, то в документе выполняется распределение общей суммы по членам бригады - работникам. После оформления выработки данные о работниках и нормативной стоимости выполненных работ передаются в подсистему расчета зарплаты для расчета начислений (если включен расчет зарплаты).

Для ведения кадрового учета и расчета зарплаты в рамках текущей конфигурации требуется включить функциональную опцию Использовать программу для кадрового учета и расчета заработной платы в разделе НСИ и администрирование - Кадры.

В подсистеме расчета зарплаты нормативная стоимость выполненных работ регистрируется по показателю Сдельный заработок.

Отнесение фактически начисленной зарплаты сотрудников на партии производства выполняется при отражении зарплаты в финансовом учете.

Исполнителями работ могут бригады и работники, вид исполнителей задается в карточке производственного подразделения. Исполнитель указывается в распоряжении на выполнение работ и оформление выработки.

Распоряжения на оформление

Выработка оформляется по документам:

  • Этап производства
  • Производство без заказов
  • Производственная операция
  • Производство без заказов

Выработку можно создать на основании сменного задания, распоряжениями будут указаны производственные операции этого сменного задания. При создании выработки на основании операций или сменных заданий статья калькуляции берется из маршрутной карты каждой операции.
Возможность детализация выработки сотрудников до конкретных производственных операций включается функциональной опцией Оформлять выработку сотрудников на основании производственных операций в разделе НСИ и администрирование - Производство - Операции.

Оформление по бригаде

Если работы выполнялись группой работников, то в шапке документа указывается бригада, период, в котором выполнялись работы, и показатели для распределения общей суммы по членам бригады - работникам. В качестве показателей могут использоваться: коэффициенты трудового участия, отработанное время, тарифный разряд. Кроме этого, сумма по работникам может быть распределена вручную.

Выполненные работы

В списке выполненных работ перечисляются виды работ, выполненные бригадой, в разрезе распоряжений. Для каждого вида работ заполняется заявленное количество (по распоряжениям), количество к оплате, расценка и рассчитывается сумма. Итоговая сумма табличной части образует сумму оплаты по бригаде за весь объем выполненных работ.
По команде Заполнить по распоряжениям список заполняется видами работ за период из шапки документа, по которым выработка еще не оформлена. По команде Подобрать виды работ доступна форма подбора.

Работники

Список содержит работников - членов бригады. Для каждого работника заполняются показатели для расчета суммы оплаты. Состав показателей задается в шапке документа.
По команде Заполнить по предыдущему составу список заполняется работниками по прошлой выработке этой бригады. При изменении состава бригады требуется добавить или удалить работника из списка, указанный состав будет применяться при следующем оформлении документа.

Оформление по работникам

При оформлении выработки по работникам в шапке документа указывает подразделение и период выполнения работ. Список работ и исполнителей указывается в списке выполненных работ.

Выполненные работы

В списке перечисляются работники, распоряжения на оформление и виды работ. В каждой строке заполняется заявленное количество (по распоряжениям), количество к оплате, расценка и рассчитывается сумма.
Расценка определяется по квалификационному разряду исполнителя: если высококвалифицированный работник выполнял работы более низкой квалификации, то расценка будет подобрана исходя из квалификации работника (более высокой).
Минимальная квалификация указывается в карточке вида работ, а расценки в разрезе квалификаций исполнителей в регистре Расценки работ сотрудников. Если квалификационный разряд исполнителя в списке выполненных работ не заполнен, то подбирается расценка по квалификации вида работ.

По команде Заполнить по распоряжениям список заполняется видами работ за период из шапки документа, по которым выработка еще не оформлена. По команде Подобрать виды работ доступна форма подбора.

Устаревшие режимы

Начиная с версии 2.4.2 упразднен функционал по оформлению выработки по прочим работам и ремонтам. Для отражения таких операций рекомендуется оформить партию производства (например, производство без заказа) и оформить выработку по партии производства стандартным способом. Выходное изделие партии производства списать на статью расходов или включить в заказ на ремонт в качестве материала и работы, требуемой для ремонта.
