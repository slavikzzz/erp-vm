Уведомление об остатках прослеживаемых товаров

Документ предназначен для формирования уведомлений об имеющихся остатках товаров, подлежащих прослеживаемости.
По всем импортным товарам, подлежащим прослеживаемости, необходимо отправить уведомления об остатках в контролирующие органы и получить РНПТ.

Оформление уведомлений

Распоряжениями для оформления уведомлений об остатках является наличие остатков собственных товаров организаций с прослеживаемыми кодами ТН ВЭД и номерами ГТД (или отсутствием номера ГТД).

Уведомление можно создать:

  • по распоряжениям - предусмотрена специальная форма Оформление уведомлений об остатках прослеживаемых товаров, доступная по гиперссылке Оформить в рабочем месте по учету прослеживаемости импортных товаров.
  • не по распоряжениям из журнала документов Уведомления по прослеживаемым товарам, доступного по гиперссылке в рабочем месте по учету прослеживаемости импортных товаров (команда Создать – Уведомление об остатках прослеживаемых товаров).

Уведомление об остатках заполняется на основании первичного учетного документа, подтверждающего право собственности участника оборота товаров, подлежащих прослеживаемости. В шапке уведомления заполняется организация, код ТН ВЭД, единица измерения заполняется единицей прослеживаемости из кода ТН ВЭД. Одно уведомление может быть отправлено только по одному коду ТН ВЭД. Уведомление формируется по остаткам прослеживаемых товаров, входящих в состав запасов или основных средств (указывается в шапке документа).

При наличии нескольких единиц измерения по одной строке будет создано несколько уведомлений. Несколько мест хранения сворачиваются в одно уведомление при совпадении организации, единицы измерения и кода ТН ВЭД.
Уведомление может оформляться на товары, указанные в нескольких строках первичного учетного документа с одним и тем же кодом единой Товарной номенклатурой внешнеэкономической деятельности Евразийского экономического союза и единицей измерения.

При необходимости табличную часть уведомления можно перезаполнить остатками прослеживаемых товаров с номерами ГТД без установленного признака РНПТ (команда Заполнить – Заполнить по остаткам).

Отправка уведомлений

Ожидание, получение и создание РНПТ

Документом уведомления об остатках прослеживаемых товаров списываются остатки с номером ГТД без признака РНПТ и приходуются остатки с номером ГТД с признаком РНПТ.

Для регистрационного номера партии товаров страна происхождения определяется из номера ГТД остатков. РНПТ записывается в наименование и регистрационный номер создаваемых элементов, номеров грузовых таможенных деклараций.
