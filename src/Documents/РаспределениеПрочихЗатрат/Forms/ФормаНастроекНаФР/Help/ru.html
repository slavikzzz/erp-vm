Распределение расходов

Документ предназначен для распределения постатейных расходов между партиями производства и направлениями деятельности. Для выполнения распределения в списке отражены нераспределенные расходы с детализацией по статьям расходов и аналитике расходов.

Для настройки распределения двойным щелчком необходимо вызвать окно настроек, либо выделить перечень строк в списке расходов и нажать кнопку "Сформировать документы распределения". Документ по умолчанию заполняется на основании правила распределения расходов заданного в статье расходов.

При включенной опции "Учет постатейных расходов по объектам возникновения затрат" документ также позволяет распределять постатейные расходы, зарегистрированные на объектах возникновения затрат (ОВЗ). В таком случае в документе не указываются статьи расходов и направления деятельности, т.к. настройка применяется ко всем расходам, накопленным в конкретном ОВЗ. При этом расходы, обособленные под направление деятельности распределяются только в пределах своего направления деятельности. Расходы без указания направления деятельности распределяются на все направления.
Помимо стандартных вариантов распределения для ОВЗ доступен вариант "Распределять между ОВЗ". Это позволяет осуществить транзит накопленных расходов на другие ОВЗ. Доли распределения между ОВЗ задаются с помощью показателя распределения.
