Операция (регламентированный учет)

Документ предназначен для отражения в регламентированном учете проводок по хозяйственным операциям, для которых в конфигурации не создано документов оперативного учета.

При заполнении документа хозяйственная операция отражается в нем в виде бухгалтерской проводки по плану счетов.

Документ предназначен для выполнения следующих операций:

  • Ввод корреспонденции счетов по бухгалтерскому и налоговому учету
  • Сторнирование записей одного или нескольких документов


Для сторнирования документа необходимо указать связанный документ или выбрать список документов и нажать кнопку Заполнить сторно. Сторнируемые проводки заполнятся автоматически.

Для документа предусмотрена печатная форма Бухгалтерская справка.
