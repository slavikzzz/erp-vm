Отчет давальцу

Отчет давальцу это финансовый документ, который регистрирует оказание услуги и результирующую стоимость переработки. В отчете указывается перечень выпущенной из давальческого сырья продукции, ее количество и стоимость услуг по переработке каждой отдельной позиции.

Оформление отчета давальцу возможно после оформления выпуска продукции. Отчет давальцу и передача давальцу оформляется независимо. Отчет вводится только по Заказу давальца.
В концепции управления производством версии 2.2 отчет давальцу оформляется после установки отметки о производстве продукции в табличной части Выходные изделия документа Этап производства или после оформления производства без заказов.

Основные данные

Давалец - клиент, которому оказана услуга по переработке. Договор определяет параметры взаимоотношений с давальцем и указывается в заказе давальца. Если в договоре установлено, что расчеты ведутся по накладным, то объектом расчетов является отчет давальцу.

В группе Услуга по переработке указывается номенклатура с типом Услуга, которая будет определять значения по умолчанию Ставки НДС и Содержания услуги по переработке.

Выпущенная продукция

Список заполняется продукцией, которая выпущена под заказ давальца. Давальческая продукция всегда выпускается обособленно, то есть в документе Выпуск продукции должно быть заполнено поле Назначение, идентифицирующее заказ давальца.

Для каждой позиции выпущенной продукции указывается цена и сумма, которые определяют стоимость услуги по переработки в целом. Основным сервисом по заполнению табличной части являются команды подбора и заполнения:

  • Подобрать по остаткам к оформлению - с помощью формы подбора выбираются позиции, доступные для оформления отчета.
  • Заполнить по остаткам к оформлению - заполнение доступными для оформления изделиями.
  • Подобрать из заказов - с помощью формы подбора выбираются позиции из заказа давальца.
  • Заполнить по заказам - заполнение доступными к оформлению позициями из заказа давальца.

Дополнительно

На закладке Дополнительно можно указать признак Цена включает НДС, статус Налогообложения НДС и Подразделение, в рамках которого собираются все затраты по переработке давальческого сырья.

Валюты документа


Для изменения валюты документа, валюты взаиморасчетов и курса взаиморасчетов следует использовать гиперссылку Валюты.

  • При изменении валюты документа будет предложено пересчитать цены и суммы документа. Пересчет производится по курсу на дату документа.
  • При изменении валюты взаиморасчетов курс будет рассчитан автоматически, а указанная сумма взаиморасчетов будет распределена по строкам документа.
  • При изменении курса сумма взаиморасчетов будет расчитана автоматчески.

Оформление счета-фактуры
  • После проведения документа нажмите на гиперссылку Оформить счет-фактуру.
  • В появившейся диалоговой форме можно указать номера и даты платежно-расчетных документов, если была произведена предоплата.
  • Нажмите на кнопку Провести и закрыть для регистрации счета-фактуры.
  • После регистрации в документе отобразится гиперссылка на созданный счет-фактуру.

Для того чтобы распечатать счет-фактуру нажмите на кнопку Печать и выберите печатную форму счета-фактуры: Счет -фактура или Счет-фактура (в валюте).

Информация в счете-фактуре автоматически изменяется при изменении документа. Внесение изменений, которые не отражены в документе (например, изменение списка платежных документов) возможно только при отключенном флаге Выставлен.
