Списание ОС при выбытии ТМЦ

Рабочее место предназначено для оформления документов Списание ОС при выбытии групповых основных средств (списание ТМЦ в эксплуатации, возврат ТМЦ из эксплуатации).

Если выбыла часть ТМЦ, относящихся к групповому ОС, то оформляется частичное списание ОС.
Сумма списания рассчитывается автоматически исходя из стоимости выбывших ТМЦ (если для всех указана стоимость) или исходя из количества (если стоимость не указана хотя бы для одного ТМЦ).
Вид цен по которому оцениваются ТМЦ указывается в учетной политике фин. учета организации.
Для корректного расчета должно быть выполнено закрытие предыдущего месяца.
При расчете стоимости предполагается, что все ТМЦ одного группового ОС, учитываются в одной единице хранения.

Выбытие ТМЦ и списание группового ОС должно быть в одном месяце.
