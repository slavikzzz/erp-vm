Инвентаризация НМА

Документ предназначен для отражения результатов инвентаризации нематериальных активов.

При вводе документа требуется обязательно указать подразделение, в котором проводится инвентаризация.

В группе Проведение инвентаризации указываются данные, необходимые для заполнения сведений о инвентаризации и причинах ее проведения. В поле Документ вводится наименование документа-основания для проведения инвентаризации. Его можно выбрать из списка или указать вручную.

Нематериальные активы


На закладке указывается список НМА и результаты проведения инвентаризации. Для автоматического заполнения табличной части документа предназначена кнопка Заполнить:

  • Заполнить - По остаткам - заполняется список НМА по остаткам на дату проведения инвентаризации.
  • Заполнить - Данные учета - обновляются данные колонки Наличие и стоимость по данным учета, исходя из данных информационной базы.
  • Заполнить - Фактические данные - автоматически заполняется колонка Наличие и стоимость по факту по данным учета. В колонке По факту подтверждается соответствие данных учета фактическому наличию НМА. Фактическая стоимость заполняется автоматически в случае наличия НМА по данным учета, если данных учета по НМА нет, Фактическую стоимость можно указать вручную.

При наличии расхождений данных учета с фактическим наличием, для отражения результатов инвентаризации вводятся соответствующие документы на основании. А так же можно воспользоваться ссылкой для создания документа в колонке Отражение результатов инвентаризации.

Список НМА можно заполнить с помощью сканера штрихкодов или терминала сбора данных.

Инвентаризационная комиссия


На странице Инвентаризационная комиссия перечисляется состав членов инвентаризационной комиссии, председатель выбирается путем установки флага в колонке Председатель.

Печатные формы


Для документа предусмотрены следующие печатные формы:

  • Инвентаризационная опись ИНВ-1а

Ввод на основании


На основании документа можно оформить следующие хозяйственные операции:

  • Принятие к учету НМА
  • Перемещение НМА
  • Списание и частичная ликвидация НМА
  • Переоценка НМА
