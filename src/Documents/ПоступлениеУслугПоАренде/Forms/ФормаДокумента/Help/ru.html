Поступление услуг по аренде

Документ предназначен для отражения полученных актов за услуги по аренде, учитываемых по договорам аренды.

Документ служит основанием для формирования Счетов-фактур полученных.

Услуги по аренде, учитываемые по договорам с контрагентами, оформляются документами Приобретение услуг и прочих активов.

Начисления

Конкретная операция начислений выбирается в строках табличной части в поле Тип начисления:

  • Услуга по аренде – сумма регулярных начислений по аренде;
  • Зачет обеспечительного платежа – ежемесячный зачет доли ранее уплаченного аванса по договору. Операция может выполняться разово в начале или в конце срока лизинга.

По кнопке Заполнить можно заполнить табличную часть документа информацией из графика начислений.
