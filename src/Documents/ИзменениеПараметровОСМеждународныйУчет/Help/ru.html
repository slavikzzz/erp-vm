Изменение параметров ОС (международный учет)

Документ предназначен для отражения в системе изменений параметров основных средств в процессе их эксплуатации в рамках международного учета. Если изменяются параметры, связанные, с амортизацией ОС, то производится расчет и начисление амортизации на дату изменения параметров. Расчет амортизации в конце месяца будет производиться по новым правилам, начиная с даты изменения параметров.

При создании документа необходимо заполнить следующие реквизиты:

Закладка Основные средства

Закладка Параметры учета

Закладка Основные средства


На закладке заполняется список основных средств, у которых необходимо изменить параметры амортизации.
Для каждого основного средства (колонка Инв. №) указывается его инвентарный номер. При выборе основного средства, инвентарный номер заполняется автоматически кодом основного средства из соответствующего справочника.

Закладка Параметры учета


На закладке уточняется порядок учета (Начислять/Не начислять амортизацию) и определяются новые параметры амортизации.
В случае если выбран метод начисления амортизации Способ уменьшаемого остатка, можно дополнительно установить соответствующий коэффициент ускорения, позволяющий применить для выбранного основного средства ускоренный алгоритм по уменьшению его остаточной стоимости.

В случае если выбран метод начисления амортизации Пропорционально объему выпущенной продукции (работ), необходимо заполнить из предопределенного перечня показатель наработки, например, пробег, а также дополнительно уточнить Предполагаемый объем наработки (продукции, работ).
