Разрешение на замену материалов

Документ Разрешение на замену материалов предназначен для регистрации возможности замены материала (или набора материалов) на аналогичный.

Применение

Разрешение применяется как для ручного подбора аналогов материалов в документах, так и для автоматической замены на аналоги при запуске заказа в производство (формировании этапов), при обеспечении производства или при заказе материалов в производство.

Жизненный цикл разрешения на замену

Для управления жизненным циклом разрешения на замену используется статус. Возможные статусы:

  • Создано - черновик разрешения.
  • Утверждено - действующее разрешение. При переводе в этот статус дата начала периода действия разрешения заполняется текущей датой, если ранее не была заполнена.
  • Прекращено - разрешение, действие которого прекращено. При переводе в этот статус дата окончания периода действия заполняется текущей датой.

Действие разрешения на замену

По умолчанию разрешение на замену задается с действием По предприятию в целом. Но область действия может быть ограничена по следующим параметрам:

  • Направление деятельности
  • Подразделение
  • Производимое изделие
  • Ресурсная спецификация
  • Этап ресурсной спецификации
  • Заказ на производство

Если разрешение на замену материалов задается для ресурсной спецификации, то набор заменяемых материалов можно подобрать по данным табличной части Материалы и услуги данной спецификации.

При наличии нескольких действующих аналогов для осуществления автоматической замены выбирается наиболее точно подходящее разрешение, с более узко заданным действием. Приоритет действия разрешений определяется наличием уточнения области действия.
Например, если есть разрешение на замену, действующее по предприятию в целом, и есть подходящее разрешение для конкретного направления деятельности, то будет выбрано разрешение с уточненным направлением деятельности.
Если есть несколько разрешений с уточнением действия по разным параметром, то более приоритетным считается разрешение, у которого уточнен параметр, расположенный ниже в списке:

  • Направление деятельности
  • Подразделение
  • Производимое изделие
  • Ресурсная спецификация
  • Этап ресурсной спецификации
  • Заказ на производство

Если по номенклатуре создано несколько разрешений с одинаковыми областями действия, тогда применяется самое свежее разрешение - с более поздней датой начала действия.
