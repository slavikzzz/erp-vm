Завершение тем, этапов по исследованиям и разработкам

Рабочее место предназначено для создания документов Признание расходов по исследованиям и разработкам с хозяйственной операцией Завершение этапов исследований и разработок.

В списке отображаются справочные данные о суммах накопленных расходов по темам, этапам исследований и разработок, соответствующим указанным отборам. Точные данные сумм расчитываются при выполнении операции закрытия месяца Признание расходов по исследованиям и разработкам по созданным документам.
