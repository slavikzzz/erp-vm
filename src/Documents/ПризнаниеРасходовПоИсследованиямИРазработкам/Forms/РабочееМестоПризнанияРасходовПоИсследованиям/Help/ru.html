Признание расходов по исследованиям

Рабочее место предназначено для создания документов Признание расходов по исследованиям и разработкам с хозяйственной операцией Признание расходов по исследованиям.

В списке отображаются справочные данные о суммах накопленных расходов по темам, этапам исследований и разработок, у которых установлен признак Исследования. Точные суммы будут расчитаны при выполнении операции закрытия месяца Признание расходов по исследованиям и разработкам по введенным документам.

Создание документа

Создание документа
  • Выделите тему, этап (одну или несколько), для которой необходимо создать документ признания расходов
  • Нажмите на кнопку Оформить признание
  • В открывшемся документе заполните данные и проведите его
