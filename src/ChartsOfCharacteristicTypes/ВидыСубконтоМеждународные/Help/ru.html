Виды субконто международные

В программе под субконто понимается объект аналитического учета. Под видом субконто понимается множество однотипных объектов аналитического учета, из которых выбирается объект.

В качестве вида субконто могут устанавливаться справочники и другие объекты.

Программа поставляется с предопределенным списком видов субконто, в дополнение к которому можно вводить неограниченное количество новых видов субконто.

В Плане счетов «Международный» для каждого счета может быть установлен признак ведения аналитического учета по трем видам субконто. Для каждого вида субконто могут быть установлены дополнительные признаки:

  • только обороты – используется в случае, когда ведение учета остатков по субконто не целесообразно, например, для вида субконто Статьи движения денежных средств;
  • валютный – используется для ведения на счете суммового учета в разрезе валюты операции дополнительно к суммам в Функциональной валюте и в Валюте представления.
