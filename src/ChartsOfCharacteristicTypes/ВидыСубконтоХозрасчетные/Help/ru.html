Виды субконто

Предприятие ® План счетов ® Виды субконто (бухгалтерский и налоговый учет)

В программе под субконто понимается объект аналитического учета. Под видом субконто понимается множество однотипных объектов аналитического учета, из которых выбирается объект.

В качестве вида субконто могут устанавливаться справочники, виды документов и другие объекты.

Программа поставляется с предопределенным списком видов субконто, в дополнение к которому можно вводить неограниченное количество новых видов субконто.

В плане счетов для каждого счета может быть установлен признак ведения аналитического учета по трем видам субконто. Для каждого вида субконто могут быть установлены дополнительные признаки:

  • учет только оборотов – целесообразно в случае, когда учет остатков по субконто не имеет смысла, например, для вида субконто "Статьи движения денежных средств";
  • суммовой учет – целесообразно в большинстве случаев.
