Статьи доходов

Статьи доходов служат для классификации доходов предприятия по неосновной деятельности (реализации внеоборотных активов, финансовой деятельности, списания кредиторской задолженности и т.п). Если предприятие формирует финансовый результат по направлениям деятельности (включен флаг Финансовый результат по направлениям деятельности в разделе НСИ и администрирование - Финансовый результат и контроллинг - Финансовый результат) - для статьи доходов можно указать способ распределения доходов, зарегистрированных по ней, между разными направлениями деятельности.
Аналитика доходов позволяет указать дополнительные признаки доходов, зарегистрированных по данной статье. В распределении доходов по направлениям деятельности аналитика доходов не учитывается.

Для оформления реализации ОС, объектов строительства или НМА, в качестве аналитики доходов необходимо выбрать соответствующий справочник: ОС / Объекты строительства или НМА / НИОКР

Если доходы по указанной статье являются не принимаемыми в налоговом учете, необходимо снять флаг Принимается к налоговому учету.
Вид прочих доходов отвечает за то, в каких разделах бухгалтерской и налоговой отчетности будут отражены суммы доходов по соответствующим статьям.

Если используется совмещение основной системы налогообложения с ЕНВД, то следует указать к деятельности с какой системой налогообложения относятся доходы.
