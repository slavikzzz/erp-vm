Для обмена кадровыми документами с порталом "Работа в России" (ПРР) используются следующие регламентные задания:

Обновление состояний документов КЭДО
- Обновляет состояния подписей в форме "Документы кадрового ЭДО" см. меню "Кадры" - "Документы кадрового ЭДО"

Обработка документов печатных форм
- Выполняет запланированную отправку документов в ПРР

Получение документов портала "Работа в России"
- Получает подписи сотрудников с ПРР

Для оперативной работы с порталом задания можно запускать вручную (что бы не ждать запуска по расписанию).
