Настройка автовыбора номенклатуры

Описание доступно в документации на ИТС.

Настройка автовыбора номенклатуры используется для:

  • описания правил, по которым подбирается материал при производстве
  • описания побочного и промежуточного выхода
  • описания выходных изделий при разборке

Настройка автовыбора номенклатуры используется, когда номенклатуру (материал, выходное изделие и т.д.) или ее характеристику нельзя указать однозначно, но можно подобрать по свойствам продукции или характеристикам продукции.

Номенклатура указывается в свойстве характеристики изделия

Характеристика определяется по свойствам основного изделия

Определяется по алгоритму

Уточняется при производстве

Дополнительные материалы на ИТС

Номенклатура указывается в свойстве характеристики изделия

Настройка используется, когда нужная номенклатура указана в одном из свойств характеристики продукции.

Пример
Выходное изделие – стул. Для стула указано свойство обивка с типом позиция номенклатуры. Значение свойства - кожа. Для исходного материала указано: задается в свойстве продукции - обивка. При заполнении заказа на производство материал кожа будет заполнен автоматически. Если впоследствии для стула поменяют обивку на кож.зам. - в заказах на производство автоматически будет указан этот материал.

Характеристика определяется по свойствам основного изделия

Пример
Материал "Обивочная ткань", имеет в характеристиках свойство "Цвет ткани".

Готовая продукция "Кресло", в котором эта ткань применяется. Причем, у кресла могут быть разные цвета сиденья и спинки.
У кресла характеристики имеют 2 свойства: "Цвет сиденья" и "Цвет спинки".

Для того, чтобы можно было сопоставить эти свойства в спецификации, надо сначала сделать свойство "Цвет ткани", как новое.
А для кресла создавать свойства уже не самостоятельные, а по образцу свойства "Цвет ткани".

У свойств, созданных по образцу, набор значений будет не собственный, а взят из свойства-образца. То есть возможные цвета спинки и сиденья будут определяться доступными цветами ткани.
Тогда их можно будет сопоставлять в спецификации в автовыборе материала.

Нужная характеристика будет определена, по сопоставлению значений ее свойств со свойствами продукции.
В настройке соответствия свойств можно сопоставить одинаковые свойства, либо свойства, созданные по образцу. Есть возможность расчета значения свойства по формуле и по алгоритму.

Определяется по алгоритму

Характеристики номенклатуры могут определяться по алгоритму. При помощи расширений конфигурации можно добавить собственный алгоритм расчета значения свойства, который потом можно использовать в спецификациях и маршрутных картах

Уточняется при производстве

Настройка используется, когда номенклатуру или ее характеристику нельзя определить автоматически и требуется выбирать вручную при непосредственном производстве.
Например при автоподборе материала, в заказе на производство будет добавлена пустая строка материала и предложено вручную выбрать нужный материал. При этом будет доступна информация о применении материала.
